Arquitetura

1

Application Framework

É a camada que acessamos via código (APIs). Gerencia o ciclo de vida dos apps, UI (Views) e recursos.

2

Runtime & Libraries

O ART executa o código otimizado. Bibliotecas nativas (C/C++) cuidam de tarefas pesadas como gráficos e banco de dados.

3

Linux Kernel & HAL

A base do sistema. O Kernel conversa com o hardware através de drivers, enquanto o HAL padroniza essa comunicação.
