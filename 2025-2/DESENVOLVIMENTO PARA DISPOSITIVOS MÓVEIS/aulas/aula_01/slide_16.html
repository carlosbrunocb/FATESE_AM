Fundamentos

O que é uma
Activity?

O componente fundamental para a construção da interface do usuário no Android.

Uma Tela Única

Pense na Activity como uma janela ou página web. Ela representa uma única tela focada em algo que o usuário pode fazer (ex: Tela de Login, Lista de Emails).

Herança AppCompat

Em Kotlin, criamos uma classe que herda de AppCompatActivity. Isso fornece compatibilidade com versões antigas do Android.

Registro no Manifesto

Toda Activity deve ser declarada no arquivo AndroidManifest.xml para que o sistema saiba que ela existe.

MainActivity.kt
class MainActivity : AppCompatActivity() {
// Ponto de entrada da tela
override fun onCreate(savedInstanceState: Bundle?) {
super.onCreate(savedInstanceState)
// Define o layout visual (XML)
setContentView(R.layout.activity_main)
}
}
