Arquitetura

MVVM

Model - View - ViewModel

O padrão oficial recomendado pelo Google para apps Android.

Visão Geral do Conceito

Separação de Responsabilidades Divide claramente UI, Regra de Negócio e Dados.
View Passiva A UI apenas observa o estado exposto pelo ViewModel.
Model Centralizado Repositório único para dados Locais (Room) ou Remotos (API).
Testabilidade Facilita a criação de testes unitários isolados.
Desacoplamento O ViewModel não sabe quem é a View (sem referência de Context).
