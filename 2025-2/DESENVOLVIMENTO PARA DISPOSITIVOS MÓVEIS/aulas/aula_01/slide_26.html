Atividade Prática

Configuração do Ambiente de Desenvolvimento

Lab 01
1

Instalar Android Studio

Faça o download e instale a versão estável mais recente (Ladybug/Jellyfish+).

2

SDK Manager

Baixe o SDK Platform e Build-Tools mais recentes.

3

Habilitar Virtualização

Verifique na BIOS/Windows Features: Hyper-V, Intel VT-x ou AMD-V ativos.

4

Criar Virtual Device (AVD)

Device Manager → Create Device. Escolha Pixel + Imagem com Google APIs.

5

Hello World!

Crie um novo projeto "Empty Views Activity" e execute no emulador (Run).

"A configuração correta do ambiente é o primeiro passo para o sucesso."
