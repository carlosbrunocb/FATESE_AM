SDK Manager

Instalando o Android 15 (Vanilla Ice Cream)

Passo a Passo

Configuração Essencial

1. Abrir o SDK Manager

No Android Studio, vá em Tools > SDK Manager ou clique no ícone na barra de ferramentas.

2. Aba: SDK Platforms

Marque a caixa para o Android 15 (Vanilla Ice Cream). Isso instalará a imagem do sistema e as APIs necessárias (API Level 35).

3. Aba: SDK Tools

Garanta que os seguintes itens estejam selecionados:

  • Android SDK Build-Tools (versão mais recente)
  • Android Emulator
  • Android SDK Platform-Tools
  • Google Play Services (Opcional, mas recomendado)

4. Aplicar e Instalar

Clique em "Apply". Aceite os termos de licença e aguarde o download e instalação.
