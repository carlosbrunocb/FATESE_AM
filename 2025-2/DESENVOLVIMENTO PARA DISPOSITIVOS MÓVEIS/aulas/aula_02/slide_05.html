Kotlin:
Funções

fun Palavra-chave

Todas as funções em Kotlin são declaradas com fun. Elas são blocos de construção essenciais para organizar a lógica do seu aplicativo.

Estrutura Básica

Definem parâmetros tipados e um tipo de retorno (opcional se for Unit).

Single-Expression

Funções curtas podem ser escritas em uma única linha com =, sem chaves e com retorno inferido.

// 1. Função Padrão (Bloco de Código)

fun somar(a: Int, b: Int): Int {

return a + b

}

// 2. Single-Expression (Concisa)

fun multiplicar(x: Int, y: Int) = x * y

// 3. Função sem retorno (Unit/Void)

fun logMensagem(msg: String): Unit {

println("Log: $msg")

}

// 4. Uso (Chamada)

fun main() {

val resultado = somar(10, 5)

logMensagem("Soma: $resultado")

}
