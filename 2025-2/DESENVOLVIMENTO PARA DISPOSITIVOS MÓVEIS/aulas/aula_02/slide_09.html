Estrutura de Arquivos

Visão Geral do Projeto

Compreender onde cada arquivo reside é essencial para navegar e desenvolver com eficiência no Android Studio.

manifests/

O "RG" do seu aplicativo. Declara Activities, permissões de uso (internet, câmera) e configurações globais.

java/ (kotlin)

O cérebro do app. Contém todo o código-fonte Kotlin (.kt), onde implementamos a lógica e regras de negócio.

res/layout/

A face do app. Contém arquivos XML que definem a interface do usuário (botões, textos, campos) de cada tela.

res/values/

Recursos reutilizáveis. Define strings (textos), cores e temas para facilitar a manutenção e internacionalização.

Gradle Scripts

Automação de build. O arquivo build.gradle.kts (Module: app) é onde adicionamos bibliotecas externas e definimos a versão do SDK.
