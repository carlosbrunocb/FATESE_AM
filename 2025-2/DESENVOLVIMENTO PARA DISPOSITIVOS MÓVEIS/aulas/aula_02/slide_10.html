Interface

Aula 02

Navegar com eficiência pela IDE é crucial para a produtividade. Conheça as principais áreas de trabalho e atalhos essenciais.

Project View: Navegação hierárquica de arquivos (Java/Kotlin, XML, Resources).

Editor & Preview: Área de codificação com suporte a visualização de Layouts em tempo real.

Toolbar: Acesso rápido a Run, Debug e gerenciamento de dispositivos (AVD Manager).

Janelas Úteis: Logcat (logs), Build (status de compilação), Terminal e Version Control.

Navegação Rápida: Pressione Shift duas vezes para buscar tudo; Ctrl+B para ir à definição.
