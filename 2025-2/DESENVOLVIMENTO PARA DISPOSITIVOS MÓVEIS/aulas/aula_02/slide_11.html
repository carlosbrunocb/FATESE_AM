Ferramentas
Chave

Android Studio

Domine as principais janelas e utilitários para otimizar seu fluxo de trabalho, desde a codificação até a depuração.

Logcat

Console de logs do sistema. Essencial para ver mensagens de erro (stack traces) e debug filtrado por pacote.

Device Manager (AVD)

Gerenciador de Dispositivos Virtuais. Permite criar e configurar emuladores com diferentes versões do Android.

Layout Editor

Ferramenta visual para construção de UI. Alterna entre modo Design (arrastar e soltar) e Text (XML puro).

Gradle Tool Window

Painel para executar tarefas de build, limpar o projeto (Clean) e gerenciar dependências externas.

Build Variants

Seletor de variantes de compilação. Permite alternar rapidamente entre versões de Debug e Release.
