ViewBinding

Substitui o antigo findViewById.

  • Gera classes de ligação automaticamente.
  • Null Safety: Evita erros de referência nula.
  • Type Safety: Garante o tipo correto da View.

// 1. Declaração da variável de binding

private lateinit var binding: ActivityMainBinding

// 2. Inicialização no onCreate

override fun onCreate(savedInstanceState: Bundle?) {

super.onCreate(savedInstanceState)

// Infla o layout e cria a instância da classe de binding

binding = ActivityMainBinding.inflate(layoutInflater)

setContentView(binding.root)

// 3. Acesso direto às views pelo ID (sem findViewById)

binding.tvTitulo.text = "Olá, ViewBinding!"

binding.btnSalvar.setOnClickListener {

// Lógica do botão...

}

}

Dica O nome da classe é gerado convertendo o nome do XML para PascalCase + "Binding"
