Entenda como o Android Studio gerencia dependências e compila seu aplicativo.

Sistema de Build

Ferramenta que automatiza a compilação, testes e empacotamento do app.

build.gradle.kts

Scripts de configuração: um para o Projeto (global) e outro para o Módulo (app).

Sync Now

Sincronização obrigatória ao alterar scripts para baixar novas dependências.

Build & Run

Compila o código Kotlin em bytecode DEX e instala o APK no dispositivo.

Dica Importante

Mantenha versões do SDK, AGP (Plugin Android) e Gradle sempre compatíveis.
