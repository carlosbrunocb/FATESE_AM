Troubleshooting

Problemas Comuns de Compilação

Falha no Sync do Gradle

Ocorre frequentemente por problemas de conexão ou versões incompatíveis.

Solução:

Verifique a internet (proxy se necessário) e as versões do AGP/Gradle no build.gradle.

SDK ou Build Tools Faltando

Erro informando que componentes essenciais não foram encontrados.

Solução:

Vá em Tools > SDK Manager. Instale a versão da API (Platform) e o Build-Tools correspondente na aba SDK Tools.

JDK Incompatível

Erro de versão do Java (ex: requer Java 11 ou 17).

Solução:

Acesse File > Settings > Build, Execution, Deployment > Build Tools > Gradle e selecione o Embedded JDK.

Emulador Lento?

A virtualização sem aceleração de hardware torna o emulador inutilizável.

Dica de Performance:

Habilite a virtualização na BIOS do computador (VT-x / AMD-V). No SDK Manager, instale o Android Emulator Hypervisor Driver ou Intel HAXM.

Manifest Merge Failed

Acontece quando há conflitos entre o AndroidManifest.xml do seu app e o de bibliotecas importadas.

  • Verifique logs detalhados clicando em "Merged Manifest" na aba inferior do editor.

  • Cheque duplicidade de permissões ou declarações de minSdk conflitantes.

Ainda com problemas?

Consulte o StackOverflow ou a documentação oficial.
