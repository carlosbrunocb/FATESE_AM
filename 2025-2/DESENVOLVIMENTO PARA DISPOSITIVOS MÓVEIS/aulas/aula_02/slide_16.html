Criando um Novo Projeto

1

New Project

Abra o Android Studio e clique no botão New Project na tela de boas-vindas ou via menu File > New > New Project.

2

Template: Empty Views Activity

Selecione "Empty Views Activity".
Atenção: Não escolha "Empty Activity" (Compose).

3

Configurações Básicas

Name HelloWorld
Language Kotlin
4

Minimum SDK

Defina como API 24 (Android 7.0) ou superior. Isso garante compatibilidade com a maioria dos dispositivos.

5

Finish & Sync

Clique em Finish. O Android Studio irá baixar dependências. Aguarde a barra de progresso finalizar.

Executar (Run)

Valide o ambiente clicando no botão Run 'app' (Shift+F10). Se aparecer "Hello World!", sucesso!
