Passos para compilar e ver seu primeiro app rodando.

Prática: Parte 1

Primeiro App Android

Verificar Layout

Confirme em res/layout/activity_main.xml se existe o TextView "Hello World!"

Compilar e Executar

Clique no botão Run (▶) na barra de ferramentas superior do Android Studio

Selecionar Dispositivo

Escolha um AVD existente ou crie um novo emulador no Device Manager se necessário

Validar Execução

Aguarde o build terminar e confirme se o app abriu corretamente na tela do emulador

Resultado Esperado

O emulador deve exibir uma tela branca com "Hello World!" centralizado.
