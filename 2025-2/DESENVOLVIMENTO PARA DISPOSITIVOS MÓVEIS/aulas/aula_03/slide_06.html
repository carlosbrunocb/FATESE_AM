Criando uma Classe

Kotlin

1. Palavra-chave

Tudo começa com class.

2. Nome

Use PascalCase (Primeira letra maiúscula). Ex: Carro, Pessoa.

3. Corpo

Definido por chaves { }. Aqui dentro ficam os Atributos e Métodos.

C Carro.kt
package com.exemplo.app

class Carro {


// Atributos (Características)

var cor: String = "Prata"

var ano: Int = 2024


// Métodos (Ações)

fun acelerar() {

println("Vrum vrum! 🏎️")

}

}
