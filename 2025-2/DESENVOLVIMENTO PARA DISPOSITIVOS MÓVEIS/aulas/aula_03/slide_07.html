Instanciando Objetos

Dica de Ouro

Quem vem do Java costuma errar aqui. Em Kotlin, NÃO existe a palavra-chave new.

val c = new Carro()

val c = Carro()

A Fábrica rodou!

Quando você chama Carro(), você está pedindo para a fábrica construir uma unidade nova e colocá-la na memória.

Kt Main.kt

fun main() {


// 1. Criando o primeiro objeto (Instância A)

val meuFusca = Carro()

meuFusca.cor = "Azul"


// 2. Criando o segundo objeto (Instância B)

// Totalmente independente do primeiro!

val suaFerrari = Carro()

suaFerrari.cor = "Vermelha"


println(meuFusca.cor) // Output: Azul

println(suaFerrari.cor) // Output: Vermelha

}
