Anatomia da Classe

Atributos (Estado)

O que são?

São as variáveis declaradas dentro da classe. Elas representam as características ou dados que o objeto carrega.

Mutabilidade

  • val Imutável (Apenas leitura)
  • var Mutável (Pode ser alterado)
nome idade

class Pessoa {


// Atributos definem o ESTADO

val nome: String = "Carlos"

var idade: Int = 30

// Lista também pode ser atributo!

val hobbies = listOf("Correr", "Ler")

}


// Acessando (Sintaxe do Ponto)

println(pessoa.nome)
