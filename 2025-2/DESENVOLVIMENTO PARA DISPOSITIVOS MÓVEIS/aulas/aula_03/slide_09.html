Anatomia da Classe

Métodos (Comportamento)

O que são?

São as funções declaradas dentro da classe. Elas definem o que o objeto sabe fazer ou como ele manipula seus dados.

Poderes do Método

  • Pode ler os atributos da própria classe (`this.nome`).
  • Pode alterar os valores dos atributos (`var`).
  • Pode retornar resultados para quem chamou.

class ContaBancaria {

var saldo = 0.0


// Um Método é uma Função interna

fun depositar(valor: Double) {

// Modificando o estado do objeto

saldo = saldo + valor

println("Depositado: R$ $valor")

}

}


// Chamando a ação

minhaConta.depositar(100.0)
