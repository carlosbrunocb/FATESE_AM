Classe Carro: Tudo Junto

Estrutura

Definimos a classe com atributos (variáveis) e métodos (funções).

Interação

O método acelerar modifica o atributo velocidade. O objeto muda de estado!

No Main

Criamos a instância e "pilotamos" o carro chamando os métodos.

Carro.kt

class Carro {

var cor: String = ""

var modelo: String = ""

var velocidade: Int = 0


fun acelerar() {

velocidade = velocidade + 10

println("Acelerando... $velocidade km/h")

}

}


fun main() {

val fusca = Carro()

fusca.cor = "Azul"

fusca.acelerar() // Output: 10 km/h

fusca.acelerar() // Output: 20 km/h

}
