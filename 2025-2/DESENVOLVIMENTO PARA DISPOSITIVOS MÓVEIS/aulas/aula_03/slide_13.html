Quiz: O código compila?

Desafio.kt

class Usuario {

private var senha = "12345"

}


fun main() {

val user = Usuario()


// Tentativa de mudar a senha:

user.senha = "abcde"

println("Senha alterada!")

}

BUG ENCONTRADO

Qual é o problema?

Tentamos acessar diretamente um atributo marcado como private fora da classe onde ele foi criado.

Compiler Output

Error: Cannot access 'senha': it is private in 'Usuario'.

Como corrigir?

Crie um método público (ex: alterarSenha()) dentro da classe para validar e permitir a alteração de forma segura.
