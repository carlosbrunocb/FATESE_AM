Qual lista usar?

Em Kotlin, somos incentivados a usar estruturas imutáveis sempre que possível.

List (Imutável)

Caixa Fechada

Uma vez criada, você não pode adicionar, remover ou alterar itens. É mais leve e segura para dados fixos.

val lista = listOf("A", "B")

// lista.add("C") <- Erro!

Usaremos no App

MutableList

Caixa Aberta

Flexível. Permite adicionar (.add) e remover (.remove) itens a qualquer momento.

val lista = mutableListOf()

lista.add("Novo item") OK!
