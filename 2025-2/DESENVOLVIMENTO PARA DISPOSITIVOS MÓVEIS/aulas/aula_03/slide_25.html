Habilitando ViewBinding

build.gradle.kts (Module: app)

plugins { ... }


android {

namespace = "com.example.frasedodia"

compileSdk = 34


// Adicione este bloco:

buildFeatures {

viewBinding = true

}


defaultConfig { ... }

}

Onde colocar?

O bloco buildFeatures deve ficar dentro do bloco android { }, mas fora de defaultConfig.
