Passo 4: View (Interface)

Pegando Dados e Atualizando

A View não sabe sortear. Ela apenas pede para o ViewModel.

O ViewBinding injeta a frase diretamente no TextView.

MainActivity.kt (nova função)

private fun atualizarTela() {


// 1. Pede a frase sorteada ao ViewModel

val fraseSorteada = viewModel.sortearNovaFrase()


// 2. Exibe no TextView da tela

binding.tvFrase.text = fraseSorteada


}
