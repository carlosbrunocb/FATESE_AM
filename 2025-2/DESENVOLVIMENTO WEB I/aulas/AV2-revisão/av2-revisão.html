1. HTML5 Avançado

Estrutura e Semântica

Contêineres:

  • <div>: Elemento de bloco (quebra linha). Para layouts.
  • <span>: Elemento inline (não quebra linha). Para estilizar partes de texto.

APIs Nativas: Geolocalização, Web Storage, WebSockets, Drag & Drop.

Formulários Modernos

Novos Inputs: email, url, tel, number, date, color.

Validação Nativa:

  • required: Campo obrigatório.
  • pattern: Validação via Regex.
  • min/max: Limites numéricos.
  • minlength/maxlength: Limites de caracteres.
Constraint Validation API: Permite personalizar mensagens de erro via JS usando setCustomValidity() e verificar estados com validity.

Gráficos: Canvas vs SVG

Canvas SVG
Baseado em Pixels (Raster) Baseado em Vetores (XML)
Renderizado via JS Parte do DOM (CSS/JS)
Alta performance (Jogos) Escalável (Ícones, Logos)

Acessibilidade & Web Components

ARIA (Accessible Rich Internet Applications): Atributos como aria-label, role="alert" para leitores de tela.

Web Components: Criação de tags personalizadas encapsuladas.

  • Custom Elements
  • Shadow DOM (Isolamento de estilo)
  • HTML Templates (<template> e <slot>)

2. CSS3 Avançado

Box Model (O conceito fundamental)

Todo elemento é uma caixa composta por:

Margin (externo) > Border > Padding (interno) > Content
Dica de Ouro: Sempre use box-sizing: border-box; para que o padding e a borda não aumentem o tamanho total do elemento definido no width.

Layouts Modernos

Flexbox (Unidimensional)

Ideal para alinhar elementos em uma linha OU coluna.

  • display: flex;
  • justify-content: Alinhamento no eixo principal.
  • align-items: Alinhamento no eixo cruzado.
  • flex-direction: row | column.

CSS Grid (Bidimensional)

Ideal para layouts complexos (linhas E colunas).

  • display: grid;
  • grid-template-columns: Define colunas (ex: 1fr 2fr).
  • gap: Espaçamento.
  • grid-area: Posicionamento de itens.

SASS / SCSS (Pré-processador)

Estende o CSS com superpoderes. Precisa ser compilado.

  • Variáveis: $primary-color: #333;
  • Nesting (Aninhamento): Escrever CSS dentro de CSS para hierarquia.
  • Mixins: Blocos de código reutilizáveis.

3. JavaScript: Fundamentos

Variáveis e Tipos

  • var: Escopo de função, sofre hoisting (evitar).
  • let: Escopo de bloco, reatribuível.
  • const: Escopo de bloco, valor constante (mas objetos são mutáveis).

Tipos Primitivos: String, Number, Boolean, Null, Undefined, Symbol, BigInt.

Funções

// Função Tradicional
function somar(a, b) { return a + b; }

// Arrow Function (ES6+)
const somar = (a, b) => a + b;

DOM e Eventos

Seleção: document.getElementById(), querySelector().

Manipulação: innerHTML, style, classList.

Eventos: onclick, addEventListener('click', func).

4. Arrays, Objetos e Numbers

Arrays (Listas)

Estruturas ordenadas. Principais métodos:

  • Adicionar/Remover: push() (fim), pop() (fim), unshift() (início), shift() (início).
  • Iteração: map() (transforma), filter() (seleciona), reduce() (acumula), forEach().
  • Busca: find(), includes(), indexOf().

Objetos

Coleção de pares chave:valor.

const pessoa = {
    nome: "Ana",
    saudar: function() { return this.nome; }
};

Numbers e Math

  • JS usa ponto flutuante de 64 bits (cuidado com precisão decimal 0.1 + 0.2 !== 0.3).
  • Math: Math.round(), Math.floor(), Math.ceil(), Math.random().
  • Conversão: parseInt(), parseFloat(), Number().

5. JavaScript Avançado

Sets e Maps

Set: Coleção de valores únicos (não permite duplicatas). Métodos: add, has, delete.

Map: Coleção chave-valor onde a chave pode ser qualquer tipo (inclusive objetos). Mantém a ordem de inserção.

WeakSet / WeakMap: Referências fracas (garbage collection).

RegExp (Expressões Regulares)

Padrões para busca e validação de strings.

  • /padrao/flags
  • Ex: /\d+/ (dígitos), /[a-z]/i (letras, case insensitive).
  • Métodos: test(), match(), replace().

Tratamento de Erros

try {
    // Código perigoso
    throw new Error("Falha!");
} catch (erro) {
    // Tratamento
} finally {
    // Executa sempre
}

6. Web APIs Nativas

Interfaces para comunicar o navegador com recursos do sistema ou servidores.

Fetch API (Comunicação)

Substituto moderno do XMLHttpRequest. Baseado em Promises.

async function carregarDados() {
    try {
        const response = await fetch('https://api.exemplo.com');
        const data = await response.json();
        console.log(data);
    } catch (error) {
        console.error("Erro:", error);
    }
}

Web Storage (Armazenamento Local)

Local Storage Session Storage
Persistente (não expira) Temporário (apaga ao fechar aba)
localStorage.setItem('chave', 'valor') sessionStorage.setItem('chave', 'valor')

Outras APIs

  • Geolocation: navigator.geolocation.getCurrentPosition().
  • DOM API: Manipulação da árvore de elementos.
  • Notification API: Exibir notificações do sistema.
