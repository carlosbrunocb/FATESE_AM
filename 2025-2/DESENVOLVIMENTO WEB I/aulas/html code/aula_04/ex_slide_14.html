Propriedades de Texto em CSS

Este é um parágrafo de exemplo com bastante texto para demonstrar a propriedade `text-align: justify;`. Ela alinha o texto tanto à esquerda quanto à direita, criando uma aparência mais uniforme e limpa, como a encontrada em livros e jornais. A propriedade `line-height` também aumenta o espaçamento entre as linhas, melhorando a legibilidade.

Uma seção deste texto será sublinhada e em itálico para mostrar o uso de uma classe e suas propriedades de estilo de texto.

Passe o mouse sobre este link para ver as propriedades `text-decoration` e `letter-spacing` mudarem.
