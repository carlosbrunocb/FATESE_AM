Guia de Cálculo de Encargos e Impostos: CLT vs. PJ

Entender como o salário ou faturamento é tributado é essencial para o planejamento financeiro. Este guia detalha os cálculos para trabalhadores CLT (Consolidação das Leis do Trabalho) e PJ (Pessoa Jurídica) em diferentes regimes.

Atenção: As tabelas e alíquotas de impostos (INSS, IRRF, Simples Nacional) mudam periodicamente. Os valores utilizados aqui são baseados nas tabelas vigentes em 2024 e servem apenas para fins ilustrativos. Sempre consulte quais os valores vigentes que estão sendo aplicados atualmente.

Trabalhador CLT (Funcionário Registrado)

O trabalhador CLT tem valores descontados diretamente do seu salário (INSS e IRRF) e um valor depositado pela empresa em uma conta separada (FGTS).

1. FGTS (Fundo de Garantia do Tempo de Serviço)

O FGTS não é descontado do salário do funcionário. É uma obrigação da empresa, que deposita mensalmente 8% do salário bruto do empregado em uma conta na Caixa Econômica.

Exemplo de Cálculo (FGTS)

  • Salário Bruto: R$ 3.500,00
  • Cálculo: R$ 3.500,00 * 0.08 (8%)
  • Valor Depositado pela Empresa: R$ 280,00

Algoritmo (Pseudocódigo) - FGTS

FUNÇÃO CalcularFGTS(salarioBruto)
    DEFINIR aliquotaFGTS = 0.08  // 8%
    DEFINIR valorDeposito = salarioBruto * aliquotaFGTS
    RETORNAR valorDeposito
FIM FUNÇÃO

2. INSS (Previdência Social)

O INSS é descontado do salário bruto. O cálculo é progressivo, ou seja, é feito "faixa por faixa" do salário, assim como o Imposto de Renda.

Tabela INSS (Vigente em 2024)

Salário de Contribuição (R$) Alíquota
Até R$ 1.412,00 7,5%
De R$ 1.412,01 até R$ 2.666,68 9%
De R$ 2.666,69 até R$ 4.000,03 12%
De R$ 4.000,04 até R$ 7.786,02 (Teto) 14%

Exemplo de Cálculo (INSS)

Vamos calcular o desconto para um Salário Bruto de R$ 4.000,00.

  • Faixa 1: R$ 1.412,00 * 0.075 (7,5%) = R$ 105,90
  • Faixa 2: (R$ 2.666,68 - R$ 1.412,00) = R$ 1.254,68 * 0.09 (9%) = R$ 112,92
  • Faixa 3: (R$ 4.000,00 - R$ 2.666,68) = R$ 1.333,32 * 0.12 (12%) = R$ 159,99
  • Faixa 4: (Salário não atinge esta faixa) = R$ 0,00
  • Total do Desconto INSS: R$ 105,90 + R$ 112,92 + R$ 159,99 = R$ 378,81

Algoritmo (Pseudocódigo) - INSS

FUNÇÃO CalcularINSS(salarioBruto)
    DEFINIR desconto = 0

    // Faixa 1 (7.5%)
    DEFINIR limiteFaixa1 = 1412.00
    DEFINIR baseFaixa1 = MIN(salarioBruto, limiteFaixa1)
    desconto = desconto + (baseFaixa1 * 0.075)

    // Faixa 2 (9%)
    DEFINIR limiteFaixa2 = 2666.68
    SE salarioBruto > limiteFaixa1 ENTÃO
        DEFINIR baseFaixa2 = MIN(salarioBruto, limiteFaixa2) - limiteFaixa1
        desconto = desconto + (baseFaixa2 * 0.09)
    FIM SE

    // Faixa 3 (12%)
    DEFINIR limiteFaixa3 = 4000.03
    SE salarioBruto > limiteFaixa2 ENTÃO
        DEFINIR baseFaixa3 = MIN(salarioBruto, limiteFaixa3) - limiteFaixa2
        desconto = desconto + (baseFaixa3 * 0.12)
    FIM SE

    // Faixa 4 (14%)
    DEFINIR limiteTeto = 7786.02
    DEFINIR baseTeto = MIN(salarioBruto, limiteTeto) // Considera o teto
    SE baseTeto > limiteFaixa3 ENTÃO
        DEFINIR baseFaixa4 = baseTeto - limiteFaixa3
        desconto = desconto + (baseFaixa4 * 0.14)
    FIM SE

    RETORNAR desconto
FIM FUNÇÃO

3. IRRF (Imposto de Renda Retido na Fonte)

O IRRF é descontado do salário, mas não sobre o valor bruto. Ele incide sobre a Base de Cálculo.

Fórmula: Base de Cálculo = Salário Bruto - Desconto INSS - Dedução por Dependentes

(Valor por dependente em 2024: R$ 189,59)

Tabela IRRF (Vigente em 2024, com novo desconto simplificado)

Base de Cálculo (R$) Alíquota Parcela a Deduzir (R$)
Até 2.259,20 Isento R$ 0,00
De 2.259,21 até 2.826,65 7,5% R$ 169,44
De 2.826,66 até 3.751,05 15% R$ 381,44
De 3.751,06 até 4.664,68 22,5% R$ 662,77
Acima de 4.664,68 27,5% R$ 896,00
Desconto Simplificado: Desde 2023, existe uma opção de "desconto simplificado" de R$ 564,80 (para a tabela de 2024) que pode ser usado no lugar das deduções de INSS e dependentes, se for mais vantajoso. O sistema da Receita Federal deve aplicar o mais benéfico. No exemplo usaremos o cálculo padrão (deduzindo o INSS).

Exemplo de Cálculo (IRRF)

Usando o mesmo salário de R$ 4.000,00 (INSS de R$ 378,81) e supondo 0 dependentes.

  • Salário Bruto: R$ 4.000,00
  • Desconto INSS: R$ 378,81
  • Dependentes: R$ 0,00
  • Base de Cálculo: R$ 4.000,00 - R$ 378,81 - R$ 0,00 = R$ 3.621,19

Aplicando a tabela de IRRF (R$ 3.621,19 se enquadra na faixa de 15%):

  • Cálculo Bruto: R$ 3.621,19 * 15% = R$ 543,18
  • Parcela a Deduzir: R$ 381,44
  • Total do Desconto IRRF: R$ 543,18 - R$ 381,44 = R$ 161,74

Algoritmo (Pseudocódigo) - IRRF

FUNÇÃO CalcularIRRF(salarioBruto, numeroDependentes)
    // 1. Calcular INSS (usando a função anterior)
    DEFINIR descontoINSS = CalcularINSS(salarioBruto)

    // 2. Calcular dedução por dependentes
    DEFINIR valorPorDependente = 189.59
    DEFINIR deducaoDependentes = numeroDependentes * valorPorDependente

    // 3. Achar a Base de Cálculo
    DEFINIR baseCalculo = salarioBruto - descontoINSS - deducaoDependentes

    // 4. Aplicar tabela do IRRF
    DEFINIR impostoPagar = 0

    SE baseCalculo <= 2259.20 ENTÃO
        impostoPagar = 0
    SENÃO SE baseCalculo <= 2826.65 ENTÃO
        impostoPagar = (baseCalculo * 0.075) - 169.44
    SENÃO SE baseCalculo <= 3751.05 ENTÃO
        impostoPagar = (baseCalculo * 0.15) - 381.44
    SENÃO SE baseCalculo <= 4664.68 ENTÃO
        impostoPagar = (baseCalculo * 0.225) - 662.77
    SENÃO // Acima de 4664.68
        impostoPagar = (baseCalculo * 0.275) - 896.00
    FIM SE

    RETORNAR impostoPagar
FIM FUNÇÃO

Trabalhador PJ (Pessoa Jurídica)

O PJ não tem "salário", ele tem "faturamento". A empresa (o CNPJ) fatura contra um cliente e paga impostos sobre esse valor. O imposto varia drasticamente conforme o Regime Tributário.

1. MEI (Microempreendedor Individual)

É o regime mais simples, para quem fatura até R$ 81.000,00 por ano (valor de 2024).

Exemplo de Cálculo (MEI)

Não há cálculo percentual sobre o faturamento. O MEI paga um valor fixo mensal através do DAS-MEI (Documento de Arrecadação do Simples Nacional).

  • Imposto Mensal (DAS): Varia um pouco (ex: ~R$ 71,00 para Serviços, ~R$ 76,00 para Comércio e Serviços).
  • Este valor já inclui INSS, ICMS (se comércio) e/ou ISS (se serviço).

Algoritmo (Pseudocódigo) - MEI

FUNÇÃO CalcularImpostoMEI(tipoAtividade)
    SE tipoAtividade == "Serviço" ENTÃO
        RETORNAR "Valor Fixo (Ex: R$ 71,60)" // Valor simbólico de 2024
    SENÃO SE tipoAtividade == "Comércio" ENTÃO
        RETORNAR "Valor Fixo (Ex: R$ 70,60)" // Valor simbólico de 2024
    SENÃO
        RETORNAR "Valor Fixo (Ex: R$ 76,60)" // Comércio e Serviço
    FIM SE
FIM FUNÇÃO

2. Simples Nacional

Regime para empresas que faturam até R$ 4,8 milhões/ano. O imposto é pago em uma guia única (DAS) e a alíquota depende do Anexo (tipo de atividade) e do faturamento acumulado.

Exemplo de Cálculo (Simples Nacional)

Vamos supor uma empresa de serviços de tecnologia (Anexo III) com faturamento mensal de R$ 15.000,00 e que está na primeira faixa de faturamento (até R$ 180.000,00 nos últimos 12 meses).

  • Faturamento do Mês: R$ 15.000,00
  • Anexo/Faixa: Anexo III, Faixa 1
  • Alíquota: 6%
  • Imposto da Empresa (DAS): R$ 15.000,00 * 0.06 (6%) = R$ 900,00
Pró-labore vs. Lucro: O dono do PJ (o "trabalhador") define um "pró-labore" (seu salário oficial), sobre o qual ele pagará 11% de INSS e o IRRF (igual ao CLT). O restante do dinheiro da empresa (Faturamento - Imposto DAS - Pró-labore - Custos) é o lucro, que pode ser transferido para sua conta pessoal (pessoa física) com isenção de Imposto de Renda (desde que a contabilidade esteja em dia).

Algoritmo (Pseudocódigo) - Simples Nacional (Simplificado)

FUNÇÃO CalcularDAS_Simples(faturamentoMes, anexo, faturamento12Meses)
    DEFINIR aliquota = 0

    // Lógica complexa de RBT12 (Receita Bruta 12 meses) seria necessária
    // Para simplificar, usaremos a alíquota inicial
    
    SE anexo == "Anexo III" E faturamento12Meses <= 180000 ENTÃO
        aliquota = 0.06 // 6%
    SENÃO SE anexo == "Anexo V" E faturamento12Meses <= 180000 ENTÃO
         aliquota = 0.155 // 15.5%
    SENÃO
        // ... outras faixas e anexos
        aliquota = 0.06 // Padrão simplificado
    FIM SE

    DEFINIR valorDAS = faturamentoMes * aliquota
    RETORNAR valorDAS
FIM FUNÇÃO

3. Lucro Presumido

Regime mais complexo, geralmente para faturamento de até R$ 78 milhões/ano. O governo "presume" qual foi seu lucro e cobra os impostos (IRPJ e CSLL) sobre essa presunção. Outros impostos (PIS, COFINS, ISS) são pagos "por fora" (não são unificados).

Exemplo de Cálculo (Lucro Presumido - Simplificado)

Vamos focar apenas no IRPJ (Imposto de Renda da Pessoa Jurídica) trimestral.
Suposição: Empresa de serviços (presunção de lucro de 32%). Faturamento no Trimestre: R$ 100.000,00.

  • Faturamento Trimestral: R$ 100.000,00
  • Base de Cálculo Presumida: R$ 100.000,00 * 0.32 (32%) = R$ 32.000,00
  • Alíquota do IRPJ: 15%
  • IRPJ a Pagar no Trimestre: R$ 32.000,00 * 0.15 (15%) = R$ 4.800,00
Isso é uma enorme simplificação. Ainda haveria cálculo de CSLL (9% sobre a base presumida), PIS (0,65%), COFINS (3%) e ISS (2% a 5%) sobre o faturamento total, tornando a carga tributária real muito maior.

Algoritmo (Pseudocódigo) - Lucro Presumido (IRPJ Trimestral)

FUNÇÃO CalcularIRPJ_LucroPresumido(faturamentoTrimestre, percentualPresuncao)
    // percentualPresuncao: Ex: 0.32 para serviços gerais, 0.08 para comércio
    
    DEFINIR baseCalculoPresumida = faturamentoTrimestre * percentualPresuncao
    DEFINIR aliquotaIRPJ = 0.15 // 15%

    DEFINIR irpjDevido = baseCalculoPresumida * aliquotaIRPJ

    // Adicional de IR (10% sobre o que exceder R$ 60.000 de lucro no trimestre)
    DEFINIR limiteAdicional = 60000
    SE baseCalculoPresumida > limiteAdicional ENTÃO
        DEFINIR excesso = baseCalculoPresumida - limiteAdicional
        DEFINIR adicionalIRPJ = excesso * 0.10 // 10%
        irpjDevido = irpjDevido + adicionalIRPJ
    FIM SE

    RETORNAR irpjDevido
FIM FUNÇÃO
