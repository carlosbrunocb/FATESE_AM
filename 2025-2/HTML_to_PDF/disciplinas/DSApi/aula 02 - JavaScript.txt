                        </div>
                        <div class="code-example">
                            // Exemplos de expressões:<br />

                    <div class="topic-icon">
                        <i class="fas fa-cube"></i>
                    </div>
                    <div class="topic-content">
                        <div class="topic-title">Blocos de Código (Block Scope)</div>
                        <div class="topic-description">
                            Blocos de código são trechos delimitados por chaves { } que agrupam instruções. Variáveis
                            declaradas com let e const possuem escopo de bloco, ou seja, só são acessíveis dentro do
                            bloco em que foram declaradas.
                        </div>
                        <div class="code-example">
                            {<br />
                            let x = 10; // acessível apenas neste bloco<br />
                            const y = 20; // acessível apenas neste bloco<br />
                            }<br />
                            // x e y não são acessíveis aqui
                        </div>
                    </div>
                </div>
                <div class="topic">
                    <div class="topic-icon">
                        <i class="fas fa-key"></i>
                    </div>

                        <div class="topic-description">
                            Palavras reservadas pela linguagem JavaScript que possuem propósitos específicos e não podem
                            ser usadas como nomes de variáveis, funções ou classes.
                        </div>
                        <div class="code-example">
                            // Exemplos de palavras-chave:<br />
                            var, let, const, if, else, for, while, function,<br />
                            return, try, catch, class, import, export
                        </div>
