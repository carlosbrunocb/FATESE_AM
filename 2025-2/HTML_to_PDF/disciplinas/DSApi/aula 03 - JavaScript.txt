<td>O maior número positivo representável</td>

<td><strong>Number.MIN_VALUE</strong></td>
<td>O menor número positivo representável</td>

<td>Representa o valor infinito positivo</td>

console.log(Number.isInteger(42));      // true
console.log(Number.isInteger(42.5));    // false

console.log(Number.isFinite(42));       // true
console.log(Number.isFinite(Infinity)); // false

console.log(Number.isNaN(NaN));         // true
console.log(Number.isNaN("NaN"));       // false (diferente de isNaN())

// Verificando inteiros seguros

console.log(Number.isSafeInteger(9007199254740992));  // false

// Propriedades estáticas

console.log(Number.MIN_VALUE);          // 5e-324

console.log(Number.NEGATIVE_INFINITY);  // -Infinity</code></pre>

<div class="content">
<div class="explanation">
<h2>BigInt em JavaScript</h2>
<p>BigInt é um tipo primitivo em JavaScript que permite representar números inteiros de tamanho arbitrário.</p>
<ul>
<li>Criado para superar a limitação do tipo Number (±2<sup>53</sup> - 1)</li>
<li>Representado com o sufixo <span class="tag">n</span> ao final do número</li>
<li>Ideal para trabalhar com números muito grandes, como IDs em bancos de dados ou cálculos financeiros precisos</li>
<li>Não pode ser misturado com números regulares em operações</li>
<li>Não pode usar Math.* com BigInt</li>
</ul>
<p>BigInt pode ser representado em diferentes bases numéricas:</p>
<ul>
<li><span class="tag">Hexadecimal</span>: 0x123n</li>
<li><span class="tag">Octal</span>: 0o123n</li>
<li><span class="tag">Binário</span>: 0b101n</li>
</ul>
</div>
<div class="code-example">
<div class="code-title">
<i class="fas fa-code fa-lg"></i> Exemplos de BigInt em JavaScript
</div>
<pre><code class="language-javascript">// Criando BigInt com sufixo n
const bigInt = 1234567890123456789012345678901234567890n;

// Criando BigInt a partir de Number com construtor
const tambemBigInt = BigInt(123456789);

// Operações com BigInt
const soma = bigInt + 1n;
const divisao = bigInt / 10n;

// Comparações
console.log(1n === 1);    // false (tipos diferentes)
console.log(1n == 1);     // true (mesmos valores)

// BigInt em diferentes bases
const hexBigInt = 0xFFFFn;         // 65535n (hexadecimal)
const octalBigInt = 0o777n;        // 511n (octal)
const binaryBigInt = 0b1111111n;   // 127n (binário)

// Limitações
// Math.max(1n, 2n);  // TypeError: não pode misturar BigInt e outros tipos</code></pre>

<div class="output-title"><i class="fas fa-terminal fa-sm"></i> Observação</div>

<div class="content">
<div class="explanation">
<h2>Operadores Bitwise</h2>
<p>Operações bit a bit (bitwise) trabalham com representações binárias dos números. JavaScript oferece vários operadores para manipulação de bits:</p>
<table class="operator-table">
<tr>
<th>Operador</th>
<th>Nome</th>
<th>Descrição</th>
</tr>
<tr>
<td><strong>&amp;</strong></td>
<td>AND</td>
<td>Retorna 1 se ambos bits forem 1</td>
</tr>
<tr>
<td><strong>|</strong></td>
<td>OR</td>
<td>Retorna 1 se pelo menos um bit for 1</td>
</tr>
<tr>
<td><strong>^</strong></td>
<td>XOR</td>
<td>Retorna 1 se os bits forem diferentes</td>
</tr>
<tr>
<td><strong>~</strong></td>
<td>NOT</td>
<td>Inverte todos os bits</td>
</tr>
<tr>
<td><strong>&lt;&lt;</strong></td>
<td>Left shift</td>
<td>Desloca bits à esquerda</td>
</tr>
<tr>
<td><strong>&gt;&gt;</strong></td>
<td>Right shift</td>
<td>Desloca bits à direita</td>
</tr>
<tr>
<td><strong>&gt;&gt;&gt;</strong></td>
<td>Zero-fill right shift</td>
<td>Desloca bits à direita preenchendo com 0</td>
</tr>
</table>
<p>Úteis para: manipulação de flags, otimização, trabalho com dados binários e gráficos.</p>
</div>
<div class="code-example">
<div class="code-title">
<i class="fas fa-code fa-lg"></i> Exemplos de Operações Bitwise
</div>
<pre><code class="language-javascript">// Representação binária
// 5 = 101 em binário
// 3 = 011 em binário

// Operações AND, OR e XOR
let a = 5 &amp; 3;   // AND: 101 &amp; 011 = 001 (1 em decimal)
let b = 5 | 3;   // OR:  101 | 011 = 111 (7 em decimal)
let c = 5 ^ 3;   // XOR: 101 ^ 011 = 110 (6 em decimal)

// Operação NOT (complemento de 1)
let d = ~5;      // NOT: ~00000000000000000000000000000101

<ul>
<li><span class="tag">&amp;=</span> - AND e atribuição</li>
<li><span class="tag">|=</span> - OR e atribuição</li>
<li><span class="tag">^=</span> - XOR e atribuição</li>
<li><span class="tag">&lt;&lt;=</span>, <span class="tag">&gt;&gt;=</span>, <span class="tag">&gt;&gt;&gt;=</span> - Shift e atribuição</li>

<li>Permitem adicionar comportamento aos objetos</li>

</ul>

const pessoa = {
  firstName: "Maria",
  lastName: "Silva",
  idade: 28,
  // Método que utiliza this para acessar propriedades
  fullName: function() {
    return this.firstName + " " + this.lastName;
  },
  // Método que verifica se a pessoa é maior de idade
  verificarIdade: function() {
    return this.idade &gt;= 18 ? "Maior de idade" : "Menor de idade";
  }
};

// Chamando os métodos
console.log(pessoa.fullName());
console.log(pessoa.verificarIdade());</code></pre>

<h3>Características Principais</h3>
<ul>

const pessoa = {
  nome: "Ana",
  idade: 28,
  profissao: "Desenvolvedora",

  saudacao: function() {
    return `Olá, meu nome é ${this.nome}`;

};

function Carro(modelo, ano) {

<li><code>delete</code> - Remove um elemento (deixa "vazios")</li>
<li><code>splice()</code> - Remove/adiciona elementos em qualquer posição</li>

frutas.push('manga');      // Adiciona ao final
console.log(frutas);       // ['maçã', 'banana', 'laranja', 'manga']

const ultima = frutas.pop();  // Remove do final
console.log(ultima);       // 'manga'

// splice() - remove 1 item a partir do índice 1
frutas.splice(1, 1, 'pêra');

const citricos = ['limão', 'tangerina'];

// ['maçã', 'pêra', 'laranja', 'limão', 'tangerina']</code></pre>

const buffer = new ArrayBuffer(16); // 16 bytes

<h2>Concatenação e Conversão</h2>
<p>Quando trabalhamos com strings em JavaScript:</p>
<ul>
<li>O operador <strong>+</strong> entre duas strings realiza <span class="tag">concatenação</span> (junção)</li>
<li>Demais operadores matemáticos (<strong>-, *, /, %</strong>) tentam <span class="tag">converter automaticamente</span> as strings para números</li>
<li>Se a string não puder ser convertida para número, o resultado será <span class="tag">NaN</span></li>
</ul>
<p>Esta conversão automática (exceto para o operador +) é uma característica peculiar do JavaScript que pode causar comportamentos inesperados se não for compreendida corretamente.</p>
<div class="warning">
<strong>Atenção!</strong> A conversão automática pode levar a erros lógicos sutis em seu código. Sempre verifique os tipos de dados antes de realizar operações.

let str1 = "15";
let str2 = "3";
console.log(str1 + str2); // "153" (concatenação)

// Outras operações tentam converter para número
console.log(str1 - str2); // 12 (subtração numérica)
console.log(str1 * str2); // 45 (multiplicação numérica)
console.log(str1 / str2); // 5 (divisão numérica)

// Strings que não podem ser convertidas para número

console.log(texto - 5);    // NaN
console.log(texto * 2);    // NaN

<div class="content">
<div class="explanation">
<h2>Métodos para Manipulação de Números</h2>
<p>JavaScript oferece diversos métodos para manipular e formatar números:</p>
<ul>
<li><span class="method-name">toString()</span> - Converte número para string. Aceita base (2-36) como parâmetro.</li>
<li><span class="method-name">toExponential()</span> - Retorna notação científica com dígitos especificados.</li>
<li><span class="method-name">toFixed()</span> - Retorna string com número de casas decimais especificado.</li>
<li><span class="method-name">toPrecision()</span> - Retorna string com comprimento especificado.</li>
<li><span class="method-name">parseInt()</span> - Analisa uma string e retorna um número inteiro.</li>
<li><span class="method-name">parseFloat()</span> - Analisa uma string e retorna um número com ponto flutuante.</li>
</ul>
</div>
<div class="code-example">
<div class="code-title">
<i class="fas fa-code fa-lg"></i> Exemplos de Métodos Number
</div>
<pre><code class="language-javascript">let num = 123.456;

// Métodos de conversão para string
console.log(num.toString());       // "123.456"
console.log(num.toString(2));      // "1111011.0111010010111100011..."
console.log(num.toExponential(2)); // "1.23e+2"
console.log(num.toFixed(2));       // "123.46"
console.log(num.toPrecision(4));   // "123.5"

// Métodos para converter strings em números
console.log(parseInt("123.456"));   // 123
console.log(parseInt("10", 2));     // 2 (base binária)
console.log(parseFloat("123.456")); // 123.456</code></pre>
