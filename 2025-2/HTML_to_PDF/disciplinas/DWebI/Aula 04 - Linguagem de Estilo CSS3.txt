              <div>
                <span class="font-bold text-white">inline:</span> Ocupa apenas o espaço necessário para seu conteúdo,
                não quebra linha. Não aceita width/height e margin vertical.
                Aceitam apenas margens e preenchimentos horizontais (margin-left, margin-right, padding-left,
                padding-right)
              </div>
            </li>
            <li class="flex items-start">
              <i class="fas fa-th-large text-blue-300 mt-1 mr-2"></i>
              <div>
                <span class="font-bold text-white">inline-block:</span> Híbrido. Eles se comportam como uma "caixa"
                (bloco), mas em uma linha (inline).
                Não quebra linha, mas aceita todas as propriedades dimensionais como block.
              </div>
            </li>

              elementos que precisam ficar lado a lado mas também precisam de dimensões definidas, como botões em um
              menu horizontal.</p>
          </div>
          <div class="mt-4 text-blue-100">
            <p><span class="font-bold text-white">Elementos naturalmente block:</span> div, p, h1-h6, section, form</p>

  <span class="text-purple-400">flex-basis</span>: <span class="text-yellow-300">0%</span>;   <span class="text-blue-300">/* tamanho inicial zero */</span>
}

<span class="text-blue-300">/* Item que não encolhe */</span>
<span class="text-green-400">.item-fixed</span> {
  <span class="text-purple-400">flex-grow</span>: <span class="text-yellow-300">0</span>;     <span class="text-blue-300">/* não cresce */</span>
  <span class="text-purple-400">flex-shrink</span>: <span class="text-yellow-300">0</span>;    <span class="text-blue-300">/* não encolhe */</span>
  <span class="text-purple-400">flex-basis</span>: <span class="text-yellow-300">200px</span>; <span class="text-blue-300">/* largura fixa */</span>
}

<span class="text-blue-300">/* Usando shorthand flex */</span>
<span class="text-green-400">.item-responsive</span> {
  <span class="text-purple-400">flex</span>: <span class="text-yellow-300">1 1 auto</span>;  <span class="text-blue-300">/* cresce e encolhe proporcionalmente */</span>
}
</pre>
<div class="relative bg-blue-900 bg-opacity-20 p-2 mt-2">
<span class="text-white text-xs">Os valores padrão são: <span class="text-yellow-300">flex-grow: 0</span>, <span class="text-purple-400">flex-shrink: 1</span>, <span class="text-green-400">flex-basis: auto</span></span>
</div>
</div>
<div class="bg-gray-800 p-3 rounded mt-auto">
<p class="text-gray-400 text-sm"><i class="fas fa-info-circle mr-2"></i> Combinando estas três propriedades, você pode criar layouts flexíveis e distribuir o espaço disponível de maneira eficiente.</p>
