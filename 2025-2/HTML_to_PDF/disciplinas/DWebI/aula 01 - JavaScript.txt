Onde usar a tag &lt;script&gt; no HTML?
</h1>
</div>
<div class="content-container">
<div class="text-content">
<p>Você pode colocar a tag script em dois lugares principais:</p>
<ul>
<li><i class="fas fa-exclamation-triangle warning"></i> No <span class="highlight">&lt;head&gt;</span>: Carrega antes do conteúdo, mas pode atrasar a renderização da página</li>
<li><i class="fas fa-check-circle recommended"></i> No final do <span class="highlight">&lt;body&gt;</span>: <strong>Recomendado</strong> - Permite que o conteúdo seja carregado antes dos scripts</li>
</ul>
<p>Colocar no final do body melhora a performance e experiência do usuário.</p>
</div>
<div class="code-content">

<span class="code-tag">&lt;title&gt;</span>Minha página<span class="code-tag">&lt;/title&gt;</span><br/>
<span class="code-comment">/* Opção 1: no head */</span><br/>
<span class="code-tag">&lt;script&gt;</span><br/>
<span class="code-keyword">function</span> funcaoHead() { ... }<br/>
<span class="code-tag">&lt;/script&gt;</span><br/>
<span class="code-tag">&lt;/head&gt;</span><br/>
<span class="code-tag">&lt;body&gt;</span><br/>
<span class="code-tag">&lt;h1&gt;</span>Conteúdo<span class="code-tag">&lt;/h1&gt;</span><br/>
<br/>
<span class="code-comment">/* Opção 2: no final do body (recomendado) */</span><br/>
<span class="code-tag">&lt;script&gt;</span><br/>
<span class="code-keyword">function</span> funcaoBody() { ... }<br/>
<span class="code-tag">&lt;/script&gt;</span><br/>
<span class="code-tag">&lt;/body&gt;</span><br/>
<span class="code-tag">&lt;/html&gt;</span>
