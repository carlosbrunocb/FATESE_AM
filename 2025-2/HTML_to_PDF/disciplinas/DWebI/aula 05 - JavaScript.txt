<div class="mt-2 text-sm text-gray-600">
<i class="fas fa-info-circle mr-1"></i> Equivalente a: (A ∪ B) - (A ∩ B)

<pre><code><span class="keyword">const</span> setA = <span class="keyword">new</span> <span class="operator">Set</span>([1, 2, 3, 4]);
<span class="keyword">const</span> setB = <span class="keyword">new</span> <span class="operator">Set</span>([3, 4, 5, 6]);

<span class="comment">// Diferença (A - B)</span>

<span class="keyword">const</span> intersection = <span class="keyword">new</span> <span class="operator">Set</span>(
  [...setA].<span class="method">filter</span>(x =&gt; setB.<span class="method">has</span>(x))
);

<span class="keyword">const</span> symmetricDifference = <span class="keyword">new</span> <span class="operator">Set</span>(
  [...union].<span class="method">filter</span>(x =&gt; !intersection.<span class="method">has</span>(x))

<span class="comment">// symmetricDifference = {1, 2, 5, 6}</span></code></pre>

<div class="w-16 h-1 bg-yellow-500"></div>
</div>
<div class="grid grid-cols-1 md:grid-cols-2 gap-8">
<!-- Coluna de texto explicativo -->
<div class="text-lg">
<p class="mb-4">
    Em teoria dos conjuntos, podemos verificar as relações entre diferentes Sets:
</p>
<ul class="list-disc pl-5 mb-4 space-y-2">
<li><strong>Subconjunto (Subset)</strong>: Um conjunto A é subconjunto de B se todos os elementos de A também estão em B.</li>
<li><strong>Superconjunto (Superset)</strong>: Um conjunto A é superconjunto de B se A contém todos os elementos de B.</li>
<li><strong>Conjuntos Disjuntos</strong>: Dois conjuntos são disjuntos se não possuem elementos em comum.</li>
</ul>
<div class="mt-6 bg-blue-50 p-4 rounded-lg border-l-4 border-blue-500">
<h3 class="font-semibold mb-2">Aplicação:</h3>
<p>Essas verificações são úteis para validação de dados, verificação de permissões, e implementação de lógica de negócios baseada em conjuntos.</p>
</div>
</div>
<!-- Coluna de código -->
<div>
<div class="code-block">
<div class="code-header">
<span>JavaScript</span>
<span class="text-gray-400 text-xs">Set Methods</span>
</div>
<div class="code-content">
<pre><code><span class="comment">// Verifica se A é subconjunto de B</span>
<span class="keyword">function</span> <span class="method">isSubsetOf</span>(setA, setB) {
  <span class="keyword">for</span> (<span class="keyword">const</span> elem <span class="keyword">of</span> setA) {
    <span class="keyword">if</span> (!setB.<span class="method">has</span>(elem)) {
      <span class="keyword">return</span> <span class="keyword">false</span>;
    }
  }
  <span class="keyword">return</span> <span class="keyword">true</span>;
}

<span class="comment">// Verifica se A é superconjunto de B</span>
<span class="keyword">function</span> <span class="method">isSupersetOf</span>(setA, setB) {
  <span class="keyword">return</span> <span class="method">isSubsetOf</span>(setB, setA);
}

<span class="comment">// Verifica se A e B são disjuntos</span>
<span class="keyword">function</span> <span class="method">isDisjointFrom</span>(setA, setB) {
  <span class="keyword">for</span> (<span class="keyword">const</span> elem <span class="keyword">of</span> setA) {
    <span class="keyword">if</span> (setB.<span class="method">has</span>(elem)) {
      <span class="keyword">return</span> <span class="keyword">false</span>;
    }
  }
  <span class="keyword">return</span> <span class="keyword">true</span>;
}</code></pre>

<div class="w-16 h-1 bg-yellow-500"></div>
</div>
<div class="grid grid-cols-1 md:grid-cols-2 gap-8">
<!-- Coluna de texto explicativo -->
<div class="text-lg">
<p class="mb-4">
                    Um <strong>WeakSet</strong> é uma coleção especial de objetos onde as referências são <em>fracas</em> (weak references).
                </p>
<p class="mb-4">
                    Ao contrário de um Set normal, um WeakSet não impede que seus elementos sejam coletados pelo garbage collector quando não há outras referências a eles.
                </p>
<div class="mt-6 bg-blue-50 p-4 rounded-lg border-l-4 border-blue-500">
<h3 class="font-semibold mb-2">Objetivo e Aplicações:</h3>
<ul class="list-disc pl-5 space-y-1">
<li>Armazenar associações a objetos sem criar memory leaks</li>
<li>Identificar objetos que passaram por determinado processo</li>
<li>Ideal para metadados temporários sobre objetos</li>
<li>Tracking de objetos DOM que podem ser removidos</li>
</ul>
</div>
</div>
<!-- Coluna de código -->
<div>
<div class="code-block">
<div class="code-header">
<span>JavaScript</span>
<span class="text-gray-400 text-xs">WeakSet</span>
</div>
<div class="code-content">
<pre><code><span class="keyword">let</span> weakSet = <span class="keyword">new</span> <span class="operator">WeakSet</span>();

<span class="comment">// WeakSet só aceita objetos</span>
<span class="keyword">let</span> objeto1 = { id: 1, nome: 'Objeto 1' };
<span class="keyword">let</span> objeto2 = { id: 2, nome: 'Objeto 2' };

<span class="comment">// Adicionando objetos</span>
weakSet.<span class="method">add</span>(objeto1);
weakSet.<span class="method">add</span>(objeto2);

<span class="comment">// Verificando existência</span>
console.<span class="method">log</span>(weakSet.<span class="method">has</span>(objeto1));  <span class="comment">// true</span>

<span class="comment">// Se removermos todas as referências:</span>
<span class="comment">// objeto1 = null;</span>
<span class="comment">// O objeto será coletado automaticamente</span>
<span class="comment">// pelo garbage collector</span></code></pre>
</div>

<div class="w-16 h-1 bg-yellow-500"></div>

<div class="grid grid-cols-1 md:grid-cols-2 gap-8">
<!-- Coluna de texto explicativo -->
<div class="text-lg">
<p class="mb-4">
    O <strong>WeakSet</strong> suporta apenas três métodos para interação:
</p>
<ul class="list-disc pl-5 space-y-2 mb-4">
<li><code>add(objeto)</code>: adiciona objeto ao WeakSet</li>
<li><code>has(objeto)</code>: verifica se objeto está no WeakSet</li>
<li><code>delete(objeto)</code>: remove objeto do WeakSet</li>
</ul>
<div class="mt-6 bg-red-50 p-4 rounded-lg border-l-4 border-red-500">
<h3 class="font-semibold mb-2">Limitações:</h3>
<ul class="list-disc pl-5 space-y-1">
<li>Não possui a propriedade <code>size</code></li>
<li>Não é iterável (não funciona com <code>for...of</code>)</li>
<li>Não possui métodos <code>clear()</code>, <code>keys()</code>, <code>values()</code></li>
<li>Só aceita objetos como valores (não aceita primitivos)</li>
</ul>
</div>
</div>
<!-- Coluna de código -->

<pre><code><span class="comment">// Criando um WeakSet</span>
<span class="keyword">const</span> ws = <span class="keyword">new</span> <span class="operator">WeakSet</span>();

<span class="comment">// Objetos para adicionar ao WeakSet</span>
<span class="keyword">const</span> obj1 = { id: 1, nome: <span class="string">'Objeto 1'</span> };
<span class="keyword">const</span> obj2 = { id: 2, nome: <span class="string">'Objeto 2'</span> };

<span class="comment">// Adicionando objetos</span>
ws.<span class="method">add</span>(obj1);
ws.<span class="method">add</span>(obj2);

<span class="comment">// Verificando existência</span>
console.<span class="method">log</span>(ws.<span class="method">has</span>(obj1));  <span class="comment">// true</span>

<p class="mb-4">
    Um <strong>WeakMap</strong> é uma coleção de pares chave/valor onde as <em>chaves devem ser objetos</em> e os valores podem ser de qualquer tipo.
</p>
<p class="mb-4">
    A principal característica é que as referências às chaves são "fracas", permitindo que objetos sem outras referências sejam coletados pelo garbage collector, mesmo sendo chaves no WeakMap.
</p>
<div class="mt-6 bg-blue-50 p-4 rounded-lg border-l-4 border-blue-500">
<h3 class="font-semibold mb-2">Vantagens e Desvantagens:</h3>
<ul class="mb-2">
<strong>Vantagens:</strong>
<li class="ml-4">✓ Evita memory leaks</li>
<li class="ml-4">✓ Ideal para metadados de objetos temporários</li>
<li class="ml-4">✓ Dados privados para objetos</li>
</ul>
<ul>
<strong>Desvantagens:</strong>
<li class="ml-4">✗ Não é iterável</li>
<li class="ml-4">✗ Não possui propriedade size</li>
<li class="ml-4">✗ Só aceita objetos como chave</li>
</ul>

const privados = new WeakMap();

<li class="mb-2">Apenas aceita objetos como chaves</li>

<ul class="list-disc pl-5 space-y-1">

<li><strong>+</strong> - uma ou mais ocorrências</li>
<li><strong>?</strong> - zero ou uma ocorrência</li>
<li><strong>{n}</strong> - exatamente n ocorrências</li>
<li><strong>{n,m}</strong> - de n até m ocorrências</li>

<ul class="list-disc pl-5 space-y-1">

<li><strong>(?!...)</strong> - lookahead negativo</li>

<ul class="list-disc pl-5 space-y-1">

<li><strong>(?:abc)</strong> - grupo sem captura</li>
