QUESTÃO 5 (1,0) - Análise de Código: Endereçamento e Cópia Rasa

Considere o seguinte código, que utiliza cópia rasa. Qual será a saída do último `print()`? 

import copy
class Ponto:
    def __init__(self, x, y):
        self.coordenadas = [x, y]

p1 = Ponto(10, 20)
p2 = copy.copy(p1) # Cópia rasa

p2.coordenadas[0] = 99

print(p1.coordenadas[0])

Qual será o resultado do código?

  1. 10
  2. **99** (O objeto `Ponto` é copiado, mas a lista interna `self.coordenadas` é a mesma referência, alterando `p1`.)
  3. O código gerará um erro.
  4. [10, 20]
  5. [99, 20]

QUESTÃO 6 (1,0) - Análise de Código: Getters com @property

Analise a classe `Conta` abaixo. Qual será o valor impresso pela última linha de código? 

class Conta:
    def __init__(self, saldo):
        self._saldo = saldo # Atributo protegido
    
    @property
    def saldo_disponivel(self):
        # Um "getter" que retorna o valor calculado
        return self._saldo * 0.9

c = Conta(1000)
print(c.saldo_disponivel)

Qual será a saída do código?

  1. 1000
  2. **900.0**
  3. 100
  4. Um erro, pois `_saldo` é protegido.
  5. O código não imprime nada.
