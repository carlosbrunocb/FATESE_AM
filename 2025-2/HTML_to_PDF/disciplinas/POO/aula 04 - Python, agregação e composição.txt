<h1 class="text-4xl font-bold mb-5 python-blue">Exemplo prático de Composição em Python</h1>
<div class="mb-6">
<p class="text-lg">A composição representa uma relação <span class="font-semibold">"parte-de"</span> onde os objetos dependentes não podem existir sem o objeto principal. No exemplo abaixo, um livro é composto por capítulos:</p>
</div>
<div class="code-container">
<div class="line-numbers">
1<br/>2<br/>3<br/>4<br/>5<br/>6<br/>7<br/>8<br/>9<br/>10<br/>11<br/>12<br/>13<br/>14<br/>15<br/>16<br/>
</div>
<div class="code-content">
<span class="code-comment"># Classe que só existe como parte de um livro</span><br/>
<span class="code-keyword">class</span> <span class="font-bold">Capitulo</span>:<br/>
<span class="code-keyword">def</span> <span class="code-method">__init__</span>(self, titulo, conteudo):<br/>
        self.titulo = titulo<br/>
        self.conteudo = conteudo<br/>
<br/>
<span class="code-comment"># Classe que contém e cria os capítulos</span><br/>
<span class="code-keyword">class</span> <span class="font-bold">Livro</span>:<br/>
<span class="code-keyword">def</span> <span class="code-method">__init__</span>(self, titulo, autor):<br/>
        self.titulo = titulo<br/>
        self.autor = autor<br/>
        self.capitulos = []  <span class="code-comment"># Os capítulos são criados dentro do livro</span><br/>
<br/>
<span class="code-keyword">def</span> <span class="code-method">adicionar_capitulo</span>(self, titulo, conteudo):<br/>
<span class="code-comment"># Cria um novo capítulo interno ao livro</span><br/>
        novo_capitulo = Capitulo(titulo, conteudo)<br/>
        self.capitulos.append(novo_capitulo)
</div>
</div>
<div class="mt-6">
<h3 class="text-xl font-semibold mb-2">Exemplo de uso:</h3>
<div class="code-container">
<div class="line-numbers">
1<br/>2<br/>3<br/>4<br/>5<br/>6<br/>7<br/>
</div>
<div class="code-content">
<span class="code-comment"># Criando um livro</span><br/>
meu_livro = Livro(<span class="code-string">"Python Avançado"</span>, <span class="code-string">"Ana Silva"</span>)<br/>
<br/>
<span class="code-comment"># Adicionando capítulos ao livro</span><br/>
meu_livro.adicionar_capitulo(<span class="code-string">"Introdução"</span>, <span class="code-string">"Conceitos básicos..."</span>)<br/>
meu_livro.adicionar_capitulo(<span class="code-string">"Composição"</span>, <span class="code-string">"Relacionamento forte..."</span>)<br/>
<span class="code-keyword">print</span>(<span class="code-string">f"O livro {meu_livro.titulo} tem {len(meu_livro.capitulos)} capítulos"</span>)
</div>
</div>
<div class="output-container">
<div class="font-semibold mb-2">Output:</div>
<pre>O livro Python Avançado tem 2 capítulos</pre>
</div>
<div class="mt-4 bg-blue-50 p-4 rounded-lg">
<p><i class="fas fa-lightbulb text-yellow-500 mr-2"></i> <span class="font-semibold">Observação:</span> Na composição, se o objeto Livro for destruído, os objetos Capitulo também serão destruídos, pois não existem independentemente.</p>
</div>
</div>

<div class="slide-container flex flex-col">

<div class="px-16 py-12">
<h1 class="text-4xl font-bold mb-5 python-blue">Código de Composição em Python</h1>
<div class="mb-6">
<p class="text-lg">A composição representa uma relação <span class="font-semibold">"parte-de"</span> onde objetos componentes são criados dentro do objeto principal e dependem dele para existir:</p>
</div>
<div class="code-container">
<div class="line-numbers">
1<br/>2<br/>3<br/>4<br/>5<br/>6<br/>7<br/>8<br/>9<br/>10<br/>11<br/>12<br/>13<br/>14<br/>15<br/>
</div>
<div class="code-content">
<span class="code-comment"># Classe componente - só existe dentro da classe principal</span><br/>
<span class="code-keyword">class</span> <span class="font-bold">Roda</span>:<br/>
<span class="code-keyword">def</span> <span class="code-method">__init__</span>(self, posicao):<br/>
        self.posicao = posicao<br/>
<br/>
<span class="code-keyword">def</span> <span class="code-method">__str__</span>(self):<br/>
<span class="code-keyword">return</span> <span class="code-string">f"Roda: {self.posicao}"</span><br/>
<br/>
<span class="code-comment"># Classe principal que compõe as rodas internamente</span><br/>
<span class="code-keyword">class</span> <span class="font-bold">Carro</span>:<br/>
<span class="code-keyword">def</span> <span class="code-method">__init__</span>(self, modelo):<br/>
        self.modelo = modelo<br/>
<span class="code-comment"># Criação das rodas dentro do carro - composição</span><br/>
        self.rodas = [Roda(pos) <span class="code-keyword">for</span> pos <span class="code-keyword">in</span> [<span class="code-string">"dianteira esquerda"</span>, <span class="code-string">"dianteira direita"</span>, <span class="code-string">"traseira esquerda"</span>, <span class="code-string">"traseira direita"</span>]]
</div>
</div>
<div class="mt-6">
<h3 class="text-xl font-semibold mb-2">Exemplo de uso:</h3>
<div class="code-container">
<div class="line-numbers">
1<br/>2<br/>3<br/>4<br/>5<br/>6<br/>
</div>
<div class="code-content">
meu_carro = Carro(<span class="code-string">"Sedan"</span>)<br/>
<br/>
<span class="code-comment"># Acessando as rodas que são criadas automaticamente</span><br/>
<span class="code-keyword">for</span> roda <span class="code-keyword">in</span> meu_carro.rodas:<br/>
<span class="code-keyword">print</span>(roda)<br/>
<span class="code-keyword">print</span>(<span class="code-string">f"Total de rodas no {meu_carro.modelo}: {len(meu_carro.rodas)}"</span>)
</div>
</div>
<div class="output-container">
<div class="font-semibold mb-2">Output:</div>
<pre>Roda: dianteira esquerda
Roda: dianteira direita
Roda: traseira esquerda
Roda: traseira direita
Total de rodas no Sedan: 4</pre>
</div>
<div class="mt-4 bg-blue-50 p-4 rounded-lg">
<p><i class="fas fa-exclamation-triangle text-red-500 mr-2"></i> <span class="font-semibold">Importante:</span> Na composição, se o objeto Carro for destruído, todos os objetos Roda também serão destruídos, pois só existem como parte do Carro.</p>
</div>
</div>

<h1 class="text-4xl font-bold mb-6 python-blue">Diferença entre Agregação e Composição</h1>
<div class="grid grid-cols-2 gap-8">
<div class="border-r pr-4">
<h2 class="text-2xl font-semibold mb-4 text-center python-blue">Agregação</h2>
<div class="pros-cons border-blue-400 mb-4">
<p class="font-semibold mb-1">Relacionamento "tem-um" (HAS-A)</p>
<p class="text-gray-700">Objetos existem <span class="font-bold">independentemente</span></p>
</div>
<div class="diagram mb-4">
<div class="class-box">Departamento</div>
<div class="relation">
<div class="diamond"></div>
</div>
<div class="class-box">Funcionário</div>
</div>
<ul class="list-disc pl-6 space-y-2 text-gray-700">
<li>Associação <span class="font-bold">fraca</span> entre objetos</li>
<li>Se o objeto "todo" for destruído, as "partes" <span class="font-bold">continuam existindo</span></li>
<li>Objeto agregado pode pertencer a múltiplos agregadores</li>
<li>Em UML: <span class="font-bold">Losango branco</span> no lado do agregador</li>
</ul>
<div class="mt-4 bg-blue-50 p-3 rounded">
<p class="font-semibold">Exemplo: Departamento e Funcionários</p>
<p class="text-sm text-gray-600">O departamento agrega funcionários, mas os funcionários continuam existindo mesmo que o departamento seja fechado.</p>
</div>
</div>
<div class="pl-4">
<h2 class="text-2xl font-semibold mb-4 text-center python-blue">Composição</h2>
<div class="pros-cons border-green-500 mb-4">
<p class="font-semibold mb-1">Relacionamento "parte-de" (PART-OF)</p>
<p class="text-gray-700">Objetos são <span class="font-bold">dependentes</span> do todo</p>
</div>
<div class="diagram mb-4">
<div class="class-box">Carro</div>
<div class="relation">
<div class="diamond filled-diamond"></div>
</div>
<div class="class-box">Motor</div>
</div>
<ul class="list-disc pl-6 space-y-2 text-gray-700">
<li>Associação <span class="font-bold">forte</span> entre objetos</li>
<li>Se o objeto "todo" for destruído, as "partes" <span class="font-bold">também são destruídas</span></li>
<li>Objeto componente pertence apenas ao composto</li>
<li>Em UML: <span class="font-bold">Losango preto</span> no lado do composto</li>
</ul>
<div class="mt-4 bg-green-50 p-3 rounded">
<p class="font-semibold">Exemplo: Carro e Motor</p>
<p class="text-sm text-gray-600">O motor é parte do carro e não faz sentido existir separado do carro específico ao qual pertence.</p>
</div>
</div>
</div>
<div class="mt-6 pt-4 border-t border-gray-200">
<h3 class="text-xl font-semibold mb-2">Decisão de Design: Quando usar cada um?</h3>
<div class="grid grid-cols-2 gap-4">
<div>
<p class="flex items-center"><i class="fas fa-check-circle text-blue-600 mr-2"></i> Use <span class="font-bold">agregação</span> quando objetos podem existir independentemente</p>
</div>
<div>
<p class="flex items-center"><i class="fas fa-check-circle text-green-600 mr-2"></i> Use <span class="font-bold">composição</span> quando objetos só fazem sentido como parte do todo</p>
</div>
</div>
</div>
