<p class="code-line">    _inst = <span class="keyword">None</span> <span class="comment"># variável de classe para armazenar instância</span></p>
<p class="code-line"></p>
<p class="code-line"> <span class="keyword">def</span> <span class="method">__new__</span>(cls, *args, **kwargs):</p>
<p class="code-line"> <span class="keyword">if</span> cls._inst <span class="keyword">is None</span>:</p>
<p class="code-line"> <span class="comment"># Cria a instância apenas se não existir</span></p>
<p class="code-line">            cls._inst = <span class="keyword">super</span>().<span class="method">__new__</span>(cls)</p>
<p class="code-line"> <span class="keyword">return</span> cls._inst</p>
<p class="code-line"></p>
<p class="code-line"> <span class="keyword">def</span> <span class="method">__init__</span>(self):</p>
<p class="code-line"> <span class="comment"># Inicialização executada a cada chamada do construtor</span></p>
<p class="code-line">        self.debug = <span class="keyword">True</span></p>
<p class="code-line">        self.api_url = <span class="string">"https://api.exemplo.com"</span></p>
</div>
<div class="usage-example">
<p class="usage-title">Como funciona:</p>
<p>• <code>__new__</code> é chamado <em>antes</em> da criação do objeto e controla a instanciação</p>
<p>• Mesmo executando <code>Config()</code> múltiplas vezes, só existe uma instância</p>
<p>• Cuidado: <code>__init__</code> será chamado a cada <code>Config()</code>, mesmo sendo o mesmo objeto</p>
