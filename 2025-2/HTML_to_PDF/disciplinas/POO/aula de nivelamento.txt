<h1 class="title text-4xl font-bold mb-10">O que são Variáveis em Python?</h1>
<div class="content-container mt-8">
<p class="text-xl mb-6">
        Variáveis são espaços na memória que armazenam dados. Em Python:
    </p>
<ul class="list-disc ml-6 text-lg space-y-4 mb-8">
<li>Não é necessário declarar o tipo de variável antes de utilizá-la</li>
<li>O tipo é inferido automaticamente com base no valor atribuído</li>
<li>Nomes de variáveis devem começar com letra ou _ (underscore)</li>
<li>Python é case-sensitive: <span class="font-bold">nome</span> e <span class="font-bold">Nome</span> são variáveis diferentes</li>
</ul>
<div class="code-example text-lg">
<p class="font-bold mb-2">Exemplos:</p>
<code>
            nome = 'Maria'<br/>
            idade = 25<br/>
            altura = 1.65<br/>
            ativo = True
        </code>
</div>
<p class="text-lg mt-6">
        A variável armazena a referência ao objeto criado, não o próprio objeto.
        Python é uma linguagem de tipagem dinâmica - o tipo da variável pode mudar durante a execução.
    </p>

        Blocos condicionais permitem que seu código tome decisões e execute diferentes ações com base em condições.
    </p>
<ul class="list-disc ml-6 text-lg space-y-4 mb-6">
<li><span class="font-bold">if:</span> Executa um bloco de código se a condição for verdadeira</li>
<li><span class="font-bold">elif:</span> Verifica outra condição se a anterior for falsa</li>
<li><span class="font-bold">else:</span> Executa um bloco se nenhuma condição anterior for verdadeira</li>
</ul>
<div class="code-example text-lg">
<p class="font-bold mb-2">Exemplo:</p>
<code>
            idade = 18<br/><br/>
            if idade &gt;= 18:<br/>
                print('Maior de idade')<br/>
            elif idade &gt; 12:<br/>
                print('Adolescente')<br/>
            else:<br/>
                print('Criança')
        </code>
</div>
<p class="text-lg mt-6">
        É possível combinar condições usando operadores lógicos:<br/>
<span class="font-semibold">and</span> (E lógico), <span class="font-semibold">or</span> (OU lógico), <span class="font-semibold">not</span> (negação)
    </p>
<p class="text-lg mt-4">
        Lembre-se: Em Python, a indentação (tabulação) é fundamental para definir os blocos de código.

<h1 class="title text-4xl font-bold mb-10">Estruturas de Repetição: While e For</h1>
<div class="content-container mt-8">
<p class="text-xl mb-6">
                Estruturas de repetição permitem executar um bloco de código múltiplas vezes, otimizando tarefas repetitivas.
            </p>
<div class="grid grid-cols-2 gap-8">
<div>
<h2 class="text-2xl font-bold text-blue-700 mb-4">While</h2>
<ul class="list-disc ml-6 text-lg space-y-3">
<li>Executa um bloco <strong>enquanto</strong> uma condição for verdadeira</li>
<li>Ideal quando não sabemos previamente quantas repetições serão necessárias</li>
<li>Requer uma condição de parada para evitar loops infinitos</li>
</ul>
<div class="code-example text-lg mt-4">
<code>
                            contador = 0<br/>
                            while contador &lt; 5:<br/>
                                print(contador)<br/>
                                contador += 1
                        </code>

</div>
<div>
<h2 class="text-2xl font-bold text-blue-700 mb-4">For</h2>
<ul class="list-disc ml-6 text-lg space-y-3">
<li>Itera sobre uma sequência (lista, tupla, string, etc.)</li>
<li>Utilizado quando sabemos previamente sobre quais elementos iterar</li>
<li>Sintaxe mais simples para percorrer coleções</li>
</ul>
<div class="code-example text-lg mt-4">
<code>
                            for numero in range(5):<br/>
                                print(numero)<br/><br/>
                            for letra in "Python":<br/>
                                print(letra)
                        </code>
</div>
</div>
</div>
<p class="text-lg mt-6">
                Ambas estruturas suportam comandos <strong>break</strong> (interromper o loop) e <strong>continue</strong> (pular para a próxima iteração).

<div class="content-container mt-8">
<p class="text-xl mb-6">
        Em Python, a indentação (tabulação) define os blocos de código, diferente de outras linguagens que usam chaves {}:
    </p>

<li>Blocos são iniciados por dois pontos (:) na linha anterior</li>
<li>Todo o código no mesmo nível de indentação pertence ao mesmo bloco</li>
</ul>
<div class="grid grid-cols-2 gap-6 mb-6">
<div>
<p class="font-bold text-green-700 mb-2">✓ Código correto:</p>

def calcular_media(notas):<br/>

    for nota in notas:<br/>
        soma += nota<br/>
    return soma / len(notas)<br/>
</code>
</div>

<div>
<p class="font-bold text-red-700 mb-2">✗ Código incorreto:</p>
<div class="code-example incorrect-code">
<code>
def calcular_media(notas):<br/>
soma = 0<br/>

        return soma / len(notas)<br/>
</code>
</div>

<span class="font-bold">Dicas importantes:</span> Seja consistente com o tipo de indentação (espaços ou tabs, não misture). A maioria dos editores pode ser configurada para converter tabs em espaços automaticamente.
