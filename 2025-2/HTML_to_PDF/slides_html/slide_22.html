22. Vue.js — Framework Progressivo

Reatividade: Sistema de reatividade automática que atualiza o DOM quando os dados mudam

Diretivas: v-if, v-for, v-bind, v-model para manipulação declarativa do DOM

SFCs: Single File Components com template, script e style juntos

Curva de aprendizado: Amigável e gradual, documentação excelente

Ecossistema: Vue Router, Pinia (gerenciamento de estado), Nuxt.js

// Exemplo básico de Vue.js

const app = Vue.createApp({

data: () => ({ msg: 'Olá Vue', count: 0 }),

methods: { inc(){ this.count++ } }

});

app.mount('#vue-root');
