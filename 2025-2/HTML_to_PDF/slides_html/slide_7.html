APIs Nativas JavaScript - Frontend

7. Browser APIs vs Server APIs

Browser APIs

Exemplos

DOM

Fetch

Canvas

Geolocation

Ambiente

Cliente (navegador) com acesso limitado ao hardware e sistema

Permissões

Baseadas no usuário e políticas do navegador (origem, CORS)

Latência

Baixa (execução local no dispositivo do usuário)

Recursos

Limitados ao contexto do navegador e capacidades do dispositivo

Server APIs

Exemplos

REST

GraphQL

WebSockets

WebHooks

Ambiente

Servidor (backend) com acesso amplo a recursos e sistemas

Permissões

Baseadas em autenticação, autorização e políticas de segurança

Latência

Maior (depende de conexão de rede, localização, infraestrutura)

Recursos

Acesso a bancos de dados, sistemas externos, serviços e infraestrutura

Integração: Frontend consome Server APIs via Fetch API, estabelecendo a comunicação cliente-servidor
