Questão 10 (Objetiva)

O padrão Singleton usa __new__ para garantir uma única instância. No entanto, o método __init__ é chamado sempre que a classe é instanciada (Config()), mesmo que __new__ retorne a instância existente. Qual a saída do código abaixo? 

class Config:
    _inst = None
    def __new__(cls):
        if cls._inst is None:
            print("Criando instância")
            cls._inst = super().__new__(cls)
            cls._inst.valor = 0 # Inicializa o valor só na criação
        return cls._inst
    
    def __init__(self):
        # __init__ é chamado toda vez
        self.valor = self.valor + 1

c1 = Config()
c2 = Config()

print(c1 is c2)
print(c2.valor)
  1. Criando instância
    False
    1
  2. Criando instância
    True
    1
  3. Criando instância
    True
    2
  4. Criando instância
    Criando instância
    False
    1
