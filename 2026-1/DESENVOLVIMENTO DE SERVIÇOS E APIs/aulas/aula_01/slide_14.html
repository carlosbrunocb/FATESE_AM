Estrutura de uma Resposta HTTP

1. Status-line

VERSION STATUS CODE REASON
Exemplo: HTTP/1.1 200 OK

2. Headers (Cabeçalhos)

Metadados da resposta (Content-Type, Content-Length, Cache-Control, etc.).

3. Body (Corpo)

Dados retornados (JSON, HTML, etc.). Pode ser opcional dependendo do status code.

exemplo_resposta.http

HTTP/1.1 200 OK Content-Type: application/json Content-Length: 48 { "id": 1, "name": "Ana", "email": "ana@ex.com" }
