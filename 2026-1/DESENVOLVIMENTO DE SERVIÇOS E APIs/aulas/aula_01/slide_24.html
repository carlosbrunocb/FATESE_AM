5xx — Erro do Servidor

500

Internal Server Error: Erro genérico; falha inesperada no processamento da requisição pelo servidor.

501

Not Implemented: O método HTTP solicitado não é suportado pelo servidor (ex: método novo ou desconhecido).

502

Bad Gateway: O servidor (atuando como gateway/proxy) recebeu uma resposta inválida do servidor upstream.

503

Service Unavailable: O servidor está temporariamente indisponível (sobrecarga ou manutenção). Geralmente acompanha header Retry-After.

504

Gateway Timeout: O servidor (atuando como gateway/proxy) não recebeu uma resposta a tempo do servidor upstream.
