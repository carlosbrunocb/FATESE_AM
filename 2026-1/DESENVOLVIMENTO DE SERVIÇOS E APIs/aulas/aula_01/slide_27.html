Atividade Prática 2: Minha Primeira Requisição

Siga os passos abaixo utilizando o Postman ou Insomnia para realizar suas primeiras requisições em uma API pública.

1

Abra o Postman ou Insomnia e crie uma nova requisição. Utilizaremos a API pública JSONPlaceholder para os testes.

2

Configure o método para GET e insira a URL https://jsonplaceholder.typicode.com/posts. Clique em Send.

3

Analise os Headers da resposta recebida, prestando atenção especial aos campos Content-Type e Cache-Control.

4

Teste uma rota inexistente adicionando /99999 ao final da URL. Observe o Status Code retornado (404 Not Found).

5

Mude o método para POST, vá na aba Body, selecione raw/JSON e insira um objeto com título e corpo. Envie a requisição.

6

Verifique a resposta da criação do recurso. O Status Code retornado pelo servidor deve ser 201 Created.
