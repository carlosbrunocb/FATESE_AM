Aula 02 Arquiteturas de API

Introdução: O que é uma API?

  • Atua como uma interface padronizada para integração entre diferentes sistemas e aplicações.

  • Expõe capacidades e recursos de software por meio de contratos bem definidos (inputs e outputs).

  • Benefícios: promove o reuso de código, escala operacional, automação e criação de ecossistemas digitais.

  • Principais Estilos: SOAP, REST, GraphQL, gRPC (e complementos como Webhooks/WebSockets).
