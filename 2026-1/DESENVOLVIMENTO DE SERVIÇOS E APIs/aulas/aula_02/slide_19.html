Códigos de Status HTTP

  • Sucesso (2xx)

    200 OK (padrão), 201 Created (novo recurso), 204 No Content (sucesso sem corpo).

  • Erro do Cliente - Requisição (4xx)

    400 Bad Request (sintaxe inválida), 404 Not Found (recurso não encontrado), 422 Unprocessable (validação lógica).

  • Erro do Cliente - Segurança (4xx)

    401 Unauthorized (não autenticado), 403 Forbidden (autenticado mas sem permissão).

  • Erro do Servidor (5xx)

    500 Internal Server Error (falha genérica), 503 Service Unavailable (sobrecarga/manutenção).

  • Padronização de Erros

    Sempre retorne um corpo JSON estruturado: código interno, mensagem legível, detalhes técnicos (em dev) e link para documentação.
