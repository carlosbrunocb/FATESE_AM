Laboratório 01

Modelagem de Recursos (URIs)

Tempo Estimado: 20 min

Cenário: Sistema de Biblioteca

Você foi contratado para desenhar a API de uma biblioteca digital moderna. Sua primeira missão é definir as URIs para as operações fundamentais do sistema, seguindo estritamente os princípios RESTful.

1 Sua Missão: Desenhar as URIs

Ação 1

Listar todos os livros disponíveis

Ação 2

Buscar um livro específico pelo seu ID

Ação 3

Listar todos os empréstimos de um usuário específico

O que entregar?

Um rascunho das URIs com seus respectivos verbos HTTP.

VERBO /caminho/do/recurso

Ferramentas Sugeridas

Editor de Texto Swagger Editor

Critérios de Sucesso

  • Semântica: Use substantivos, não verbos.

  • Hierarquia: A estrutura da URL reflete a relação pai-filho?

  • Pluralidade: Consistência no uso de plurais para coleções.

  • Clean URL: Evite query strings para identificação de recursos (ex: ?id=1).
