Laboratório 02

Caça ao SOAP vs REST

Tempo Estimado: 25 min

Missão: Detetive de Arquitetura

O objetivo é identificar as diferenças estruturais "na selva". Você deve analisar documentações reais para confrontar a rigidez dos contratos SOAP com a flexibilidade do REST.

2 Etapas da Investigação

Passo 1

Analisar WSDL (SOAP)

Identifique tags como <soap:Envelope> e definições de tipos.

Passo 2

Explorar Swagger (REST)

Localize paths hierárquicos, verbos HTTP e JSON.

Entregável

Um quadro comparativo com 5 diferenças observadas.

Exemplo de Achado:

SOAP: XML Verboso REST: JSON Limpo

Ferramentas

Navegador Postman

O que procurar?

  • Complexidade dos nomes das tags.

  • Estrutura de envelopes (Envelope/Body).

  • Legibilidade humana (Human-readable).

  • Definição estrita de tipos (XSD).
