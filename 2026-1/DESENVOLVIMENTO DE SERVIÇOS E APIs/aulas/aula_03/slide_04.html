NODE.JS • CONCEITO

O que é Node.js?

Runtime JavaScript para executar código fora do navegador — com foco em I/O e serviços.

Resumo rápido

RUNTIME

Motor: V8 (Google)

Modelo: orientado a eventos

I/O: não bloqueante (libuv)

Definição

Node.js é um ambiente de execução (runtime) que permite rodar JavaScript no sistema operacional.

Ele não é um framework: é a base para criar servidores, APIs, automações e ferramentas de linha de comando (CLIs).

Assíncrono por padrão

Ótimo para rede e disco sem travar a execução.

APIs de sistema

Acesso a fs, http, stream, etc.

Ecossistema NPM

Bibliotecas, ferramentas e automação via scripts.

Full-stack (JS/TS)

Mesma linguagem no front e no back-end.

Ponto de atenção

Node.js é excelente em I/O; para CPU pesada, use workers/filas e arquitetura adequada.

Exemplo mínimo

node index.js
