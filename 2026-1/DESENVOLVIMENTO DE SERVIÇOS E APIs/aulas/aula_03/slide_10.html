NPM — Node Package Manager

Registro de pacotes + ferramenta de linha de comando para instalar, versionar e automatizar projetos Node.js.

Resumo rápido

PACKAGE MANAGER

Instala: bibliotecas e CLIs

Versões: Semver + lockfile

Automatiza: scripts de projeto

O que é

O NPM é o gerenciador de pacotes do ecossistema Node.js — com um registro de pacotes e uma CLI para consumi-los.

Ele mantém o package.json e registra versões instaladas em package-lock.json, facilitando reprodutibilidade e colaboração.

Dependências

Organiza dependencies e devDependencies.

Segurança

Audita vulnerabilidades e ajuda a manter pacotes atualizados.

Scripts

Padroniza comandos: start, dev, test, etc.

Ecossistema

Reuso: milhares de bibliotecas prontas para projetos reais.

Boa prática

Evite editar package-lock.json manualmente; use comandos do npm para manter consistência.

Comandos essenciais

CLI

Terminal

npm init -y
npm install date-fns
npm install -D eslint
npm start
npm run dev

Fluxo mental (bem prático)

Inicializar projeto

package.json

Instalar dependências

node_modules

Congelar versões

package-lock

Rodar scripts

npm run

Tradução prática

“Consigo instalar bibliotecas, padronizar comandos e reproduzir o ambiente em outra máquina.”
