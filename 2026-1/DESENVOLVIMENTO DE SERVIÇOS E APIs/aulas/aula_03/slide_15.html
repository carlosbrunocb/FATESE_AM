CONFIGURAÇÃO • AMBIENTE

Instalação do Node.js (LTS)

Objetivo: ter o runtime e o NPM prontos para criar projetos e rodar scripts no terminal.

Checklist rápido

LTS

Fonte: nodejs.org

Inclui: npm

Validação: node -v / npm -v

Dica: prefira LTS para estabilidade

Passo a passo (recomendado)

Instale e valide o ambiente

Windows / macOS / Linux

1

Baixe a versão LTS

Acesse https://nodejs.org e escolha LTS (mais estável para produção e laboratório).

LTS

Installer

nodejs.org

2

Instale (com NPM incluso)

Siga o instalador. Em geral, basta avançar mantendo as opções padrão. O npm vem junto com o Node.js.

No Windows, permita a atualização do PATH se solicitado.

3

Verifique no terminal

Abra um terminal novo e confirme as versões instaladas:

Terminal

node -v
npm -v

Se der erro de comando não encontrado

Reinicie o terminal/sessão. Se persistir, verifique o PATH do sistema.

Próximo passo após instalar

Criar o projeto com npm init -y e rodar com node index.js.

Critério de sucesso: versões exibidas no terminal

Dicas por sistema

boas práticas

Windows

Use PowerShell/Windows Terminal. Se necessário, reinstale garantindo a opção de adicionar ao PATH.

macOS

Prefira zsh no Terminal. Após instalar, feche e reabra o terminal para recarregar variáveis.

Linux

Considere usar um gerenciador de versões (NVM) para alternar entre versões sem conflito.

Alternativa recomendada: NVM

gerenciar versões

Útil quando você precisa de diferentes versões do Node em projetos distintos.

Exemplo (macOS/Linux)

nvm install --lts
nvm use --lts
node -v

Nota

No Windows, use nvm-windows (instalador separado).
