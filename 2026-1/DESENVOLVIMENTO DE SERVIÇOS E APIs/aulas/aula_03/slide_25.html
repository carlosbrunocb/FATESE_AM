Atividade Prática 1 — Instalação e Verificação

Objetivo: deixar o ambiente pronto (Node.js + NPM) e criar o primeiro package.json.

Entrega (evidências)

OK

Print/registro: node -v e npm -v

Arquivo: package.json criado

Pasta: disciplina/aula03/

Dica: use Node LTS

Passo a passo

Configure e valide o ambiente

Terminal

1

Instale o Node.js (LTS)

Baixe em https://nodejs.org e selecione a opção LTS.

nodejs.org

LTS

Installer

2

Verifique as versões

Abra um terminal novo e execute:

Terminal

node -v
npm -v

Se aparecer “command not found”, reinicie o terminal e verifique o PATH.

3

Crie a pasta da disciplina e o manifesto

Crie uma pasta de trabalho e inicialize o projeto com npm init -y:

Exemplo

mkdir ads-backend
cd ads-backend
mkdir aula03
cd aula03
npm init -y

Validação final

Confirme se o arquivo package.json foi criado na pasta do projeto.

Após concluir

Você estará pronto para instalar pacotes (npm i) e rodar scripts (node index.js).

Critério de sucesso: versões + package.json

Troubleshooting rápido

ajuda

node/npm não reconhecido

Reabra o terminal. No Windows, reinstale marcando “Add to PATH”.

Permissões (Linux/macOS)

Evite instalar global com sudo sem necessidade. Considere usar NVM.

Pasta do projeto

Use nomes simples: aula03, sem espaços, para facilitar no terminal.

Chame o professor/monitor e mostre o erro completo do terminal.
