Camada de Rotas (Routes)

Definição de Endpoints e Métodos HTTP

express.Router()
src/routes/tasks.routes.js
1const express = require('express');
2const router = express.Router();
3const controller = require('../controllers/tasks.controller');
4
5// Mapeamento: Rota + Verbo HTTP -> Controller
6
7// GET /tasks -> Lista todas
8router.get('/', controller.getAll);
9
10// POST /tasks -> Cria nova tarefa
11router.post('/', controller.create);
12
13// Rotas com parâmetros (ex: /tasks/1)
14router.get('/:id', controller.getById);
15router.put('/:id', controller.update);
16router.delete('/:id', controller.remove);
17
18module.exports = router;

Conceitos Chave

Porteiro da Aplicação

Definem os caminhos (URLs) de entrada e os Métodos HTTP (GET, POST, etc.) aceitos.

Sem Lógica de Negócio

Rotas não devem processar dados ou acessar o banco. Elas apenas repassam a requisição para o Controller.

Conexão

Mapeia Req/Res para a função correta.
Ex: GET / chama controller.getAll.
