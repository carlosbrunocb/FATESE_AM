Nodemon no Desenvolvimento

Produtividade com reinicialização automática

Hot Reload
1

Instalação

Instale como dependência de desenvolvimento (-D).

$ npm install -D nodemon
2

Configuração do Script

Adicione o script "dev" no package.json.

"scripts": { "dev": "nodemon app.js" }
3

Execução

Inicie o servidor em modo de observação.

$ npm run dev

Dica Avançada

Crie um arquivo nodemon.json na raiz para configurações personalizadas:

// nodemon.json
{
"watch": ["src"],
"ext": "js,json",
"ignore": ["test/*.js"]
}
Monitora apenas pasta /src
Ignora arquivos de teste
