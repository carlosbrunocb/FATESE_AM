Status Codes HTTP Essenciais

Códigos de resposta fundamentais para APIs RESTful

res.status(code).json(...)

Sucesso (Success)

200

OK

Requisição processada com sucesso. Padrão para GET e PUT. Retorna dados no corpo.

201

Created

Recurso criado com sucesso. Padrão para POST. Geralmente retorna o objeto criado.

204

No Content

Sucesso, mas sem corpo de resposta. Comum em DELETE ou atualizações sem retorno.

Erros (Client & Server)

400

Bad Request

Erro do cliente. Dados inválidos, JSON malformado ou parâmetros obrigatórios faltando.

404

Not Found

O recurso solicitado (ID, rota) não existe no servidor.

500

Internal Server Error

Erro inesperado no servidor (bug no código, falha no banco de dados). Evite expor stack traces.
