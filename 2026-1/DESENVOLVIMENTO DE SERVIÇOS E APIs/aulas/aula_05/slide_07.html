Query Params (req.query)

Filtros, busca e paginação via URL (GET)

Desenvolvimento de Serviços e APIs

Express.js

Quando usar Query Params?

Leitura + filtros

Filtros e ordenação

Ex.: status, categoria, sort, order.

Busca por palavra-chave

Ex.: ?keyword=node ou ?q=express.

Paginação

Ex.: ?page=2 e limit=10 para controlar volume.

Boas práticas

Defina valores padrão, valide tipos (Number) e normalize strings.

Exemplo de URL

GET

/usuarios?status=ativo&page=2&limit=10

status

"ativo"

page

"2"

limit

"10"

Como o Express lê

req.query (sempre strings)

{
  status: "ativo",
  page: "2",
  limit: "10"
}

Dica prática

Converta paginação: const page = Number(req.query.page || 1).
