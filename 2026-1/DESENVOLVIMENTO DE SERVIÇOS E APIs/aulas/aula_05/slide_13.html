Benefício

Instalação

Dependência de desenvolvimento (-D)

Setup rápido (terminal)

npm init -y
npm install express
npm install -D nodemon

Dica

Garanta o Node.js instalado e rode os comandos na pasta do projeto.

Scripts no package.json

"scripts": {
  "start": "node src/server.js",
  "dev": "nodemon src/server.js"
}

Execução

npm run dev (desenvolvimento) • npm start (produção/local)

Atenção

Em testes no Postman/Insomnia, confira a porta e o caminho base (ex.: /api).
