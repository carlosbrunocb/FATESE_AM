Código: setup + GET /search

Extraindo Query Params com req.query e respondendo JSON

Express.js

req.query

Exemplo mínimo funcional

Rota recebe /search?keyword=... e devolve o valor em JSON.

Status 200

server.js

const express = require('express')
const app = express()

// Middleware nativo para ler JSON no body (POST/PUT)
app.use(express.json())

// Query param: /search?keyword=node
app.get('/search', (req, res) => {
  const { keyword } = req.query
  return res.status(200).json({ keyword })
})

app.listen(3000, () => {
  console.log('Servidor rodando em http://localhost:3000')
})

Observação

req.query sempre chega como texto. Se precisar de número, converta (ex.: parseInt).

Como testar

Requisição

GET /search?keyword=express

Resposta esperada

{ "keyword": "express" }
