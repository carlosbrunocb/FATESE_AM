Aula 06: Middlewares no Express.js

02

Objetivos da Aula

O que aprenderemos hoje sobre o ciclo de requisições.

Ciclo de Vida HTTP

Compreender o conceito de Middleware e como ele intercepta o ciclo de solicitação-resposta (Request-Response).

Tipos de Middlewares

Implementar middlewares de nível de aplicação (global) e de nível de roteador (específico).

Validação de Dados

Criar middlewares de segurança para validar dados de entrada (body/params) antes que cheguem ao Controller.

Tratamento de Erros

Estruturar um middleware global (Error Handler) para padronizar as respostas de falha da API.
