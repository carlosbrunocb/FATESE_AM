Ecossistema Express

Integrados vs De Terceiros

Origem, manutenção e propósito de cada tipo de middleware.

VS

Integrados

Nativos (Built-in)

  • Já vêm instalados com o pacote express (zero dependências extras).

  • Focados em tarefas essenciais: parsing de corpo (body) e arquivos estáticos.

  • Mantidos pela equipe do Express.js.

// Parsing de JSON no body
app.use(express.json());

// Parsing de dados de formulário
app.use(express.urlencoded({ extended: true }));

// Servir arquivos estáticos
app.use(express.static('public'));

De Terceiros

Comunidade (NPM)

  • Requerem instalação via gerenciador de pacotes (ex: npm install cors).

  • Resolve problemas específicos: segurança, logs, cookies, autenticação.

  • Vasto ecossistema open source.

const cors = require('cors');
const helmet = require('helmet');
const morgan = require('morgan');

app.use(cors());
app.use(helmet());
app.use(morgan('dev'));
