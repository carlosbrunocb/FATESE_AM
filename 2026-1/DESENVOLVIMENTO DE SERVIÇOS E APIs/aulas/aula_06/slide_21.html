Flowchart: Express Middleware Stack
Ordem Sequencial (Top-Down)

O Express executa linha a linha. express.json() deve vir antes das rotas para que o req.body esteja disponível. O Logger registra tudo que passa.

Rotas e Validação

Dentro do roteador, middlewares específicos (como validação) protegem os controllers. Se falhar, o fluxo é desviado para o tratamento de erro.

Rota 404

Se nenhuma rota der match, a requisição "cai" no final da pilha. É comum ter um middleware aqui para gerar um erro 404 explícito.

Error Handler (O Último)

O middleware com 4 parâmetros (err, req, res, next) deve ser SEMPRE o último. Ele captura qualquer next(err) chamado anteriormente.
