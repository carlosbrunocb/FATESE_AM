Passo 1: Implementar Logger

1

Objetivo: Registrar Requisições

  • Crie a pasta src/middlewares caso não exista.

  • Crie o arquivo logger.js dentro dessa pasta.

  • Importe e registre o middleware no server.js usando app.use().

  • Atenção: Certifique-se de registrar ANTES das rotas!

Critério de Aceitação

Ao fazer qualquer requisição (GET, POST, etc.) via Postman ou navegador, o terminal do servidor deve exibir uma linha com: [DATA_HORA] METODO URL.

src/middlewares/logger.js
const logger = (req, res, next) => {
console.log( `[${new Date().toISOString()}] ${req.method} ${req.url}` );
next(); // Não esqueça!
};

module.exports = logger;
src/server.js
const express = require('express');
const logger = require('./middlewares/logger');
const app = express();

// 1. Logger (Primeiro de tudo)
app.use(logger);

// 2. Body Parser
app.use(express.json());

// 3. Rotas
app.use('/api', routes);
