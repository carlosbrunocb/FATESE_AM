Passo 2: Middleware de Validação

2

Objetivo: Validar Dados de Entrada

  • Crie o arquivo src/middlewares/validaLivro.js.

  • Verifique se titulo, autor e ano estão presentes no req.body.

  • Se faltar algo, retorne status 400 (Bad Request).

  • Importe e injete o middleware apenas nas rotas POST e PUT em livroRoutes.js.

Critério de Aceitação

Tente cadastrar um livro via Postman sem enviar o campo "titulo". O servidor deve retornar status 400 com JSON de erro, e o controller de criação NÃO deve ser executado.

src/middlewares/validaLivro.js
const validaLivro = (req, res, next) => {
const { titulo, autor, ano } = req.body;

// Validação simples
if (!titulo || !autor || !ano) {
return res.status(400).json({
erro: "Dados inválidos"
});
}
next(); // Dados ok, pode passar!
};
module.exports = validaLivro;
src/routes/livroRoutes.js
const express = require('express');
const router = express.Router();
const valida = require('../middlewares/validaLivro');

// Validação apenas na criação e atualização router.post('/livros', valida, controller.criar); router.put('/livros/:id', valida, controller.editar);

// Leitura não precisa validar body
router.get('/livros', controller.listar);

module.exports = router;
