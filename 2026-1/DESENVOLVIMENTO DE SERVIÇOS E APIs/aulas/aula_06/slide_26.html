Passo 3: Error Handler e Estrutura Final

3

Objetivo: Tratamento Centralizado

  • Crie o arquivo src/middlewares/errorHandler.js.

  • Use a assinatura de 4 parâmetros: (err, req, res, next).

  • Registre no server.js usando app.use(errorHandler).

  • Essencial: Deve ser o ÚLTIMO middleware, após todas as rotas!

Critério de Aceitação

Qualquer erro não tratado (ex: `throw new Error('Ops')` em uma rota) deve retornar um JSON padronizado com status 500, em vez de derrubar o servidor.

src/middlewares/errorHandler.js
const errorHandler = (err, req, res, next) => {
console.error(err.stack);
res.status(500).json({
erro: "Erro Interno",
mensagem: "Algo deu errado."
});
};

module.exports = errorHandler;
src/server.js (Estrutura Final)
const express = require('express');
const app = express() ;

// 1. Middlewares Globais
app.use(express.json());
app.use(logger); // Loga tudo

// 2. Rotas (algumas com validaLivro)
app.use('/api', routes);

// 3. Tratamento de Erros (FINAL)
app.use(errorHandler);

module.exports = app;
