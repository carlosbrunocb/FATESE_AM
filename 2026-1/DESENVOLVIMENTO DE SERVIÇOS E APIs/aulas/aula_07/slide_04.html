Formatos de Dados
O que é um Formato de Dados?
Um formato de dados é uma convenção textual/estrutural para representar e trocar informações entre sistemas. Define como os dados são organizados, armazenados e transmitidos.
Interoperabilidade: Permite comunicação entre sistemas diferentes
Validação: JSON Schema, XSD para garantir integridade
Performance: Tamanho do payload e velocidade de processamento
Compatibilidade: Suporte a sistemas legados e modernos
