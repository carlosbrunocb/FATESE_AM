Formatos de Dados
Definição
JSON (JavaScript Object Notation) é um formato leve de troca de dados, legível por humanos e fácil de processar por máquinas. É derivado da sintaxe do JavaScript e se tornou o padrão de facto para APIs REST.
Tipos nativos: string, number, boolean, null, object, array
Sintaxe enxuta: Estrutura chave-valor e arrays
Serialização: JSON.stringify() para converter em string
Desserialização: JSON.parse() para converter em objeto
Exemplo JSON
{
"id": 1,
"titulo": "1984",
"autor": "George Orwell",
"ano": 1949,
"disponivel": true,
"tags": ["distopia", "clássico"]
}
Vantagens do JSON
Menor verbosidade
Parseamento rápido
Ideal para mobile
Suporte universal
JSON é o formato preferido para APIs REST e SPAs
Limitações e Atenções
Sem comentários: JSON não suporta comentários
Sem tipo Date: Use ISO 8601 para datas
Segurança: Nunca use eval() para parse
