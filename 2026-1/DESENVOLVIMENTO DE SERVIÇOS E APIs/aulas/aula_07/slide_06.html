Formatos de Dados
Exemplo Prático - Objeto Livro
// Exemplo de objeto JSON com estrutura completa
{
  "id": 1,
  "titulo": "1984",
  "autor": {
    "nome": "George Orwell",
    "nacionalidade": "Britânico"
  },
  "ano": 1949,
  "tags": ["distopia", "clássico"],
  "disponivel": true
}
Erros Comuns
Aspas simples: JSON requer aspas duplas
Vírgula sobrando: Último item não pode ter vírgula
Comentários: JSON não suporta comentários
Manipulação em Node.js
// Serialização: Objeto -> String JSON
const json = JSON.stringify(objeto);
 
// Desserialização: String JSON -> Objeto
const obj = JSON.parse(jsonString);
 
// Validação com try-catch
try {
  const obj = JSON.parse(str);
} catch (e) {
  console.error("JSON inválido");
}
Boas Práticas
Nomenclatura: Use camelCase ou snake_case
Validação: Sempre valide antes de parsear
Tratamento: Use try-catch para erros
Datas: Formato ISO 8601 (YYYY-MM-DD)
