Formatos de Dados
Definição
XML (eXtensible Markup Language) é uma linguagem de marcação para representar, armazenar e transportar dados de forma estruturada e auto-descritiva. Define regras para codificar documentos em formato legível por humanos e máquinas.
Baseado em tags: Hierarquia rígida (árvore) de elementos
Extensível: Você define suas próprias tags
Validação: Suporte a DTD/XSD para contratos fortes
Exemplo XML
<?xml version="1.0" encoding="UTF-8"?>
<biblioteca>
  <livro id="1">
    <titulo>1984</titulo>
    <autor>George Orwell</autor>
  </livro>
</biblioteca>
Vantagens do XML
Contratos fortes (XSD)
Transformação XSLT
Assinaturas digitais
Gov/corporativo
XML é ideal para documentos complexos e integrações governamentais.
Limitações e Atenções
Verboso: Payload significativamente maior que JSON
Parsing: Requer processamento mais pesado (DOM/SAX)
Complexidade: Curva de aprendizado superior ao JSON
