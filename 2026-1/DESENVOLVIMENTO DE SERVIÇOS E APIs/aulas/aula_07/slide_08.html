Formatos de Dados
Exemplo Prático - Documento XML
<?xml version="1.0" encoding="UTF-8"?>
<biblioteca>
<livro id="1" categoria="ficcao">
<titulo>1984</titulo>
<autor>
<nome>George Orwell</nome>
<nacionalidade>Britânico</nacionalidade>
</autor>
<ano>1949</ano>
</livro>
</biblioteca>
Regras de Bem-formação
Elemento raiz único: Todo documento XML deve ter um elemento raiz
Tags fechadas: Todas as tags devem ser fechadas corretamente
Atributos entre aspas: Valores de atributos devem estar entre aspas
Elementos vs Atributos
<livro id="1" categoria="ficcao">
  <titulo>1984</titulo>
  <autor>George Orwell</autor>
</livro>
Dica: Use atributos para metadados curtos e elementos para dados hierárquicos
Boas Práticas XML
Validação: Use DTD ou XSD para validação
Encoding: Defina UTF-8 para caracteres especiais
Namespaces: Use namespaces para evitar conflitos
Escaping: Escape caracteres especiais (&amp;, &lt;, &gt;)
