Formatos de Dados
Instalação e Parsing (XML → JSON)
// Instalação: npm install xml2js
const xml2js = require('xml2js');
 
// Parser: XML → Objeto JavaScript
const parser = new xml2js.Parser({
explicitArray: false,
ignoreAttrs: false
});
 
const xmlFornecedor = `<fornecedor> <livro id="99"><titulo>O Guia do Mochileiro</titulo></livro> </fornecedor>`;
const resultado = await parser.parseStringPromise(xmlFornecedor);
Building (JSON → XML)
// Builder: Objeto → XML
const builder = new xml2js.Builder({
rootName: 'biblioteca',
headless: true
});
 
const xml = builder.buildObject({ livro: livros });
