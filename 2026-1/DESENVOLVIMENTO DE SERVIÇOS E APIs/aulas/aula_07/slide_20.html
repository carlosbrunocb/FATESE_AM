Passo 1 - Instalação da biblioteca xml2js
Configuração inicial do projeto para manipulação de XML em Node.js
Aula 07
Laboratório Prático
Comandos de Instalação
1
Instalar xml2js
Execute no terminal do projeto da Livraria
$ npm install xml2js
2
Verificar instalação
Confirme que a biblioteca foi instalada corretamente
$ npm list xml2js
3
Testar importação
Verifique se o módulo carrega sem erros
$ node -e "console.log(!!require('xml2js'))"
Importação no Código
// Importar xml2js no serviço
const xml2js = require('xml2js');
 
// Criar parser e builder
const parser = new xml2js.Parser();
const builder = new xml2js.Builder();
Dicas de Configuração
Commit package-lock
Commit do package-lock.json para reprodutibilidade
Segurança XXE
Mitigue XXE e expansão de entidades
UTF-8
Defina charset UTF-8 nas respostas
Postman
Use Postman para validar Accept/Content-Type
Pontos de Atenção
Versão Node.js
Use Node.js LTS para compatibilidade
Parsing seguro
Só parseie XML de fontes confiáveis
