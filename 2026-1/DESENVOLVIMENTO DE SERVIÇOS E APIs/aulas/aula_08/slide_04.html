Definição de ORM
ORM (Object-Relational Mapping) é uma técnica de programação que permite mapear as tabelas de um banco de dados relacional para objetos na linguagem de programação. No nosso caso, mapeia tabelas do PostgreSQL/MySQL para objetos JavaScript.
Como Funciona
Cada tabela vira uma classe (Model), cada linha vira uma instância, e as colunas viram propriedades do objeto.
Benefício Central
Abstrai o acesso a dados, padroniza operações CRUD e reduz código repetitivo. O desenvolvedor interage com objetos em vez de escrever SQL diretamente.
