Aula 08
Persistência SQL
Definição
Sequelize é um ORM para Node.js que mapeia tabelas relacionais para Models JavaScript e abstrai operações SQL em métodos de alto nível. Suporta PostgreSQL, MySQL, MariaDB, SQLite e SQL Server.
