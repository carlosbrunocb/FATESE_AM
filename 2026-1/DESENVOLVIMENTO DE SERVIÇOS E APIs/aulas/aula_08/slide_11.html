Models - Representação de Tabelas
Models são classes que definem a estrutura das tabelas, incluindo colunas, tipos de dados, validações e timestamps.
const Livro = sequelize.define('Livro', { titulo: { type: DataTypes.STRING }, autor: { type: DataTypes.STRING }, ano: { type: DataTypes.INTEGER } });

Sync - Sincronização com BD
Sincroniza os modelos com o banco de dados. Opções: force (recria tabelas) e alter (ajusta schema).
// Cria tabelas se não existirem await sequelize.sync({ force: false }); // Recria tabelas (cuidado: apaga dados) await sequelize.sync({ force: true });
Migrations - Controle de Versão
Scripts versionados para criar/alterar tabelas com histórico e rollback. Permitem deploy seguro em múltiplos ambientes.
// Migration para criar tabela module.exports = { up: async (queryInterface, Sequelize) => { await queryInterface.createTable('livros', { id: { type: Sequelize.INTEGER, primaryKey: true } }); } };
Assincronicidade - Async/Await
Todas as operações de I/O retornam Promises. Use async/await para aguardar resultados do banco de dados.
const listarTodos = async () => { return await Livro.findAll(); }; const criar = async (dados) => { return await Livro.create(dados); };
