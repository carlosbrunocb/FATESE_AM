Criando o Banco de Dados no PostgreSQL

Um passo essencial antes de conectar a aplicação via Sequelize

Status SGBD
PostgreSQL Ativo
Atenção

O Sequelize é um ORM excelente para criar tabelas e gerenciar dados, mas ele NÃO cria o banco de dados (database) automaticamente por padrão.

Você precisa criar o contêiner do banco manualmente no seu SGBD antes de rodar a aplicação Node.js.

Ferramentas para executar o SQL
  • pgAdmin 4
    Interface gráfica oficial do PostgreSQL. (Query Tool)
  • psql
    Ferramenta de linha de comando.
  • DBeaver / DataGrip
    Clientes universais de banco de dados.
Script DDL (Data Definition Language)
SQL
-- 1. Conecte-se ao seu servidor PostgreSQL
-- 2. Abra uma nova janela de Query
-- 3. Execute o comando abaixo:

CREATE DATABASE livraria_db
    WITH
    OWNER = postgres
    ENCODING = 'UTF8'
    CONNECTION LIMIT = -1;

-- Se quiser apenas o comando mais simples:
CREATE DATABASE livraria_db;

Atenção ao arquivo .env

O nome que você colocar no comando CREATE DATABASE deve ser exatamente o mesmo que você configurou na variável DB_NAME no arquivo .env na etapa anterior.
