Dicas Importantes
Use DataTypes.STRING(255) para limitar o tamanho e DataTypes.INTEGER para campos numéricos inteiros.
Exemplo de Model com DataTypes
const Livro = sequelize.define('Livro', {
id: {
type: DataTypes.INTEGER,
autoIncrement: true,
primaryKey: true
},
titulo: {
type: DataTypes.STRING(255),
allowNull: false
},
ano: {
type: DataTypes.INTEGER,
validate: { min: 0 }
}
}, { tableName: 'livros', timestamps: true });
Validações Disponíveis
allowNull unique defaultValue validate primaryKey autoIncrement
