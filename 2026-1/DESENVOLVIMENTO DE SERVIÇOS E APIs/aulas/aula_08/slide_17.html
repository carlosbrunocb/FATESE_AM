Sincronização com Banco: sequelize.sync()
Criação/atualização automática de tabelas conforme os Models
3 opções
Sync, Force, Alter
Bootstrap com Sincronização
Código prático de inicialização
1 const sequelize = require('./config/database');
2
3 (async () => {
4 try {
5 await sequelize.authenticate();
6 console.log('DB conectado.');
7
8 // Básico (dev) - cria tabelas se não existirem
9 await sequelize.sync();
10
11 // Recria tabelas (APAGA DADOS) - somente em dev!
12 // await sequelize.sync({ force: true });
13
14 // Tenta ajustar schema sem perder dados
15 // await sequelize.sync({ alter: true });
16 } catch (err) {
17 console.error('Erro:', err);
18 }
19 })();
Atenção: Nunca use force: true em produção! Ele recria as tabelas e apaga todos os dados.
Opções de Sync
sync(): Cria tabelas se não existirem
sync({ force: true }): Recria tabelas (apaga dados) DEV
sync({ alter: true }): Ajusta schema sem perder dados
Boas Práticas
• Use sync() em desenvolvimento
• Use migrations em produção
• Desative logging em produção
• Monitore pool de conexões
