2 const livros = await Livro.findAll({

4 order: [['createdAt', 'DESC']],

7

12
13 // 3. Buscar por PK
14 const livro = await Livro.findByPk(1);
15
16 // 4. Atualizar (via instância)
17 if (livro) await livro.update({ ano: 1952 });
18
19 // 5. Deletar (por condição)
20 await Livro.destroy({ where: { id: 3 } });
Dicas Práticas
Filtragem: Use where com operadores (Op.like, Op.gt) para consultas complexas
Ordenação: Combine order com limit/offset para paginação
Update: Prefira update via instância quando já tiver o registro
Delete: Use destroy com where para deletar múltiplos registros
