Script SQL Completo
PostgreSQL - Criação de banco, tabela e seed
1 -- 1) Criar o banco (rode conectado ao servidor Postgres)
2 CREATE DATABASE livraria_db;
3
4 -- 2) Conectar ao banco (no psql: \c livraria_db)
5 USE livraria_db;
6
7 -- 3) Criar a tabela
8 CREATE TABLE IF NOT EXISTS livros (
9 id SERIAL PRIMARY KEY,
10 titulo VARCHAR(255) NOT NULL,
11 autor VARCHAR(255) NOT NULL,
12 ano INTEGER NOT NULL,
13 "createdAt" TIMESTAMP WITH TIME ZONE DEFAULT CURRENT_TIMESTAMP,
14 "updatedAt" TIMESTAMP WITH TIME ZONE DEFAULT CURRENT_TIMESTAMP
15 );
16
17 -- 4) Inserir dados de exemplo (Seed)
18 INSERT INTO livros (titulo, autor, ano, "createdAt", "updatedAt") VALUES
19 ('O Senhor dos Anéis', 'J.R.R. Tolkien', 1954, NOW(), NOW()),
20 ('1984', 'George Orwell', 1949, NOW(), NOW()),
21 ('Dom Quixote', 'Miguel de Cervantes', 1605, NOW(), NOW());
Dica: Use NOW() para timestamps automáticos. As aspas duplas em "createdAt" são necessárias no PostgreSQL para camelCase.
Passos para Executar
Ordem recomendada
1
Criar banco: Execute CREATE DATABASE no psql ou pgAdmin
2
Conectar: Use \c livraria_db ou selecione o banco
3
Criar tabela: Execute o comando CREATE TABLE
4
Inserir dados: Execute os comandos INSERT INTO
Nota: O Sequelize também pode criar a tabela automaticamente com sequelize.sync()
