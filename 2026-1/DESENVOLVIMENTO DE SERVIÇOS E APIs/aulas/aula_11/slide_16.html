Desacoplamento e manutenibilidade
Antes da Refatoração
MVC tradicional com acoplamento
// Controller com lógica de negócio
const criarLivro = async (req, res) => {
// Controller gerencia tudo
const livro = await LivroModel.create(req.body);
// Validação inline
if (livro.ano > 2024) {
throw new Error("Ano inválido");
}
};
Controller obeso com múltiplas responsabilidades
Alto acoplamento entre camadas
Após a Refatoração
Arquitetura Service-Repository
// Controller delega para Service
const criarLivro = async (req, res) => {
try {
const livro = await livroService.criar(req.body);
res.status(201).json(livro);
} catch (error) {
res.status(400).json({ erro: error.message });
}
};
Controller limpo e focado em HTTP
Baixo acoplamento entre camadas
