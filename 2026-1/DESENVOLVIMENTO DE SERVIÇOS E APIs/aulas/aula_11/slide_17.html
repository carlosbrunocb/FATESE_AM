Cenário
A nossa livraria agora faz vendas! Precisamos de um endpoint onde o cliente informa o ID do livro que deseja comprar. A API deve verificar se há estoque disponível. Se houver, a venda é concluída e o estoque diminui. Se não houver, a API deve retornar um erro avisando que o livro está esgotado.
