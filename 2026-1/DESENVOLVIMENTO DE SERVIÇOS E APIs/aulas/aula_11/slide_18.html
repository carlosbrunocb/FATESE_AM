Função atualizarEstoque
// Função para atualizar o estoque de um livro
// Parâmetros: id (do livro), novaQuantidade
const atualizarEstoque = async (id, novaQuantidade) => {
try {
// Verifica se o livro existe
const livro = await LivroModel.findById(id);
if (!livro) {
throw new Error("Livro não encontrado");
}
// Atualiza o estoque
return await LivroModel.findByIdAndUpdate(
id,
{ estoque: novaQuantidade },
{ new: true }
);
} catch (error) {
throw new Error(`Erro: ${error.message}`);
}
};
Explicação dos Parâmetros
id: String/ObjectId
→ Identificador único do livro
→ Ex: "507f1f77bcf86cd799439011"
novaQuantidade: Number
→ Nova quantidade em estoque
→ Deve ser >= 0
Retorno: Promise<Object>
→ Documento atualizado
Fluxo de Atualização de Estoque
A função atualizarEstoque recebe o ID do livro e a nova quantidade, verifica se o livro existe, atualiza o campo "estoque" e retorna o documento atualizado. Se o livro não existir, lança um erro.
