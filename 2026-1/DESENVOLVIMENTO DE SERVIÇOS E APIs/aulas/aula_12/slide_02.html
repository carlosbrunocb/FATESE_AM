Contexto da Aula
API da Livraria

Nesta aula, continuamos a evolução da API da Livraria com foco em segurança de autenticação. Implementaremos autenticação stateless usando JWT (JSON Web Tokens) para proteger as rotas sensíveis do sistema.

Segurança
Implementação de autenticação segura com hashing de senhas e tokens JWT
Autenticação
Fluxo completo de login/registro com validação de credenciais
Proteção
Middleware de proteção para rotas sensíveis da API
Persistência
Armazenamento seguro de dados de usuário
