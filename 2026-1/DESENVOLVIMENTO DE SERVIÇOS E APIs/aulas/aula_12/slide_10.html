O que é JWT (JSON Web Token)?

Padrão RFC 7519 para transmissão segura de informações

Autenticação Stateless
Definição e Características
O JWT (JSON Web Token) é um padrão aberto (RFC 7519) que define uma maneira compacta e autossuficiente para transmitir informações de forma segura entre as partes como um objeto JSON.
Header:
{ "alg": "HS256", "typ": "JWT" }
Payload:
{ "sub": "123", "name": "John Doe" }
Signature:
HMACSHA256(header.payload, secret)
Autenticação Stateless - Não requer armazenamento no servidor
Vantagens do JWT
• Compacto: Transmissão rápida via URL, POST ou header HTTP
• Autossuficiente: Contém todas as informações necessárias
• Seguro: Assinatura digital garante integridade
