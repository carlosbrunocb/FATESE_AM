O que é Bcrypt?
Bcrypt é um algoritmo de hashing de senhas que aplica um salt aleatório e um fator de custo (rounds) para tornar o processo de hashing mais lento e resistente a ataques de força bruta.
// Instalação
npm install bcryptjs

// Hashing de senha
const hash = await bcrypt.hash(password, 10);

Tempo de Processamento
