Testando no Postman
Passo a passo para testar a API de autenticação JWT
Pronto para testar
Fluxo de Teste
1
Tentar acessar rota protegida sem token
Faça uma requisição POST para /api/livros sem enviar o token JWT. O servidor deve retornar erro 401.
POST /api/livros
2
Registrar novo usuário
Envie uma requisição POST para /api/auth/registrar com nome, email e senha.
POST /api/auth/registrar
3
Realizar login e obter token
Faça login com o usuário criado e copie o token JWT da resposta.
POST /api/auth/login
4
Configurar token no Postman
Vá para a aba Authorization, selecione Bearer Token e cole o token obtido.
Authorization: Bearer <token>
