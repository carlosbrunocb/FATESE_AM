Conceito Fundamental
O que é Autenticação?
Definição
Autenticação é o processo de verificar a identidade de um usuário através de credenciais válidas. É a primeira camada de segurança que garante que apenas usuários autorizados acessem o sistema.
Identidade: Verifica quem é o usuário
Credenciais: Email, senha, biometria
Token: JWT para sessão
Expiração: Tempo limitado de acesso
