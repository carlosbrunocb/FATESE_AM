RBAC - Definição e Benefícios

Controle de Acesso Baseado em Papéis (Role-Based Access Control)
Duração: 40 min
O que é RBAC?

O Controle de Acesso Baseado em Papéis é um método de restrição de rede baseado nos cargos dos usuários individuais dentro de uma organização.

Papel (Role): Uma coleção de permissões
Usuário: Entidade autenticada que recebe um ou mais papéis
Benefícios do RBAC
  • Simplicidade: Gerenciamento centralizado de permissões
  • Manutenção: Fácil atualização de acessos
  • Governança: Auditoria e compliance facilitados
  • Segurança: Princípio do menor privilégio
Estrutura de Papéis
ADMIN
Gerencia catálogo, usuários e configurações
CLIENTE
Visualiza catálogo e realiza compras
GERENTE
Supervisiona operações e relatórios
Exemplo de Implementação
// Middleware de autorização
const autorizaRole = (rolesPermitidas) => {
return (req, res, next) => {
if (!rolesPermitidas.includes(req.user.role)) {
return res.status(403).json({
erro: "Acesso negado"
});
}
next();
};
};
