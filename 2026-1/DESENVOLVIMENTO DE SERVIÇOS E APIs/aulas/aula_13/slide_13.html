Vulnerabilidade: XSS em Contexto de API

Cross-Site Scripting em APIs REST e mitigações
Alta Severidade
Vetor de Ataque
XSS em APIs
APIs geralmente retornam JSON, mas XSS pode surgir em painéis administrativos ou clients que renderizam HTML. O atacante injeta scripts maliciosos em dados persistidos.
Severidade: Alta
Sintomas Comuns
Scripts executados no browser
Redirecionamentos maliciosos
Roubo de cookies/session
Modificação de DOM
Mitigações
Proteções Recomendadas
1. Escapar/Encode: Sanitizar dados em UIs consumidoras
2. Content-Type: Definir headers corretos (application/json)
3. Sanitização: Limpar campos antes de persistir
4. CSP: Content Security Policy para restringir execução
// Exemplo de sanitização de input
const sanitized = input.replace(/</g, '&lt;')
.replace(/>/g, '&gt;');
