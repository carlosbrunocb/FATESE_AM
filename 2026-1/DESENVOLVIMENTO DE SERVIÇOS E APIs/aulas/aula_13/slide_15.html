O que é Helmet?

Helmet é um middleware para Express.js que configura automaticamente cabeçalhos HTTP de segurança para proteger aplicações contra vulnerabilidades comuns.

  • Remove cabeçalho X-Powered-By
  • Adiciona HSTS (HTTP Strict Transport Security)
  • Configura X-Frame-Options
  • Ativa XSS Protection
Implementação Básica
const helmet = require('helmet');
const express = require('express');
const app = express();
// Aplica todas as proteções do Helmet
app.use(helmet());
