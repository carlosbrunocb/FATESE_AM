Exemplo - Rate Limiter (express-rate-limit)

Implementação de limitação de requisições para proteção contra ataques
Proteção Ativa
Configuração do Rate Limiter
// Instalação: npm install express-rate-limit
const rateLimit = require('express-rate-limit');
// Configuração do limitador
const limiter = rateLimit({
windowMs: 15 * 60 * 1000, // 15 minutos
max: 100, // limite de 100 requisições
message: {
erro: "Muitas requisições deste IP. Tente novamente mais tarde."
},
standardHeaders: true,
legacyHeaders: false,
});
// Aplicação do middleware
app.use('/api/', limiter);
Importante
O rate limiter deve ser aplicado antes das rotas protegidas para garantir a segurança.
Configurações do Rate Limiter
Window: 15 minutos
Max Requests: 100 requisições
Mensagem: Limite excedido
Proteção: Ativa
