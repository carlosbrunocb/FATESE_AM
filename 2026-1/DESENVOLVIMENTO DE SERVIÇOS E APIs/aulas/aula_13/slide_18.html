Modelo de Usuário com Papel (Mongoose)

Evolução do modelo de usuário para incluir controle de acesso baseado em papéis
RBAC Implementado
src/models/Usuario.js
const mongoose = require('mongoose');
const UsuarioSchema = new mongoose.Schema({
nome: { type: 'String', required: true},
email: { type: 'String', required: true, unique: true},
senha: { type: 'String', required: true},
// NOVO CAMPO: Papel do usuário
role: {
type: 'String',
enum: ['CLIENTE', 'ADMIN'], // Só aceita esses dois valores
default: 'CLIENTE' // Todo novo usuário é cliente por padrão
}
}, { timestamps: true });
module.exports = mongoose.model('Usuario', UsuarioSchema);
Importante
O campo role utiliza enum para garantir que apenas valores válidos sejam aceitos no banco de dados.
