autenticaJWT

200 OK - Sucesso

Cadeia de Middlewares
// Configuração das rotas com middlewares
router.post(
'/livros',
autenticaJWT, // 1. Valida token
autorizaRole(['ADMIN']), // 2. Verifica papel
validaLivro, // 3. Valida dados
controller.criarLivro // 4. Executa
);
