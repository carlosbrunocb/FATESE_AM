Integração nas Rotas (RBAC em Ação)

Implementação de controle de acesso baseado em papéis nas rotas da API
RBAC Ativo
Configuração das Rotas com RBAC
const express = require('express');
const router = express.Router();
// Importação dos middlewares
const autenticaJWT = require('../middlewares/autenticaJWT');
const autorizaRole = require('../middlewares/autorizaRole');
// Rota Pública: Qualquer um pode listar
router.get('/livros', controller.listarLivros);
// Rota de Cliente: Precisa estar logado para comprar
router.post(
'/livros/:id/comprar',
autenticaJWT,
autorizaRole(['CLIENTE', 'ADMIN']),
controller.comprarLivro
);
// Rotas Administrativas: Somente ADMIN
router.post(
'/livros',
autenticaJWT,
autorizaRole(['ADMIN']),
validaLivro,
controller.criarLivro
);
Importante
A ordem dos middlewares é crucial: autenticaJWT antes de autorizaRole
Níveis de Acesso
ADMIN: Gerenciar catálogo completo
CLIENTE: Visualizar e comprar
PÚBLICO: Listar livros
