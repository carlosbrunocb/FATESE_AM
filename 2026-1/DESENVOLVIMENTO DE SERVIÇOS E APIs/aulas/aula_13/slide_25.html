Checklist de Testes de Autorização

Cenários essenciais para validar o controle de acesso RBAC
Testes em execução
Cenário 1: CLIENTE tenta DELETE
Usuário: CLIENTE
403 Forbidden
Tentativa de deletar um livro sendo um usuário com papel CLIENTE. Deve ser bloqueado.
1
Login como CLIENTE
2
Obter token JWT
3
Tentar DELETE /api/livros/1
4
Verificar resposta 403
Resultado Esperado
Status 403 Forbidden com mensagem "Acesso Negado. Seu perfil não tem permissão."
Cenário 2: ADMIN com Sucesso
Usuário: ADMIN
200 OK
Tentativa de deletar um livro sendo um usuário com papel ADMIN. Deve ser permitido.
1
Login como ADMIN
2
Obter token JWT com role ADMIN
3
Executar DELETE /api/livros/1
4
Verificar resposta 200/204
Resultado Esperado
Status 200 OK ou 204 No Content com livro deletado.
