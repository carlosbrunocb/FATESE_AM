OpenAPI Specification (OAS)
Padrão aberto para descrever APIs REST em JSON/YAML
Padrão Aberto
Especificação independente de linguagem e framework para descrever APIs RESTful
Formato JSON/YAML
Descrição em formato JSON ou YAML, agnóstica a linguagens de programação
Suporte Completo
Suporta descrição de endpoints, schemas, segurança, servers e mais
Características Principais
Endpoints HTTP (GET, POST, PUT, DELETE)
Schemas de requisição e resposta
Parâmetros de query, path, header e body
Códigos de status HTTP
Esquemas de segurança (OAuth, JWT, API Key)
Exemplo de Estrutura
"openapi": "3.0.0",
"info": {
"title": "API Livraria",
"version": "1.0.0"
},
"paths": {
"/livros": { ... }
}
