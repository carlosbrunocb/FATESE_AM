Query Parameters: Conceito e Uso
Parâmetros de busca para filtrar, ordenar e paginar coleções de recursos
O que são Query Parameters?
Query Parameters são pares chave-valor que aparecem após o ? na URL. Eles são usados para filtrar, ordenar ou paginar uma listagem de recursos, diferindo dos Path Parameters que identificam recursos específicos.
/api/livros?titulo=Node&pagina=1&limite=20
Endpoint: /api/livros
Filtro: titulo=Node
Paginação: pagina=1
Limite: limite=20
Uso Comum de Query Parameters
Filtragem por campos
Ordenação de resultados
Paginação
Busca parcial
Exemplo Prático
Implementação de busca por título com filtro case-insensitive:
const buscarTodos = async (titulo) => {
  let filtro = {};
  if (titulo) {
    filtro.titulo = { $regex: titulo, $options: 'i' };
  }
  return await LivroModel.find(filtro);
};
Parâmetros Disponíveis
titulo Filtra por título do livro string
autor Filtra por nome do autor string
ano Filtra por ano de publicação number
pagina Número da página number
limite Itens por página number
