/**
* @openapi
* /livros:
* post:
* summary: Cria um novo livro no catálogo
* description: Rota protegida. Apenas usuários com papel 'ADMIN' podem cadastrar novos títulos.
* tags:
* - Livros
* security:
* - BearerAuth: []
* requestBody:
* required: true
* content:
* application/json:
* schema:
* type: object
* required: [titulo, autor, ano]
* properties:
* titulo:
* type: string
* example: "Node.js Prático"
* autor:
* type: string
* example: "John Doe"
* ano:
* type: number
* example: 2024
* estoque:
* type: number
* example: 15
* responses:
* 201:
* description: Livro criado com sucesso
* 401:
* description: Não autenticado. Token ausente ou inválido.
* 403:
* description: Não autorizado. Apenas administradores possuem acesso.
* 400:
* description: Erro na validação dos dados enviados.
*/
router.post('/livros', autenticaJWT, autorizaRole(['ADMIN']), validaLivro, controller.criarLivro);
