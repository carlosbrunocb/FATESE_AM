Configurar rotas versionadas

const express = require('express');
const rotasV1 = require('./routes/v1/livroRoutes');
const rotasV2 = require('./routes/v2/livroRoutes');
const errorHandler = require('./middlewares/errorHandler');

const app = express();
app.use(express.json());

// Injeção de Rotas Versionadas
app.use('/api/v1', rotasV1); // Bloco legado
app.use('/api/v2', rotasV2); // Bloco novo

app.use(errorHandler);

module.exports = app;
Fluxo de Rotas
Como as requisições são direcionadas
/api/v1
Versão 1
/api/v2
Versão 2
/livros
Endpoint
Como funciona?
O app.use direciona todas as requisições que começam com /api/v1 para o controlador V1 e /api/v2 para o V2.
Middleware de erro: O errorHandler captura erros de todas as rotas e retorna respostas padronizadas.
