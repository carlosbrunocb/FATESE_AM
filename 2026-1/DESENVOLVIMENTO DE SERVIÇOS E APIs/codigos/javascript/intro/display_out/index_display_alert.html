Demonstração da Função alert()

A função alert() exibe uma caixa de diálogo com uma mensagem e um botão de "OK". É muito útil para notificar o usuário sobre algo importante.

Exemplo Básico

Clique no botão abaixo para ver um alerta simples. A função será chamada no carregamento da página.

Exemplo Prático

Clique neste botão para ver um alerta que utiliza uma variável para criar uma mensagem personalizada.
