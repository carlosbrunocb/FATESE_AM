Exemplos de Funções JavaScript

Função Declarativa

Este é o tipo de função mais tradicional, declarada com a palavra-chave function. Ela é "elevada" (hoisted), o que significa que pode ser chamada antes de ser declarada no código.

Função Anônima (Expressão de Função)

Uma função anônima não tem nome e é atribuída a uma variável ou constante. Ela não é elevada (hoisted), então deve ser declarada antes de ser utilizada.
