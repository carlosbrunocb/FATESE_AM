Estrutura do Código no Arduino

setup()

Executa uma única vez ao iniciar o programa. Usado para configurações iniciais: definir modos de pinos, inicializar comunicação serial e configurar periféricos.

loop()

Executa continuamente após o setup(). Contém a lógica principal do programa que se repete indefinidamente enquanto o dispositivo estiver ligado.

Exemplo Mínimo — Piscar LED

                
const int LED = 2;


void setup() {
    pinMode(LED, OUTPUT);
    Serial.begin(115200);
}


void loop() {
    Serial.println("Hello World ESP32!");
    digitalWrite(LED, HIGH);
    delay(500);
    digitalWrite(LED, LOW);
    delay(500);
}

Dica: O GPIO 2 é comumente usado para LED integrado em muitas placas ESP32. Sempre use resistor em série (220-330Ω) ao conectar LEDs externos. Verifique o pinout específico da sua placa.
