Código Completo — Blink + Serial

Controle de LED

O código alterna o estado do LED conectado ao GPIO 2, ligando-o por 500ms e desligando-o por 500ms, criando o efeito de piscar.

Monitor Serial

Mensagens são enviadas via Serial a 115200 baud para monitorar o estado do LED em tempo real, facilitando debugging e validação.

Exemplo Completo — ESP32 Blink com Saída Serial

// Define o pino do LED (GPIO 2 é comum em ESP32)
const int LED = 2;


void setup() {
    // Configura o pino como saída
    pinMode(LED, OUTPUT);
    // Inicia comunicação serial a 115200 baud
    Serial.begin(115200);
    Serial.println("ESP32 Iniciando...");
}


void loop() {
    digitalWrite(LED, HIGH); // Liga o LED
    Serial.println("LED ON");
    delay(500);
    digitalWrite(LED, LOW); // Desliga o LED
    Serial.println("LED OFF");
    delay(500);
}

Observações importantes: Ajuste o pino conforme seu hardware específico. Sempre use resistor em série (220-330Ω) ao conectar LEDs externos. No Wokwi, adicione o resistor no diagrama entre o GPIO e o anodo do LED. Pinos strapping (0, 2, 5, 12, 15) devem ser usados com atenção durante o boot.
