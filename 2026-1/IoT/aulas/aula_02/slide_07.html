Montagem de LED com Resistor (Wokwi)

Atividade Prática - Laboratório

1

Adicione 3 LEDs (verde, amarelo, vermelho) e 3 resistores (220 Ω)

No Wokwi, clique no botão "+" para adicionar componentes

2

Conexões sugeridas: LED Verde → GPIO 19; LED Amarelo → GPIO 18; LED Vermelho → GPIO 5

Conecte o anodo (perna longa) de cada LED ao GPIO correspondente via resistor

3

Ligue o outro terminal de cada resistor ao GND

Use o pino GND do ESP32 para fechar o circuito

4

Anodo (perna longa) no GPIO via resistor; catodo (curta) ao GND

Dica: rotule os fios e valide a polaridade antes de simular
