Padrão de Temporização com millis()

Variáveis necessárias: unsigned long tempoAnterior; e const long intervalo;

No loop: unsigned long agora = millis();
Depois: if (agora - tempoAnterior >= intervalo) { tempoAnterior = agora; ação(); }

Cuidado com overflow: a subtração unsigned lida corretamente com a volta do contador após ~49,7 dias

Vantagem: o loop continua rodando livremente, permitindo multitarefa cooperativa no ESP32
