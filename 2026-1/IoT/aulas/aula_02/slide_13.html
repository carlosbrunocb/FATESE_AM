Boas Práticas de Código (Clean Code Embarcado)

Use const para pinos e tempos; adote nomes claros e descritivos

Separe lógica (estados) de I/O (digitalWrite) para facilitar manutenção

Evite delay() no loop principal; prefira millis() para temporização

Comente intenções e não apenas o óbvio; modularize quando o código crescer
