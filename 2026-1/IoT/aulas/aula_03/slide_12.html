Debounce por Software (simples)

Pseudocódigo de Debounce

1

if (leitura == LOW) {

2

  delay(50); // aguarda 50ms

3

4

  if (leitura == LOW) {

5

    // confirma o estado

6

    acionaEvento();

7

  }

8

}

Estratégia

Ao detectar mudança, aguardar curto período e confirmar estado.

Janela Típica

20–50 ms (usaremos 50 ms neste projeto).

Vantagens

Fácil implementação e suficiente para botões comuns. Elimina leituras falsas causadas por ruído mecânico.

Observação

Usa delay() para simplificar, mas bloqueia o processador durante a espera.
