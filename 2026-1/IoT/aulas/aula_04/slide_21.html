Leitura do LDR - Exemplo

Monitoramento de luminosidade com conversão de dados para escala percentual.

leitura_ldr.ino

// Pino onde está o nó do divisor de tensão
const int pinoLDR = 32;

void setup() {
  Serial.begin(115200);
  pinMode(pinoLDR, INPUT);
}

void loop() {
  // 1. Leitura bruta (0 a 4095)
  int leitura = analogRead(pinoLDR);

  // 2. Conversão para porcentagem (aprox.)
  // Assumindo montagem: +Luz -> +Tensão
  float porcentagem = (leitura / 4095.0) * 100.0;
  
  Serial.print("LDR Bruto: ");
  Serial.print(leitura);
  Serial.print(" | Luz: ");
  Serial.print(porcentagem);
  Serial.println("%");
  
  delay(500);
}

O Divisor de Tensão

O LDR varia resistência, não voltagem. É necessário um resistor fixo (10kΩ) em série para criar um nó de tensão mensurável no pino 32.

Normalização de Dados

Transformamos a escala abstrata (0-4095) em algo legível (0-100%) usando regra de três. A leitura analogRead retorna int, então divida por 4095.0 (float).

Comportamento da Luz

Na configuração LDR no VCC e Resistor no GND:
• Mais Luz = Menor Resistência LDR = Maior Tensão (Valor perto de 4095).
• Escuro = Maior Resistência LDR = Menor Tensão (Valor perto de 0).
