Como o PWM entra aqui?

Para atuadores visuais, PWM é o método padrão para dimming. Na prática de hoje, isso aparece no efeito de “LED respirando”.

duty cycle → brilho

Dica: comece com LED para entender PWM e só depois evolua para cargas maiores (fitas LED, lâmpadas, motores).
