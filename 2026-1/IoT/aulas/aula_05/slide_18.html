Motor DC vs
Servomotor

Como escolher o atuador certo?

A decisão depende do que você precisa controlar: velocidade (giro contínuo) ou posição (ângulo exato). O PWM é usado nos dois casos, mas com significados diferentes.

PWM no DC

Duty ↑ → mais potência → mais velocidade (com driver).

PWM no Servo

Pulso (1–2 ms) → ângulo (período ~20 ms).

Regra de ouro

Precisa posição? Use servo. Precisa giro contínuo? DC.

Exemplo em IoT

Ventilador/Exaustor: motor DC (controle de velocidade). Tranca/Cancela: servomotor (controle de posição).

Comparativo rápido

Principais diferenças para escolher o atuador no projeto.

Motor DC

Giro contínuo

Movimento

Roda “para sempre” (enquanto alimentado).

Controle típico

Velocidade/torque via duty cycle (PWM).

Feedback

Em geral, não tem posição absoluta (sem sensor extra).

Em hardware real, use driver/MOSFET/ponte H para corrente.

Servomotor

Posição específica

Movimento

Vai até um ângulo (ex.: 0°–180°).

Controle típico

Largura do pulso (≈ 1–2 ms) em 50 Hz.

Feedback

Possui controle interno (sensor/potenciômetro).

Pode exigir fonte dedicada. Sempre use terra comum.

Resumo para memorizar

Motor DC = “controle de potência/velocidade”. Servo = “controle de posição por pulso”.

DC: PWM (duty) → velocidade | Servo: PWM (pulso 1–2 ms @ 50 Hz) → ângulo

Conexão com a prática: nesta aula, o LED usa PWM por duty cycle e o servo usa PWM por largura de pulso.
