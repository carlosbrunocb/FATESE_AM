Fundamentos Teóricos

I2C — Endereçamento

Endereços de 7 Bits

O padrão mais comum utiliza 7 bits para endereço. O byte transmitido contém: [Addr(7 bits)] + [R/W(1 bit)]. Isso permite até 128 dispositivos teóricos (na prática menos, devido a endereços reservados).

Exemplo: Display OLED SSD1306

A maioria dos módulos OLED 0.96" vem configurada de fábrica com o endereço hexadecimal 0x3C. Alguns permitem alteração para 0x3D via jumper na placa traseira.

Evitando Conflitos (Scanner I2C)

Se conectar múltiplos sensores, seus endereços devem ser únicos. Use um sketch "I2C Scanner" para listar todos os dispositivos detectados no barramento antes de iniciar o projeto.

Hardware: Pull-ups e Cabos

Para que o endereçamento funcione sem erros, garanta resistores de pull-up adequados (4.7kΩ ou 10kΩ) nas linhas SDA/SCL e use cabos curtos para evitar capacitância parasita.
