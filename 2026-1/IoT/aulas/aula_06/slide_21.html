Bibliotecas (Libraries)

Importamos Wire.h para comunicação I2C, as bibliotecas da Adafruit para desenhar gráficos e controlar o driver SSD1306, e a biblioteca padronizada para sensores DHT.

Objetos Globais

Instanciamos os objetos display e dht. Eles servem como nossa "interface" de software para controlar o hardware físico conectado aos pinos.

Multitarefa (Millis)

Definimos variáveis para controlar o tempo. O DHT22 é lento e precisa de pelo menos 2 segundos entre leituras. Usar millis() evita congelar o processador com delay().
