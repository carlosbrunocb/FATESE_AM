Setup e Inicialização

main.cpp
20void setup() {
21 Serial.begin(115200);
22
23 // Inicializa o Sensor
24 dht.begin();
25
26 // Inicializa o Display OLED (Endereço I2C padrão: 0x3C)
27 if(!display.begin(SSD1306_SWITCHCAPVCC, 0x3C)) {
28 Serial.println("Falha ao inicializar o OLED!");
29 for(;;); // Trava o sistema se o display falhar
30 }
31
32 display.clearDisplay();
33 display.setTextSize(1);
34 display.setTextColor(WHITE);
35 display.setCursor(0, 10);
36 display.println("Sistema Iniciado!");
37 display.display();
38 delay(2000);
39}

Inicialização Básica

Iniciamos a comunicação Serial para diagnósticos e ativamos o sensor DHT. O comando dht.begin() prepara o barramento single-wire para leitura.

Validação Crítica

Tentamos iniciar o OLED no endereço 0x3C. Se o método retornar false (falha), entramos em um loop infinito for(;;). Isso impede que o código continue rodando com hardware defeituoso.

Feedback Visual

Configuramos tamanho e cor do texto. O comando display.display() é essencial: ele envia os dados da memória (buffer) para a tela física. Sem ele, nada aparece.
