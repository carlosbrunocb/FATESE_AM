Temporização (Millis)

Utilizamos millis() para capturar o tempo atual. Subtraímos o tempo da última leitura para verificar se o intervalo de 2 segundos já passou, permitindo que o processador faça outras tarefas enquanto espera.

Leitura dos Sensores

Os métodos readHumidity() e readTemperature() solicitam os dados ao sensor DHT22. O sensor pode levar até 250ms para responder, por isso o intervalo entre leituras é crucial.

Tratamento de Erros

A função isnan() (Is Not a Number) verifica se a leitura retornou um valor válido. Se o sensor estiver desconectado ou com defeito, evitamos o processamento de dados corrompidos com o return.
