Arquitetura dual-core, conceitos de kernel e implementação de alarme

O que é FreeRTOS?

É um Sistema Operacional de Tempo Real (RTOS) que permite ao ESP32 gerenciar múltiplas tarefas simultaneamente. Ele oferece um escalonador que decide qual task deve rodar com base em prioridades, garantindo previsibilidade.

ESP32 Dual-Core
Core 0

Core 1
loop() Arduino, Tasks
Task 1: Leitura de sensores
Task 2: Atualização display
Task 3: Controle de sirene
