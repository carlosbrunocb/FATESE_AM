Wokwi: Montagem do Circuito (ESP32 + PIR + Buzzer + LED)

Montagem passo a passo do circuito virtual para o alarme de intrusão

Componentes Necessários
ESP32 DevKit v1
Microcontrolador principal
Sensor PIR
Detecção de movimento
Buzzer
Alerta sonoro
LED Vermelho
Indicador visual
Observações Técnicas
• GPIO 26 e 27 são seguros para uso geral
• Entradas 34-39 são apenas entrada
• Use pull-down para evitar ruído no PIR
Ligações (Passo a Passo)
1
PIR: VCC → 3.3V, GND → GND, OUT → GPIO 27
2
LED: Anodo → GPIO 26 via resistor 220Ω, Catodo → GND
3
Buzzer: Pino + → GPIO 26, Pino - → GND
4
GND: Comum entre todos os componentes
