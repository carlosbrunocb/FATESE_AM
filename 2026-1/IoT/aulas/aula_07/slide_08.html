Código: Alarme de Intrusão - Blocos Comentados

Implementação completa com Setup, ISR, Task e Loop explicados

1. Setup - Configuração Inicial
void setup() { // Inicialização serial Serial.begin(115200); // Configuração dos pinos pinMode(pinoPIR, INPUT); pinMode(pinoAlarme, OUTPUT); // Configura interrupção no GPIO 27 attachInterrupt(digitalPinToInterrupt(pinoPIR), detectouMovimento, RISING); // Criação da Task no FreeRTOS xTaskCreate(tocarSirene, // Função da Task "Task_Sirene", // Nome 10000, // Stack NULL, // Parâmetro 1, // Prioridade &TaskAlarme // Handle ); }
Configuração: Inicializa serial, configura pinos, ativa interrupção no GPIO 27 e cria a Task para sirene
2. ISR - Interrupt Service Routine
// ISR: Executada automaticamente quando PIR detecta movimento void IRAM_ATTR detectouMovimento() { // Apenas altera o estado - rápido e direto intrusaoDetectada = true; }
ISR: Função chamada automaticamente pelo hardware. IRAM_ATTR garante execução na RAM mais rápida
3. Task - Controle da Sirene
void tocarSirene(void *parameter) { for(;;) { // Loop infinito da Task if (intrusaoDetectada) { // Toca alarme intermitente digitalWrite(pinoAlarme, HIGH); vTaskDelay(200 / portTICK_PERIOD_MS); digitalWrite(pinoAlarme, LOW); vTaskDelay(200 / portTICK_PERIOD_MS); } else { digitalWrite(pinoAlarme, LOW); vTaskDelay(100 / portTICK_PERIOD_MS); } } }
Task: Executa em paralelo com o loop. vTaskDelay coopera com o escalonador
4. Loop - Tarefas Pesadas
void loop() { if (intrusaoDetectada) { Serial.println("ALERTA: Intrusão Detectada!"); // Simula envio de dados pesado delay(5000); intrusaoDetectada = false; } delay(10); }
Loop: Tarefas pesadas (envio de dados) sem afetar a sirene graças ao FreeRTOS
