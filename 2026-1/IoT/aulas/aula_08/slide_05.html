Modo Station (STA)
Conectando o ESP32 a uma rede Wi-Fi existente
Cliente Wi-Fi
Conceito
O ESP32 atua como cliente Wi-Fi, associando-se a uma rede existente e obtendo IP via DHCP. É o modo mais comum para IoT, permitindo que o dispositivo envie dados para servidores na nuvem, consumir APIs REST e realizar atualizações OTA.
