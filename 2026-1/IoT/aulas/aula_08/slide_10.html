Parte 1: WiFi Scanner - Conceito
Ensinando o ESP32 a "ouvir" o ambiente e listar redes disponíveis
Escaneamento
Objetivo
O WiFi Scanner é o primeiro passo de qualquer dispositivo inteligente: "ouvir" o ambiente para encontrar redes disponíveis. A função WiFi.scanNetworks() retorna o número de redes encontradas e permite acessar informações como SSID e RSSI.
1
Ativar STA
WiFi.mode(WIFI_STA)
2
Scan
WiFi.scanNetworks()
3
Resultado
SSID + RSSI
int n = WiFi.scanNetworks();
// Retorna: número de redes encontradas
// Para cada rede: WiFi.SSID(i), WiFi.RSSI(i)
Interpretação de RSSI
Excelente
-30 dBm
Bom
-50 a -67 dBm
Fraco
-70 a -80 dBm
Limite
< -85 dBm
Consumo de Energia
O scan consome energia considerável. Para dispositivos a bateria, recomenda-se:
Intervalo adequado: 5s no laboratório, aumente em produção para reduzir consumo
Frequência: Varreduras menos frequentes para poupar bateria
Observações
• O scan bloqueia o loop principal
• Use WiFi.scanNetworks(true) para modo assíncrono
• Evite escanear durante transmissões críticas
• Impacto no throughput e latência
