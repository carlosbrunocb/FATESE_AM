Implementação no ESP32
Dicas práticas: Retorne 200 nas rotas válidas; 404 para rota inexistente; 405 ao bloquear método não suportado; 500 para exceções internas. Use server.send(200, "text/html", content) para respostas bem-sucedidas.
Exemplo de Resposta HTTP no ESP32
void rotaRaiz() { // Responde com status 200 (OK) e envia HTML server.send(200, "text/html", gerarHTML()); } void rotaLigar() { // Liga o hardware e responde digitalWrite(pinoLED, HIGH); server.send(200, "text/html", gerarHTML()); }
