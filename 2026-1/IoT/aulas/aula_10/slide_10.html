Funções de Rota (Handlers)

rotaRaiz, rotaLigar, rotaDesligar - Controle de GPIO via HTTP

ESP32 WebServer
Implementação das Rotas
//--------------------------------------------------------- // Controladores de Rota (O que fazer quando acessam a URL) // --------------------------------------------------------- // Rota raiz - Página inicial void rotaRaiz() { // Responde com status 200 (OK) e envia o HTML server.send(200, "text/html", gerarHTML()); } // Rota para LIGAR o LED void rotaLigar() { digitalWrite(pinoLED, HIGH); // Liga o hardware Serial.println("Comando via Web: LIGAR"); server.send(200, "text/html", gerarHTML()); // Recarrega página } // Rota para DESLIGAR o LED void rotaDesligar() { digitalWrite(pinoLED, LOW); // Desliga o hardware Serial.println("Comando via Web: DESLIGAR"); server.send(200, "text/html", gerarHTML()); } // Rota para erro 404 void rotaNaoEncontrada() { server.send(404, "text/plain", "Página não encontrada"); }
Configuração no Setup
// Configuração das rotas no setup() void setup() { // ... código anterior ... // Define as rotas do servidor server.on("/", HTTP_GET, rotaRaiz); server.on("/ligar", HTTP_GET, rotaLigar); server.on("/desligar", HTTP_GET, rotaDesligar); // Rota para erro 404 server.onNotFound(rotaNaoEncontrada); // Inicia o servidor web server.begin(); Serial.println("Servidor HTTP iniciado."); }
1
Rota Raiz (/)
Página inicial que serve HTML com interface de controle
2
Rota Ligar (/ligar)
Aciona GPIO HIGH, loga ação e retorna página
3
Rota Desligar (/desligar)
Aciona GPIO LOW, loga ação e retorna página
4
Rota Erro 404
Trata URLs não encontradas
5
Boas Práticas
