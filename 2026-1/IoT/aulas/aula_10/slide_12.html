Teste e Demonstração no Wokwi

Validação prática do WebServer ESP32

Pronto para teste
Passos para Execução
1
Clique em Play no Wokwi e aguarde a conexão
O ESP32 vai conectar à rede Wi-Fi "Wokwi-GUEST"
2
Abra o Serial Monitor e anote o IP
Procure por "Acesse o Servidor Web pelo IP: 192.168.x.x"
3
Acesse http://<IP-ESP32>/ no navegador
Use o IP exibido no Serial Monitor
4
Teste as rotas /ligar e /desligar
O LED no GPIO 2 deve mudar de estado
Validação do Sistema
✓
Acesse / → página HTML carregada
Deve exibir a interface de controle
✓
Clique em /ligar → LED acende
GPIO 2 deve receber HIGH
✓
Clique em /desligar → LED apaga
GPIO 2 deve receber LOW
✓
Verifique logs no Serial
Deve aparecer "Comando via Web: LIGAR/DESLIGAR"
Dicas de Simulação
Se o LED não acender, verifique a polaridade
Confira SSID: "Wokwi-GUEST" (sem senha)
Baud rate: 115200 no Serial Monitor
Recarregue a página (CTRL+R) se necessário
