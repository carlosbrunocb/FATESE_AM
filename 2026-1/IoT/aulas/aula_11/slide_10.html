JavaScript Object Notation

O formato padrão de troca de dados para APIs modernas e IoT

O que é JSON?

JSON é um formato leve de troca de dados, puramente textual, fácil de ler e interpretar. Tornou-se o padrão absoluto para APIs modernas.

Características: UTF-8, Independente, Hierárquico.

Benefícios para IoT

Baixo consumo de banda
Parsing ultra rápido
Menor uso de RAM
Integração Simples
