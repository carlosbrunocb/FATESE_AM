  "timezone": "America/Manaus",
  "timezone_abbreviation": "-03",

    "time": "iso8601",
    "interval": "seconds",

  },

    "time": "2026-05-16T12:15",
    "interval": 900,

  }
}
Campos Principais
Valores extraídos do JSON
current.temperature_2m Temp.
28.5°C
current.relative_humidity_2m Umid.
82%
timezone Fuso
America/Manaus
Caminhos de Acesso
Navegação via código
doc["current"]["temperature_2m"] → 28.5
doc["current"]["relative_humidity_2m"] → 82
doc["timezone"] → "America/Manaus"
Dica: A estrutura current_units descreve as unidades de medida retornadas pela API para evitar confusão entre métrico/imperial.
