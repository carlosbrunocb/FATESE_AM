O que é ArduinoJson?
Biblioteca C++ para JSON em dispositivos embarcados
Baixo uso de RAM Parsing seguro Streaming
Biblioteca otimizada para ESP32, AVR e ARM com suporte a Streams para economizar memória e filtros para extrair apenas os dados necessários.

Exemplo Mínimo
Código básico de parsing
// Incluir biblioteca
#include <ArduinoJson.h>

JsonDocument doc;
DeserializationError error = deserializeJson(doc, payload);

if (!error) {
  float temp = doc["current"]["temperature_2m"];
  int umidade = doc["current"]["relative_humidity_2m"];
}
Dica: Use doc.clear() para liberar memória após uso em loops.
