Atividade Prática - App "Consulta de Clima"
Aula 11 | ESP32 + Open-Meteo

Implementação Completa

ESP32 realiza GET na Open-Meteo, faz parsing do JSON e aplica regra de alerta

Descrição do Projeto
Objetivos e funcionalidade
O ESP32 (no Wokwi) realiza uma requisição GET na API Open-Meteo, processa a resposta JSON usando a biblioteca ArduinoJson e aplica uma regra simples: alerta se temperatura > 30°C.
WiFi HTTP JSON
