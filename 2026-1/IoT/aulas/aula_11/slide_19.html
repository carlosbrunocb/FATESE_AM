API Pública de Previsão do Tempo

Sem necessidade de autenticação - Ideal para protótipos IoT

Características
API pública e gratuita
Sem autenticação Latência baixa JSON HTTP/HTTPS
Método: GET • Formato: JSON • Ideal para: Protótipos IoT e projetos educacionais
Endpoint Base
URL da API
http://api.open-meteo.com/v1/forecast
Parâmetros principais: latitude, longitude, current
URL Completa
Para Manaus/AM
http://api.open-meteo.com/v1/forecast?latitude=-3.119&longitude=-60.021&current=temperature_2m,relative_humidity_2m
latitude: -3.119
Manaus
longitude: -60.021
AM
Campos Úteis
Dados retornados
current.temperature_2m
Temperatura atual
current.relative_humidity_2m
Umidade
timezone
Fuso horário
current.time
Horário
Boas Práticas: Respeitar limites de requisições (intervalos com millis), tratar códigos HTTP (200/4xx/5xx) e timeouts
