Passos de Teste
1. HiveMQ Web Client
Acesse o cliente web do HiveMQ para assinar seu tópico e observar as publicações do ESP32 em tempo real.
2. Teste de Atuação
Envie '1' ou '0' no tópico .../led para ligar/desligar o LED do simulador Wokwi remotamente.
3. Troubleshooting
Verifique IDs únicos, estado da conexão (client.state()), QoS 0/1, e mensagens retained.
4. Versionamento
Commit e push do código no GitHub para entrega e acompanhamento do exercício.
Dicas Importantes
  • Use IDs únicos para evitar conflitos no broker público
  • Monitore o estado da conexão MQTT regularmente
  • Configure QoS adequado para cada tipo de mensagem
