Objetivos da Aula
Aprendizados esperados para integração IoT com nuvem
1
Compreender PaaS IoT
Entender o papel das plataformas de Computação em Nuvem (PaaS) na arquitetura de sistemas IoT.
  • Diferença entre IaaS, PaaS e SaaS
  • Serviços especializados para IoT
  • Escalabilidade e disponibilidade
Fundamental
2
Telemetria e Analytics
Assimilar o conceito de Telemetria e a diferença entre ingestão de dados brutos e análise de dados.
  • Coleta automatizada de dados
  • Processamento em tempo real
  • Detecção de anomalias
Prático
3
Segurança Cloud
Compreender mecanismos de autenticação e segurança na nuvem via Tokens de Dispositivos.
  • Device Tokens e API Keys
  • Headers de autenticação
  • Proteção contra acessos não autorizados
Crítico
4
Programação ESP32
Programar o ESP32 para estruturar payloads em JSON e transmitir via HTTP POST.
  • Montagem de payloads JSON
  • Requisições HTTP POST
  • Integração com TagoIO
Prática
Duração: 2 horas (120 minutos)
Método: Aula expositiva + Prática guiada
Entregável: Código funcional no GitHub
