Array de objetos JSON, cada um representando uma variável de telemetria
Campos da Estrutura
variable * string
Nome da variável (ex: temperatura, umidade)
value * number/string
Valor da medida (pode ser numérico ou texto)
unit opcional string
Unidade de medida (ex: °C, %)
time opcional ISO8601
Timestamp do dado (se não enviado, usa hora do servidor)
metadata opcional object
Informações adicionais sobre o dado
location opcional object
Coordenadas geográficas (lat, lng)
Obrigatórios: variable, value
Formato: Array de objetos JSON
Timestamp: UTC ISO8601
