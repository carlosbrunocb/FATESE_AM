Preparação do Ambiente no TagoIO
Configuração passo a passo para integração do ESP32 com a plataforma cloud
1
Criar Conta Completo
Acessar tago.io e criar uma conta gratuita de desenvolvedor
• Preencher formulário de registro
• Confirmar e-mail de ativação
• Acessar dashboard principal
2
Adicionar Dispositivo Completo
No menu lateral, acessar Devices → Add Device
• Clicar em "Add Device"
• Selecionar tipo "Custom HTTPS/MQTT"
• Nomear o dispositivo
3
Obter Device Token Completo
Na aba Tokens, copiar o Default Token gerado
• Clicar na aba "Tokens"
• Copiar o token gerado
• Guardar em local seguro
4
Configurar ESP32 Em andamento
Inserir o token no código do ESP32 e configurar WiFi
• Substituir "COLE_AQUI_O_SEU_DEVICE_TOKEN"
• Configurar SSID e senha WiFi
• Verificar conexão
5
Testar Conexão Pendente
Executar o código e verificar se dados chegam ao TagoIO
• Compilar e fazer upload do código
• Monitorar Serial Monitor
• Verificar Live Inspector
6
Validar Dados Pendente
Confirmar que os dados estão sendo recebidos corretamente
• Acessar aba Live Inspector
• Verificar variáveis recebidas
• Validar timestamps
Status: 3 de 6 passos completos
Tempo estimado: 15 minutos
