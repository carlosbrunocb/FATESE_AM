Bibliotecas e Ferramentas
Stack tecnológico para integração ESP32 com nuvem
WiFi.h
Conectividade Wi-Fi
Biblioteca nativa do ESP32 para gerenciamento de conexão Wi-Fi, monitoramento de status e reconexão automática.
#include <WiFi.h>
// Inicializa WiFi
WiFi.begin(ssid, password);
Nativa ESP32
HTTPClient.h
Cliente HTTP
Biblioteca para requisições HTTP/HTTPS, suporte a POST, GET, PUT e manipulação de headers.
#include <HTTPClient.h>
// Cria cliente HTTP
HTTPClient http;
Nativa ESP32
ArduinoJson
Manipulação JSON
Biblioteca para serialização e desserialização de JSON, otimizada para memória limitada.
#include <ArduinoJson.h>
// Cria documento JSON
JsonDocument doc;
