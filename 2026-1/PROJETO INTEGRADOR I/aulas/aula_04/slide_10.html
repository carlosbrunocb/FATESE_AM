Atividade Prática: Refinamento de Funções

Transforme a “Lista de Funções” em Feature Principal + RF + RNF (MVP viável)

Projeto Integrador I

Tarefas do grupo (entregável de hoje)

Use o escopo da Aula 03 como base e refine para um MVP executável

Anti‑NASA

1

Definir a Feature Principal

Escolha 1 (máx. 2) função(ões) que, sem ela(s), o sistema perde a razão de existir.

Pergunta: “Se eu remover isso, ainda faz sentido?”

2

Escrever pelo menos 5 Requisitos Funcionais (RF)

Liste as ações essenciais do fluxo do MVP. Padronize com RF01, RF02…

Modelo

“O sistema deve permitir…”

Qualidade do texto

Curto, claro e testável

3

Escrever pelo menos 3 Requisitos Não Funcionais (RNF)

Defina travas e qualidades para viabilizar o MVP (plataforma, desempenho, tecnologia, segurança).

Plataforma

Web (ex.)

Desempenho

≤ 3s (ex.)

Tecnologia

POO (ex.)

Atualizar o Documento de Escopo

Adicione as listas de RF/RNF ao documento do PI I (derivadas das funcionalidades principais).

Numere: RF01… RNF01…

Fluxo de trabalho (visual rápido)

Da lista de ideias para um MVP com requisitos claros

Saída: escopo refinado

Lista de Funções

Brainstorm / Aula 03

Feature Principal

1–2 no máximo

RF + RNF

Escopo Atualizado

Base p/ User Stories

Critérios de aceitação (check final)

Antes de entregar, revise se está claro e viável

Testável

Cada RF/RNF deve permitir verificação (passa/falha) ou evidência (métrica).

Viável no semestre

Nada de “IA complexa + 10 integrações”. Se for “NASA”, simplifique.

Foco na dor do usuário

O MVP resolve o problema central sem depender de “perfumarias”.

Padronizado

Requisitos numerados (RF01… / RNF01…) e 1 requisito por frase.

Dicas rápidas para acelerar

Se a função não muda a vida do usuário no MVP, mova para versão futura.

Prefira RNF mensuráveis (≤ 3s, 3 cliques, Web).

Escreva RF como ações: cadastro → consulta → ação.

Frase para cortar excesso

“Foque no que dói mais no seu cliente agora.”

O que entregar

Documento de Escopo atualizado com Feature Principal + RF + RNF

5 RF + 3 RNF

Ao finalizar, valide: “resolve a dor principal?” + “dá para entregar no semestre?”.
