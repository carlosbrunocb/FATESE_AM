Ferramenta

GitHub Projects: Seu Centro de Comando

Integração total entre código, issues e planejamento

Funcionalidades Poderosas

Múltiplas Views

Visualize como Board (Kanban), Table (Planilha) ou Roadmap (Gantt) com um clique.

Automações Integradas

Issues movem-se automaticamente quando PRs são abertos, revisados ou mergeados.

Insights e Campos Customizados

Gráficos de queima (burn-up/down) e campos personalizados para status, prioridade e tamanho.

Guia Rápido: Configuração

1

Criar Novo Projeto

Na aba Projects do repositório ou organização, clique em New Project.

2

Selecionar Template

Escolha Board para começar com o layout Kanban clássico.

3

Configurar Colunas

Defina os status: Todo, In Progress, Review, Done.

4

Vincular Repositório

Conecte o projeto ao repositório para puxar Issues e Pull Requests.

Dica Pro: Use a view de "Roadmap" para visualizar datas de entrega e planejar Sprints ao longo do tempo.
