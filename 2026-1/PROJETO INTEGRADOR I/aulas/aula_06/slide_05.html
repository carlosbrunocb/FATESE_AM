Roteiro do Workshop
Estrutura completa do conteúdo e atividades
60 minutos
1
Fundamentação

O que é documentação técnica? Papel estratégico do README como "página de vendas" técnica do repositório.

Markdown Essencial: Títulos, ênfases, listas e tabelas para documentação clara.

2
Dossiê do Projeto

Introdução: Contexto do setor e problema identificado.

Problema & Delimitação: A "dor" específica que será atacada.

3
Jornada de Valor

Tríade: Principal → Secundária → Resultado (tabela de valor).

Justificativa: Dados que provam que a solução é necessária.

4
Estudo de Caso

AdoptMatch: Referência profissional de README bem estruturado.

Prática Guiada: Edição do README.md no GitHub.
