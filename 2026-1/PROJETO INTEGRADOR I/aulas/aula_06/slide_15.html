# AdoptMatch 🐾 - Match por Afinidade
> Plataforma inteligente para conectar adotantes e pets
com base no estilo de vida.
## 1. Introdução
A adoção no Brasil enfrenta desafio crítico:
alta taxa de devolução.
## 2. O Problema
- 80% de devoluções por incompatibilidade
- Adoção por impulso visual
## 3. Justificativa
Triagens estruturadas reduzem devoluções
em até 40%.
## 4. Jornada de Valor
| Nível | Componente | Função |
| Principal | Match | Quiz Afinidade |
| Secundária | Chat | Suporte |
| Resultado | Valor | -40% devoluções |
# Instalação
1. git clone https://github.com/org/adoptmatch
2. cd adoptmatch && npm install
3. npm run dev
