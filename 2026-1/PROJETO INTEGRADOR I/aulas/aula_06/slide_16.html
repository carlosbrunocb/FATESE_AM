Criando o README.md no repositório
Prática Entrega 3
10 Passos para Criar o README
Em andamento
1
Abrir o repositório
Acesse o repo da Aula 05 e crie/edite README.md
2
Título + Pitch
Adicione nome do projeto + frase de valor em 1 linha
3
Introdução
Contexto do setor (3-4 linhas) com dados relevantes
4
Problema & Delimitação
Defina a dor específica e escopo do projeto
5
Justificativa
1-2 dados com fonte (link) para provar necessidade
6
Tríade de Valor
Tabela: RF01 (Principal), RF02 (Secundária), Meta
7
Instalação & Uso
3 passos com comandos para rodar o projeto
8
Evidências
Prints/gifs do fluxo-chave (ou placeholders)
9
Commit Semântico
"docs: adiciona README inicial com Tríade de Valor"
10
Revisão Cruzada
Colega de outro grupo lê e comenta (5 min)
Checklist de Qualidade
Regra dos 15s: mensagem clara no topo
Pirâmide invertida: valor → detalhes
CTAs claros: como executar/testar
IDs RFxx na Tríade de Valor
Ortografia revisada
