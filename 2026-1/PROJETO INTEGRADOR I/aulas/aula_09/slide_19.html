Coerência (1.1 → 1.2)
O Objetivo Geral deve resolver explicitamente o problema descrito no 1.1. Existe uma rastreabilidade: Contexto → Gap → Proposta → Objetivo Geral → Objetivos Específicos → KPIs.

(Não conecta com o gap de devoluções)

(Resolve o gap identificado)
