O que é Scrumban?
O Scrumban é uma abordagem híbrida que combina a cadência e papéis do Scrum com a gestão visual de fluxo do Kanban. Ideal para projetos acadêmicos, oferece flexibilidade mantendo estrutura e disciplina, permitindo adaptação contínua com visibilidade do andamento.
Estrutura Prática
Sprints curtas (1-2 semanas) + Board visual
Políticas WIP por coluna (limitação de trabalho)
DoR/DoD definidos para qualidade
Dailies, Reviews e Retros nos marcos de Sprint
Vantagens para Projetos Acadêmicos
Flexibilidade com disciplina - Mantém estrutura
Entregas frequentes - Feedback contínuo
Visibilidade do andamento - Board visual
Baixo overhead - Para equipes pequenas
Planejamento
Sprint Planning semanal
Papéis
PO, SM, Devs
Board
GitHub Projects
Métricas
Lead Time, Cycle Time
Dica: Use Scrumban para equilibrar estrutura e flexibilidade no projeto acadêmico
