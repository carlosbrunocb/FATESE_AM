O que é GitHub Projects?
Ferramenta nativa de gestão de projetos integrada ao GitHub que permite criar boards Kanban visuais, organizar tarefas em Milestones, categorizar com Labels e distribuir responsabilidades entre Assignees. Essencial para rastreabilidade completa do trabalho acadêmico.
