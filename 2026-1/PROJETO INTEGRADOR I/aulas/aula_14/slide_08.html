Princípio editorial
Um Dossiê não cataloga
tecnologias — ele justifica escolhas.
EVITAR · catálogo
"O projeto utiliza HTML5, CSS3 e JavaScript no front-end. O back-end é em Node.js. O banco de dados é MongoDB. React foi usado para a interface."
→ Lista de tecnologias sem causa. A banca não aprende nada sobre por quê dessas decisões.
BUSCAR · argumento
"A escolha do React decorre do requisito não-funcional de tempo de resposta < 2s na triagem; sua arquitetura por componentes permite reuso direto entre o painel da ONG e o app do adotante."
→ Tecnologia ligada a requisito, requisito ligado a necessidade.
