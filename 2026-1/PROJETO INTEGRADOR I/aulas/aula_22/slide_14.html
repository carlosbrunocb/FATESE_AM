Categorias de RNF
Use a coluna "Categoria" da tabela para enquadrar cada RNF — banca cobra essa rotulação.
CAT. 01
Performance
Tempo de resposta, throughput, capacidade simultânea.
Ex: match retorna em ≤ 3s
CAT. 02
Segurança
Criptografia, autenticação, LGPD, autorização por perfil.
Ex: senhas com hash bcrypt
CAT. 03
Usabilidade
Padrões de UI, acessibilidade, responsividade, idiomas.
Ex: Mobile First a partir de 360px
CAT. 04
Portabilidade
Navegadores, SO, dispositivos compatíveis.
Ex: Chrome, Safari, Firefox últimos
CAT. 05
Confiabilidade
Disponibilidade, recuperação de falhas, backup.
Ex: uptime ≥ 99% horário comercial
CAT. 06
Tecnologia
Stack obrigatória, frameworks, banco, hospedagem.
Ex: React + Node + PostgreSQL
