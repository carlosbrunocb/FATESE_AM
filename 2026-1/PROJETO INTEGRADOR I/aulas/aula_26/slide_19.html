Arquitetura Visual e Fluxo de Navegação

A construção da interface gráfica do AdoptMatch foi fundamentada nos princípios de User Interface (UI) e User Experience (UX), adotando o padrão arquitetural Mobile First para garantir a acessibilidade da plataforma em dispositivos móveis.

Todo o ecossistema visual foi prototipado em alta fidelidade utilizando a plataforma Figma, permitindo a simulação interativa das rotas do sistema. O fluxo de navegação do Produto Mínimo Viável (MVP) foi desenhado de forma linear para reduzir a carga cognitiva do usuário durante o processo de triagem.

A lógica de transição entre os frames está esquematizada no Quadro 3. A Figura 2 apresenta as três principais telas do protótipo de alta fidelidade renderizadas no Figma, evidenciando o design system adotado, que faz uso de cores acolhedoras (tons pastéis de verde e laranja) para transmitir confiança e empatia.
