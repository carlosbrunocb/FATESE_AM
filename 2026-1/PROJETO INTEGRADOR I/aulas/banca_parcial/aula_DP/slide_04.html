Jornada de Valor - Conceito
Estruturando o pensamento para slides e texto com foco na tríade: Principal → Secundária → Resultado
Principal (Killer Feature)
Razão de existência do projeto
É a razão da existência. Sem ela, o projeto morre. É o core value proposition.
Deve ser única e inovadora. Diferencial competitivo claro.
Testável e validável. Hipótese mensurável.
Exemplo: Quiz de Afinidade no AdoptMatch - algoritmo que conecta adotantes e pets.
Secundária (Suporte)
Ponte para a Killer Feature
Só entra no MVP se for a "ponte" para a Principal funcionar.
Suporte indispensável. Sem ela, a Principal não funciona.
Validação intermediária. Testes de usabilidade.
Exemplo: Chat interno para confirmar visita após o match.
Resultado (Valor)
Ganho real para o cliente
Ganho mensurável. Economia de tempo, dinheiro ou impacto social.
Resultado tangível. Métricas de sucesso claras.
Validação final. Testes de aceitação.
Exemplo: Redução de 40% na taxa de devolução de animais.
