Fluxo de Valor - Exemplo: AdoptMatch
Clique nos cards para ver detalhes
Principal
Killer Feature
Razão de existência do projeto. Sem ela, o projeto morre. É o diferencial que resolve o problema central.
Exemplo AdoptMatch:
Quiz de Afinidade (10 perguntas) → Algoritmo de Match
Pergunta de validação:
"Se eu remover X, o problema continua resolvido?"
Secundária
Suporte
Só entra no MVP se for a "ponte" para a Killer Feature funcionar.
Exemplo AdoptMatch:
Chat interno para validar visita após match
Justificativa:
Necessário para concretizar adoção
Resultado
Valor
Ganho real para o cliente: Economia de tempo, dinheiro ou impacto social.
Exemplo AdoptMatch:
40% redução na taxa de devolução
Métrica de sucesso:
Redução de devoluções em 6 meses
Anti-exemplo: Evitar Alto Esforço / Baixo Valor

"Reconhecimento facial para saber se o cachorro está feliz"
- Complexo de implementar e não resolve o problema central de devolução de pets. Priorize features que realmente impactam o resultado final.
