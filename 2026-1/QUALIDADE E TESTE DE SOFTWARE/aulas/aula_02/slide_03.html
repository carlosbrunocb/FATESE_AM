Erro, Defeito e Falha

Humano Código Execução

Erro

Ação Humana

É um equívoco, conceito errado ou mal-entendido por parte do ser humano (analista, desenvolvedor, etc.).

Exemplo

"O analista interpretou que o desconto deveria ser aplicado antes do imposto, mas a regra era depois."

Defeito

Bug (No Artefato)

É a manifestação física do erro no código, documento ou configuração. Uma imperfeição que pode causar falha.

Exemplo

function calcTotal(v) {
  return v * 0.9;
} // Lógica errada

Falha

Na Execução

É o comportamento incorreto perceptível pelo usuário ou sistema. Ocorre quando o código defeituoso é executado.

Exemplo

O usuário vê na tela "Total: R$ 90,00" quando o correto seria "R$ 95,00". O resultado final está errado.

Relação Causal: Erros geram Defeitos, que, se não detectados, manifestam-se como Falhas.
