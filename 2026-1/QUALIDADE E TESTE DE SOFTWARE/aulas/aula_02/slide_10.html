Bloco B: Software Testing Life Cycle

STLC: Visão Geral

Ciclo de Vida de Teste

O que é STLC?

É o conjunto de etapas sequenciais que organiza todo o processo de validação, garantindo que os critérios de qualidade sejam atendidos de ponta a ponta no projeto.

Principais Objetivos

  • Prevenir Defeitos

    Identificar falhas antes que cheguem à produção.

  • Evidenciar Qualidade

    Fornecer visibilidade sobre o status do software.

  • Apoiar Decisões

    Oferecer dados para decisões de release (Go/No-Go).

Principais Artefatos Gerados

Plano de Testes

Estratégia, escopo, recursos e cronograma.

Casos de Teste

Roteiros passo a passo com resultados esperados.

Dados de Teste

Massa de dados necessária para execução.

Evidências

Screenshots, logs e vídeos de comprovação.

Defeitos (Bugs)

Registro detalhado das falhas encontradas.

Relatório Final

Resumo executivo, métricas e lições.
