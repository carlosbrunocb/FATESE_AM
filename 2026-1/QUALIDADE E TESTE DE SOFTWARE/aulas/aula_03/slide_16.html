Ferramentas de QA

Foco em Processo & Prevenção

GitHub / GitLab

Gestão de Código

Essencial para Branch Protection, Pull Requests e controle de versão. É onde as regras de prevenção são aplicadas antes do merge.

Jenkins / GitHub Actions

CI/CD Pipeline

Automação de esteiras (pipelines). Executa linters e testes automaticamente a cada commit, bloqueando falhas.

SonarQube

Análise Estática

Analisa a qualidade do código (Code Smells, Segurança, Duplicação) e define Quality Gates obrigatórios.

Ferramentas de QC

Foco em Produto & Detecção

Jira / Trello

Gestão de Defeitos

Rastreamento de bugs, gestão do ciclo de vida do defeito e organização das tarefas de teste.

Selenium

Teste Funcional

Framework para automação de testes web. Simula a interação do usuário com o navegador (E2E).

Postman

Teste de API

Ferramenta para testar requisições HTTP, validar status codes, payloads e contratos de API.
