script.py
def CalcularSoma(a, b):
    return a + b
Method name "CalcularSoma" doesn't conform to snake_case

O que são?

Analisadores estáticos que "leem" o código sem executá-lo. Eles agem como um corretor gramatical para sua linguagem de programação, identificando desvios de padrão e erros antes do teste.

Benefícios

  • Padronização: Todo o time escreve código igual.
  • Prevenção: Pega bugs (variáveis não usadas, loops infinitos).
  • Qualidade: Reduz a complexidade ciclomática.
