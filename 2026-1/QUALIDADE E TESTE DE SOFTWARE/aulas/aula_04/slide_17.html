Exemplo de Código: Análise Manual

bad_code.py Original
def calcular_desconto(valor):
    if valor > 100:
        return valor * 0.1 # Erro Lógico!     return valor
good_code.py Corrigido
def calcular_desconto(valor):
    """Retorna valor final c/ desconto se > 100."""
    return valor * 0.9 if valor > 100 else valor

Problemas Identificados

Erro de Lógica (Grave):
O código retorna apenas o valor do desconto (0.1), e não o valor final com o desconto aplicado (0.9).
Nomenclatura Confusa:
A função diz calcular_desconto, mas o fluxo sugere que deveria retornar o preço final a pagar.
Falta de Documentação:
Ausência de Docstring explicando o comportamento da função e o que ela espera receber/retornar.
Edge Case:
Comportamento para valor exatamente 100 não está explícito (cai no else implícito).
