Glossário: Ferramentas
Flake8 Linter

Ferramenta leve para Python que verifica conformidade com a PEP 8, erros de sintaxe e complexidade ciclomática.

Black Formatter

"The Uncompromising Code Formatter". Reformata código Python automaticamente, eliminando discussões sobre estilo.

Prettier Formatter

Formatador de código opinativo para JS, TS, HTML e CSS. Garante estilo consistente ao salvar o arquivo.

Jenkins CI/CD

Servidor de automação open source muito popular para construir, testar e fazer deploy de software.

GitLab CI CI/CD

Ferramenta integrada ao GitLab para pipelines de DevOps, definidos via arquivo .gitlab-ci.yml.

Semgrep SAST

Ferramenta de análise estática rápida e moderna que permite escrever regras personalizadas para encontrar bugs e falhas de segurança.
