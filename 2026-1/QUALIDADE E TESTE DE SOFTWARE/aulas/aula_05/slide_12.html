Aula 05 Ferramentas

Instalação e Configuração Pytest

1

Criar Ambiente Virtual

Isole as dependências do projeto para evitar conflitos.

python -m venv .venv
2

Instalar Pytest e Plugins

Instale o framework principal e o plugin de cobertura.

pip install pytest pytest-cov
3

Estruturar Projeto

Organize seu código fonte e testes em diretórios separados.

src/ tests/
4

Executar Testes

Rode o comando na raiz do projeto para descobrir testes automaticamente.

pytest -v
