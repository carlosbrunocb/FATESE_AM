Fase 1: Red Ciclo TDD

Exemplo Prático: A Falha Inicial

1O Cenário

Precisamos de uma função eh_maior_de_idade(idade) que retorna True para idades >= 18. No TDD, começamos pelo teste, não pela função.

2A Ação (Write Test)

Criamos o arquivo test_idade.py importando a função e fazendo uma asserção simples.

3A Execução (Run)

Rodamos o pytest. Como a função ainda não existe no arquivo sistema.py (ou o arquivo nem existe), o teste DEVE falhar.

Expectativa

Erro de importação (ImportError) ou nome não definido (NameError). Essa falha confirma que o teste está "conectado" e a funcionalidade não existe.

tests/test_idade.py
Editor de Código
from sistema import eh_maior_de_idade def test_deve_retornar_true_se_maior_ou_igual_a_18(): # Arrange (Preparação) idade = 20 # Act (Ação) & Assert (Verificação) assert eh_maior_de_idade(idade) is True
