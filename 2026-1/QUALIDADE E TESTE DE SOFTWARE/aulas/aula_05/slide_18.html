Aula 05 Análise de Qualidade

Cobertura de Código (Coverage)

Ferramentas e Comandos

pip install pytest-cov

Instala o plugin necessário para integrar a análise de cobertura ao Pytest.

pytest --cov=src

Executa os testes medindo a cobertura apenas dos arquivos dentro da pasta src.

... --cov-report=term-missing

Exibe no terminal quais linhas exatas (números) não foram executadas pelos testes.

coverage html

Gera um relatório visual completo em HTML na pasta htmlcov/index.html.

--cov-fail-under=90

Faz o comando falhar se a cobertura total for menor que 90% (bom para CI/CD).

Resultado no Terminal

Configuração Persistente
pyproject.toml
# Configuração para não precisar digitar flags sempre [tool.pytest.ini_options] addopts = "--cov=src --cov-report=html" [tool.coverage.run] omit = [ "tests/*", "**/__init__.py" ]
