Aula 05

Recap — Conceitos Principais

Testes de unidade: devem ser isolados, extremamente rápidos e determinísticos.

TDD (Red-Green-Refactor): ciclo de desenvolvimento que guia o design do software e garante testabilidade.

Pytest: framework robusto com execução simplificada, assertions nativos e sistema de fixtures.

Ferramentas de Apoio: Coverage para análise de cobertura e Mocks para isolamento de dependências.

Aplicação Prática: Exercício FizzBuzz como base para fixação do fluxo de pensamento TDD.
