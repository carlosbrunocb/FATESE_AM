O que são Testes Unitários?

São testes automatizados que verificam o funcionamento da menor unidade de código testável de uma aplicação, como uma função, um método ou uma classe, de forma isolada.

Rápidos

Executam em milissegundos, permitindo feedback instantâneo durante o desenvolvimento.

Isolados

Não dependem de banco de dados, rede ou sistema de arquivos (usam Mocks se necessário).

Determinísticos

Para a mesma entrada, sempre produzem o mesmo resultado. Sem flutuações.

Documentação Viva

Explicam como o código deve se comportar em diferentes cenários e bordas.

Convenção Pytest

Arquivos de Teste

test_*.py ou *_test.py

Ex: test_calculadora.py

Funções de Teste

def test_cenario():

O prefixo test_ é obrigatório.

O Pytest descobre e executa automaticamente tudo que segue esse padrão.
