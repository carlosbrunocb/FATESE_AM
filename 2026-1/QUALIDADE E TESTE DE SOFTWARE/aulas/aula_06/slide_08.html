Ferramentas e Tecnologias

Ecossistema Python para QA

Linguagem

Python 3

Linguagem base para todo o desenvolvimento. Simples, legível e com vasta biblioteca padrão para automação.

Framework

Pytest

Framework de testes poderoso. Oferece fixtures, parametrização e relatórios detalhados de falhas.

Ambiente

VS Code / PyCharm

Editores com suporte nativo a testes. Permitem rodar e depurar (debug) testes visualmente.

Gerenciamento

Pip & Venv

Pip: Instala pacotes.
Venv: Isola bibliotecas do projeto para evitar conflitos de versão no sistema.

Qualidade Extra

Complementares

Flake8 / Black: Linting e formatação.
Pytest-cov: Relatórios de cobertura.
