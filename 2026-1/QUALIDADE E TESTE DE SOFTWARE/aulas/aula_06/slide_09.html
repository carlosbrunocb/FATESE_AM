Python 3 — Fundamento

O que é?

Linguagem de programação interpretada, de alto nível e propósito geral. É a base tecnológica do nosso projeto de QA e automação.

Para que serve?

Utilizada para implementar a lógica de negócio (o software em si) e escrever os scripts de teste que validarão essa lógica.

Pontos de Atenção

  • Certifique-se de usar a versão 3.8+

  • No Windows, o comando pode ser py ou python

  • No Linux/Mac, frequentemente é python3
