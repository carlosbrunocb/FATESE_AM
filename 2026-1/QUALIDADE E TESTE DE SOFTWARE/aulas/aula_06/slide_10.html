Automação de Testes

Pytest — Framework de Testes

O que é?

O framework mais popular do ecossistema Python. Permite escrever testes pequenos e legíveis, mas escala para suportar testes funcionais complexos em aplicações e bibliotecas.

Vantagens Principais

Descoberta Automática: Encontra arquivos test_*.py.

Assert Introspection: Mensagens de erro detalhadas sem boilerplate.

Fixtures: Gerenciamento poderoso de setup/teardown.

Parametrização: Rode o mesmo teste com múltiplas entradas.

zsh — pytest execution
# 1. Executar todos os testes da pasta atual
$ pytest
collected 3 items
tests/test_calc.py ... [100%]
3 passed in 0.02s
# 2. Modo Verboso (-vv) e filtrar por nome (-k)
$ pytest -vv -k "desconto"
collected 5 items / 2 deselected / 3 selected
tests/test_calc.py::test_desconto_10_porcento PASSED [ 33%]
tests/test_calc.py::test_sem_desconto PASSED [ 66%]
tests/test_calc.py::test_desconto_invalido FAILED [100%]
# 3. Parar na primeira falha (-x) e saída curta (-q)
$ pytest -x -q
F
======================= 1 failed in 0.04s =======================
$_
