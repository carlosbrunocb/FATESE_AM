Ambiente de Desenvolvimento

IDEs — VS Code e PyCharm

VS Code

  • Leve e Customizável

  • Extensão oficial Microsoft

  • "Test Explorer" Integrado

PyCharm

  • IDE Robusta "All-in-one"

  • Configuração automática

  • Análise de código estática

1

Configure o Interpretador (Venv)

Aponte a IDE para o Python dentro da pasta venv/.
Isso garante que o pacote pytest instalado seja reconhecido e utilizado, evitando erros de "Module Not Found".

2

Habilite a Descoberta de Testes

No VS Code, acesse a aba "Testing" (ícone de Erlenmeyer) e configure para usar o framework Pytest. Indique que a pasta raiz dos testes é tests/.

3

Execução e Análise Visual

Use os botões de "Play" (▶) que aparecem ao lado de cada função test_. Em caso de falha, use breakpoints (bolinha vermelha) e o modo Debug para inspecionar variáveis passo a passo.
