Passo 1 — Criação do Projeto

Espaço de Trabalho

O primeiro passo é criar um diretório dedicado para isolar nosso projeto. Isso garante que as dependências e arquivos de teste não entrem em conflito com outros projetos no seu computador.

Comandos Essenciais

mkdir: (Make Directory) Cria uma nova pasta.

cd: (Change Directory) Entra na pasta criada.

Boas Práticas: Use nomes minúsculos e sem espaços (snake_case).

Git (Opcional): Iniciar versionamento desde o começo.

bash — setup
# 1. Criar a pasta do projeto
$ mkdir lab_qualidade_software
# 2. Entrar na pasta criada
$ cd lab_qualidade_software
# 3. Verificar se estamos na pasta correta
~/lab_qualidade_software $ pwd
/home/aluno/lab_qualidade_software
# 4. (Opcional) Iniciar repositório Git
~/lab_qualidade_software $ git init
Initialized empty Git repository in /home/aluno/lab_qualidade_software/.git/
~/lab_qualidade_software $_
