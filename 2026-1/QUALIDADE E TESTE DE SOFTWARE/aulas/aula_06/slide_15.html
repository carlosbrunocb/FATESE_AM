Passo 3 — Instalação do Pytest

Instalação

Com o ambiente virtual ativo, instalamos o pacote pytest. Ele será adicionado como uma dependência do projeto, garantindo que todos os desenvolvedores usem a mesma ferramenta.

Boas Práticas

Confirme se o (venv) está ativo no terminal.

Use pip freeze para salvar as versões.

Mantenha o requirements.txt versionado no Git.

bash — pip install
# 1. Instalar o pytest no ambiente virtual
(venv) $ pip install pytest
Collecting pytest
Downloading pytest-7.4.0-py3-none-any.whl (323 kB)
Installing collected packages: iniconfig, packaging, pluggy, pytest
Successfully installed pytest-7.4.0
# 2. Gerar lista de dependências (congelar versões)
(venv) $ pip freeze > requirements.txt
# 3. Verificar o conteúdo do arquivo gerado
(venv) $ cat requirements.txt
iniconfig==2.0.0
packaging==23.1
pluggy==1.2.0
pytest==7.4.0
(venv) $_
