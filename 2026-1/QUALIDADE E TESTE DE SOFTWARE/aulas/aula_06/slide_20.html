Ciclo TDD: Red-Green-Refactor

Fase GREEN — Implementação Mínima

Objetivo Principal

Escrever o código de produção suficiente para que o teste, que antes falhava, agora passe com sucesso. A prioridade é a funcionalidade, não a elegância.

Regras da Fase Green

Implemente a lógica mais simples possível.

Não tente otimizar agora (evite over-engineering).

Rode os testes imediatamente após codificar.

Se ficar verde, avance para Refactor.

src/calculadora.py
def calcular_preco_final(valor):
# Lógica mínima: se maior que 100, desconta 10%
if valor > 100:
return valor * 0.9
return valor
