Mão na Massa: Exemplo com Mockito

Simulando o comportamento de uma dependência externa (Ex: Banco de Dados):

import static org.mockito.Mockito.*;
import org.junit.jupiter.api.Test;

class PedidoServiceTest {

    @Test
    void deveCalcularFreteComSucesso() {
        // 1. Criação do Mock
        CorreiosService correiosMock = mock(CorreiosService.class);
        
        // 2. Configuração de Expectativa (Stubbing)
        when(correiosMock.calcularFrete("01001-000")).thenReturn(15.50);
        
        // 3. Injeção e Execução
        PedidoService service = new PedidoService(correiosMock);
        double valorFinal = service.finalizarPedido("01001-000");
        
        // 4. Verificação
        assertEquals(115.50, valorFinal);
        verify(correiosMock, times(1)).calcularFrete("01001-000");
    }
}

Note que o CorreiosService real nunca foi chamado, garantindo velocidade e isolamento no teste!
