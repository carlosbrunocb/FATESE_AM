Lógica Real
src/pagamento.py 
def processar_venda(valor, servico_bancario):
    """
    Processa uma venda usando um serviço externo.
    Dependência: servico_bancario (injetada)
    """
    
    # Chamada ao método do objeto injetado
    status = servico_bancario.autorizar(valor)
    
    if status == "Sucesso":
        return f"Venda de R${valor} confirmada"
        
    return "Venda negada"

Teste de Integração

Isolado com Mock
tests/test_integracao.py 
from src.pagamento import processar_venda

class MockBanco:
    """Dublê que simula o serviço bancário"""
    def autorizar(self, v):
        # Simula resposta imediata sem rede
        return "Sucesso"

def test_integracao_negocio_e_banco():
    # 1. Arrange (Preparação)
    # Criamos o objeto falso (Mock)
    banco = MockBanco()
    
    # 2. Act (Ação)
    # Injetamos o mock no lugar do serviço real
    resultado = processar_venda(100, banco)
    
    # 3. Assert (Verificação)
    assert resultado == "Venda de R$100 confirmada"

Conceito Chave: Injeção de Dependência

Note que processar_venda não cria o banco internamente ("new Banco()"). Ele recebe o banco como argumento, permitindo que passemos um Mock durante o teste.
