Cenário Base

Nova Tarefa

Toda vez que a calculadora realizar um cálculo (ex: 2 + 3), ela deve salvar essa operação em um arquivo de texto persistente chamado historico.txt.

Isso cria uma dependência de sistema de arquivos!

Desafio QA

O que você precisa entregar nesta aula

Implementar Módulo de Log

Criar uma classe ou função responsável apenas por escrever no arquivo (princípio de responsabilidade única).

Teste de Integração (com Mock/Spy)

Validar se, ao chamar calc.somar(2,3), a calculadora chama corretamente o método de log. Use mocks para não criar arquivos reais durante este teste.

Teste de Sistema (End-to-End)

Simular o fluxo real: Iniciar calculadora -> Fazer 3 somas -> Abrir o arquivo gerado -> Verificar se as 3 linhas estão lá. (Aqui usamos arquivo real/temporário).

Bônus: Tratamento de Erro

O que acontece se o disco estiver cheio ou sem permissão? Simule uma exceção no Mock e garanta que a calculadora não trave.
