01

Injeção de Dependência

Evite acoplamento forte instanciando o Logger dentro da Calculadora.

class Calculadora:
def __init__(self): self.log = Logger()

def __init__(self, logger):
self.log = logger

02

Fixture tmp_path

Não escreva no seu disco rígido real. Use a fixture nativa do Pytest.

O tmp_path cria uma pasta temporária única para cada teste e a apaga depois. Perfeito para testes de integração de arquivos.

Critérios de Aceitação

Seu teste de integração deve validar:

Resultado Correto da Operação

A calculadora continua funcionando?

assert resultado == 5

Persistência do Arquivo

O arquivo de log foi realmente criado no caminho especificado?

assert arquivo.exists()

Formato do Log

A linha escrita segue o padrão "a + b = r"?

assert "2 + 3 = 5" in arquivo.read_text()

Sem Efeitos Colaterais Globais

O teste não deve deixar arquivos de lixo no projeto (uso correto de fixtures).
