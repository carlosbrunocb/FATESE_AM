Ferramentas e Recursos

Ambiente Pronto

Para realizar os testes de integração e sistema com mocks, precisamos de um ambiente Python configurado corretamente com as bibliotecas de teste.

"Ferramentas certas reduzem o atrito e aumentam o foco na qualidade."

Dica de Ouro

Use ambientes virtuais (venv) para isolar as dependências do laboratório.

IDEs Recomendadas

Utilize VS Code ou PyCharm. Certifique-se de ter as extensões de Python instaladas para suporte a linting e execução de testes visuais.

Framework de Teste

Usaremos o Pytest como runner principal pela sua simplicidade e poder com fixtures.

Versionamento

Entrega via GitHub. Faça um fork do repositório base, crie uma branch para o exercício e submeta via Pull Request.

Instalação

Instale o framework e o plugin de mock:

pip install pytest pytest-mock requests
