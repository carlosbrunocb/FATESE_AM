Cobertura de Instruções (Statement Coverage) Métrica básica de cobertura
Percentual de linhas/instruções executadas pelo conjunto de testes
Definição

Statement Coverage: Métrica que mede o percentual de linhas/instruções executadas pelo conjunto de testes.

Fórmula: Cstmt = (instruções executadas / instruções executáveis) × 100%

Vantagens
✓ Simples de medir
✓ Rápida de obter
✓ Bom ponto de partida
Limitações
✗ Não garante ramos
✗ Ignora condições
✗ Pode deixar bugs
Fórmula de Cálculo
Cstmt = (IE / IE_total) × 100%
IE = Instruções executadas | IE_total = Total de instruções executáveis
Exemplo: Função soma_pos
Python
1 def soma_pos(a, b):
2 s = a + b # I1
3 if s < 0: # I2 (decisão)
4 s = 0 # I3 (executa só se True)
5 return s # I4
Teste 1: (2, 3) → s=5
Executa: I1, I2, I4 (if=False)
Teste 2: (-5, 1) → s=-4
Executa: I1, I2, I3, I4 (if=True)
Instruções
4/4 (100%)
Cobertura
