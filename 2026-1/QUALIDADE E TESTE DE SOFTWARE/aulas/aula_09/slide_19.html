O que é Pytest?

Pytest é um framework de testes para Python que permite criar testes simples e escaláveis.

Instalação
pip install pytest
✓ Testes unitários, integração e funcional
✓ Assertions nativos (assert)
✓ Fixtures para setup/teardown
✓ Plugins extensíveis
Comandos Básicos
pytest
pytest -v
pytest -k
pytest -x
pytest --cov
