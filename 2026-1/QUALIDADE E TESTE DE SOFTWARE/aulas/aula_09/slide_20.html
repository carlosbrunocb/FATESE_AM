Coverage.py e pytest-cov Ferramentas para medição de cobertura de código
Instalação, configuração e tipos de relatórios de cobertura
Instalação
$ pip install pytest-cov coverage
pytest-cov: Plugin pytest para cobertura
coverage: Ferramenta standalone de cobertura
Comandos Básicos
pytest --cov=modulo - Testa módulo específico
pytest --cov-branch - Inclui cobertura de branch
pytest --cov-report=html - Gera relatório HTML
