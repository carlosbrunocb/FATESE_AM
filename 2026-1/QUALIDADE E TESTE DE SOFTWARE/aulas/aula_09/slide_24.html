Módulo III - Planejamento e Gerenciamento de Testes
Executando e Verificando a Cobertura Guia prático de análise
Passo a passo para executar testes, gerar relatórios e interpretar resultados de cobertura
1
Instalar dependências
Execute o comando para instalar pytest e pytest-cov:
$ pip install pytest pytest-cov
2
Rodar testes com cobertura
Execute testes com relatório de cobertura no terminal:
$ pytest -q --cov=src --cov-branch --cov-report=term-missing
3
Gerar relatório HTML
Gere relatório HTML detalhado para análise visual:
$ pytest --cov=src --cov-branch --cov-report=html
4
Abrir relatório HTML
Abra o arquivo htmlcov/index.html no navegador para visualizar:
Como ler o relatório
Verde: Linhas cobertas (100%)
Amarelo: Parcialmente cobertas
Vermelho: Não cobertas
Cobertura atual: 85%
Branches: 12/14
Dicas para análise
Identifique faltantes: Verifique linhas em vermelho no relatório
Adicione casos: Crie testes para branches não exercitados
Comando final: pytest --cov=src --cov-branch --cov-report=html
Use --cov-fail-under=90 para CI/CD
