Estrutura de Arquivos do Projeto
Organização do projeto aula_11_regressao com Pytest
Em andamento

Estrutura de Diretórios

aula_11_regressao/
├── src/
│    └── calculadora.py # Código fonte com bugs
├── tests/
│    └── test_calculadora.py # Testes de regressão
├── conftest.py # Configurações do Pytest
└── pytest.ini # Configuração de markers

Comandos de Execução

# Navegar até a raiz do projeto
cd aula_11_regressao

# Executar todos os testes
pytest -v

# Executar testes específicos por marker
pytest -m regressao -v

# Executar com cobertura
pytest --cov=src --cov-report=html

Ferramentas Necessárias

Python 3.x
Linguagem principal
Pytest
Framework de testes
Git
Controle de versão
VS Code
IDE recomendada

Observações

Usar @pytest.mark.regressao para marcar testes críticos
Execute pytest --markers para ver todos os marcadores disponíveis
