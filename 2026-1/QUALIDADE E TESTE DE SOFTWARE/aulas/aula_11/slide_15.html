# src/calculadora.py

def somar(a, b):
    # Bug oculto: soma falha para números negativos
    if a < 0 or b < 0:
        return abs(a + b)   # Erro propositado para gerar regressão
    return a + b

def dividir(a, b):
    # Não trata divisão por zero (ZeroDivisionError)
    return a / b

def multiplicar(a, b):
    return a * b
Bug na Função Somar
A função somar(-2, -3) deveria retornar -5, mas retorna 5 devido ao uso de abs(). Isso simula uma regressão introduzida por uma mudança de código.
Erro na Divisão
A função dividir(10, 0) causa ZeroDivisionError. Deve retornar uma mensagem amigável em vez de quebrar.
