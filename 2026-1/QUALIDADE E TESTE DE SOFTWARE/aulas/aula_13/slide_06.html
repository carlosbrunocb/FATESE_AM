O que é o Selenium?
O Selenium é o framework mais utilizado mundialmente para automação de navegadores. Ele é composto por três componentes principais que trabalham em conjunto para oferecer uma solução completa de automação de testes web.
Selenium IDE
Record & Playback
Ferramenta de "record and playback" para prototipagem rápida. Permite gravar ações do usuário e reproduzi-las posteriormente.
  • Gravação automática de ações
  • Edição de scripts
  • Exportação para múltiplas linguagens
Uso Ideal:
Prototipagem rápida e testes simples
Selenium WebDriver
API Principal
O componente principal que fornece uma interface para enviar comandos diretamente ao navegador de forma nativa.
  • Controle nativo do navegador
  • Suporte a múltiplos navegadores
  • Execução em modo headless
Uso Ideal:
Automação de testes funcionais
Selenium Grid
Execução Distribuída
Utilizado para executar testes em diferentes máquinas e navegadores em paralelo, otimizando o tempo de execução.
  • Execução paralela
  • Distribuição em múltiplos nós
  • Escalabilidade
Uso Ideal:
Testes em larga escala
