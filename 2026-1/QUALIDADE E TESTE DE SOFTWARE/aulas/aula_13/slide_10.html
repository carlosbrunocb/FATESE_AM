Instalação e Configuração do Ambiente

Configuração do Python, Selenium e WebDrivers

Configurado
1. Instalação do Python
1
Baixe Python 3.x do site oficial: python.org
2
Durante instalação, marque "Add Python to PATH"
3
Verifique: python --version
# Verificar versão do Python python --version # Deve mostrar Python 3.x.x
2. Instalação do Selenium
1
Abra o terminal ou prompt de comando
2
Execute: pip install selenium
3
Verifique: pip show selenium
# Instalar Selenium pip install selenium # Instalar WebDriver Manager pip install webdriver-manager
3. Configuração do WebDriver
1
Baixe o ChromeDriver compatível com sua versão do Chrome
2
Adicione ao PATH ou use webdriver-manager
3
Verifique: chromedriver --version
# Usando webdriver-manager (recomendado) from webdriver_manager.chrome import ChromeDriverManager driver = webdriver.Chrome(ChromeDriverManager().install())
4. Verificação Final
1
Crie um arquivo test_selenium.py
2
Execute o script de teste básico
3
Verifique se o navegador abre
# Teste básico from selenium import webdriver driver = webdriver.Chrome() driver.get("https://www.google.com") driver.quit()
