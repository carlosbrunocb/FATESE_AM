Localização de Elementos (Locators)

Como identificar e interagir com elementos em páginas web

Selenium Locators
Visão Geral
Para interagir com um site, o script precisa "encontrar" os elementos (botões, campos de texto, links). Os locators são estratégias usadas para identificar elementos no DOM. Cada método tem suas vantagens e é adequado para diferentes cenários.
ID
O mais rápido e seguro. Geralmente único na página. Ideal para elementos com identificador específico.
driver.find_element(By.ID, "login-btn") driver.find_element(By.ID, "email")
Name
Útil para campos de formulário. Pode não ser único, mas é comum em inputs.
driver.find_element(By.NAME, "q") driver.find_element(By.NAME, "username")
XPath
Caminho na estrutura XML do HTML. Muito potente, mas pode ser frágil.
//div[@id='main'] //input[@name='email']
CSS Selector
Baseado nas classes de estilo. Geralmente mais rápido que o XPath.
.btn-primary #login-form input
Dica Importante
Prefira ID quando disponível. Use XPath ou CSS para elementos dinâmicos. Evite localizadores baseados em posição ou texto.
Atenção
Localizadores frágeis podem quebrar com mudanças na interface. Sempre verifique a estabilidade.
