Locators: ID e Name

Estratégias de localização de elementos na página web

Prática
Quando usar ID e Name
O ID é o localizador mais rápido e seguro, pois geralmente é único na página. O Name é útil para campos de formulário e pode ser usado quando o ID não está disponível.
  • ID: Geralmente único e mais rápido para localização
  • Name: Útil em formulários e inputs
  • Performance: ID é mais rápido que outros locators
Dica
Sempre prefira ID quando disponível. Se o ID não existir, use Name para campos de formulário.
Exemplos de Uso
ID
By.ID, "login-btn"
Name
By.NAME, "email"
ID
By.ID, "password"
Name
By.NAME, "q"
Exemplo de Código Python
Python
# 1. Importar bibliotecas from selenium import webdriver from selenium.webdriver.common.by import By # 2. Configurar o driver driver = webdriver.Chrome() try: # 3. Abrir página driver.get("https://example.com/login") # 4. Localizar por ID login_btn = driver.find_element(By.ID, "login-btn") # 5. Localizar por Name email_field = driver.find_element(By.NAME, "email") # 6. Interagir com elementos email_field.send_keys("usuario@email.com") login_btn.click() finally: driver.quit()
