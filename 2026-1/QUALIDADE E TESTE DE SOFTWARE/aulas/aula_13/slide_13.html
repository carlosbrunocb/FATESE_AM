Locators: XPath

Localização poderosa e flexível de elementos

Prática
O que é XPath?
XPath é uma linguagem de consulta usada para navegar em documentos XML e HTML. Permite localizar elementos usando caminhos na estrutura do documento.
  • Absoluto: Caminho completo desde a raiz (/html/body/div)
  • Relativo: Busca em qualquer parte (//div[@class='x'])
  • Flexível: Suporta funções e operadores
Atenção
XPath pode ser frágil se o site mudar muito. Prefira localizadores mais estáveis quando possível.
Tipos de XPath
Absoluto
/html/body/div[1]/input
Relativo
//input[@id='email']
Com contains()
//div[contains(@class,'btn')]
Com starts-with()
//a[starts-with(@href,'/')]
Exemplo de XPath em Python
Python
# 1. Importar bibliotecas from selenium import webdriver from selenium.webdriver.common.by import By # 2. Configurar o driver driver = webdriver.Chrome() try: # 3. Abrir página driver.get("https://example.com") # 4. Localizar por XPath absoluto elemento = driver.find_element(By.XPATH, "/html/body/div[1]/input") # 5. Localizar por XPath relativo botao = driver.find_element(By.XPATH, "//button[@id='login']") # 6. XPath com contains() link = driver.find_element(By.XPATH, "//a[contains(text(),'Sobre')]") finally: driver.quit()
