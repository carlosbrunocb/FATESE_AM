Locators: CSS Selector

Seletores CSS para localização de elementos web

Prática
Vantagens do CSS Selector
CSS Selectors são geralmente mais rápidos que XPath e oferecem uma sintaxe mais limpa e legível. São ideais para localizar elementos baseados em classes, IDs e hierarquia.
  • Performance: Mais rápido que XPath em navegadores modernos
  • Legibilidade: Sintaxe mais limpa e fácil de entender
  • Flexibilidade: Suporte a pseudo-classes e atributos
Dica
Use CSS Selectors para localizar elementos por classe, ID ou atributos. Prefira seletores mais específicos.
Tipos de Seletores CSS
Por ID
#login-btn
Por Classe
.btn.primary
Por Atributo
[type="submit"]
Por Tag
input[name="email"]
Exemplo de Código Python
Python
# 1. Importar bibliotecas from selenium import webdriver from selenium.webdriver.common.by import By # 2. Configurar o driver driver = webdriver.Chrome() try: # 3. Abrir página driver.get("https://example.com/login") # 4. Localizar por ID (CSS #) login_btn = driver.find_element(By.CSS_SELECTOR, "#login-btn") # 5. Localizar por classe (CSS .) email_field = driver.find_element(By.CSS_SELECTOR, ".form-control") # 6. Localizar por atributo password = driver.find_element(By.CSS_SELECTOR, "input[type='password']") # 7. Interagir com elementos email_field.send_keys("usuario@email.com") login_btn.click() finally: driver.quit()
