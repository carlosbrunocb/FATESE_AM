Inspecionando Elementos no Navegador

Como usar as DevTools para localizar elementos e testar seletores

Prática
1
Abrir DevTools
Pressione F12 (Windows/Linux) ou Cmd+Opt+I (macOS) para abrir as ferramentas de desenvolvedor.
2
Selecionar Elemento
Clique no ícone Ctrl+Shift+C para ativar o modo de seleção e clique no elemento desejado.
3
Analisar HTML e Atributos
Observe a estrutura HTML, identifique IDs, classes e atributos únicos que podem ser usados como locators.
4
Copiar Seletor/XPath
Clique com o botão direito no elemento e selecione "Copy" > "Copy selector" ou "Copy XPath".
5
Validar no Console
Use o console para testar seletores: $x("//tag[@attr='val']") para XPath ou document.querySelector() para CSS.
