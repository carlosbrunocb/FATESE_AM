Interações Comuns com Elementos

Click, send_keys, atributos, estado e mais

Prática
Métodos de Interação
O Selenium permite diversas interações com elementos web. Conheça os métodos mais comuns para automatizar ações do usuário.
Click
element.click()
Send Keys
element.send_keys("texto")
Clear
element.clear()
Displayed
element.is_displayed()
Enabled
element.is_enabled()
Get Attribute
element.get_attribute("value")
Dica
Sempre verifique se o elemento está visível e habilitado antes de interagir com ele.
Exemplo Prático de Interações
Python
# Exemplo de interações com elementos from selenium import webdriver from selenium.webdriver.common.by import By driver = webdriver.Chrome() try: driver.get("https://example.com/form") # Localizar elementos email = driver.find_element(By.ID, "email") submit = driver.find_element(By.ID, "submit") # Limpar campo e inserir texto email.clear() email.send_keys("usuario@email.com") # Verificar se está habilitado if submit.is_enabled(): submit.click() # Verificar mensagem de sucesso success = driver.find_element(By.CLASS_NAME, "success") print("Mensagem:", success.text) finally: driver.quit()
