Validações, Assertions e Esperas

Estratégias para garantir a qualidade dos testes automatizados

Prática
Tipos de Validações
As validações em testes automatizados garantem que o sistema está funcionando conforme esperado. Utilize assertions para verificar estados, textos, URLs e elementos.
Título da página
assert "texto" in driver.title
URL atual
assert "url" in driver.current_url
Elemento visível
assert element.is_displayed()
Texto no elemento
assert "texto" in element.text
Esperas Explícitas
Espera por elemento
WebDriverWait(driver, 10).until(EC.presence_of_element_located((By.ID, "id")))
Espera por visibilidade
WebDriverWait(driver, 10).until(EC.visibility_of_element_located((By.ID, "id")))
Espera por click
WebDriverWait(driver, 10).until(EC.element_to_be_clickable((By.ID, "id")))
Espera por texto
WebDriverWait(driver, 10).until(EC.text_to_be_present_in_element((By.ID, "id"), "texto"))
Exemplo de Validação e Espera
Python
from selenium import webdriver from selenium.webdriver.common.by import By from selenium.webdriver.support.ui import WebDriverWait from selenium.webdriver.support import expected_conditions as EC driver = webdriver.Chrome() try: driver.get("https://example.com") # Espera explícita por elemento wait = WebDriverWait(driver, 10) elemento = wait.until( EC.presence_of_element_located((By.ID, "login-btn")) ) # Validação de título assert "Página de Login" in driver.title # Validação de elemento assert elemento.is_displayed() # Validação de texto mensagem = driver.find_element(By.ID, "msg") assert "Bem-vindo" in mensagem.text print("Teste passou com sucesso!") finally: driver.quit()
