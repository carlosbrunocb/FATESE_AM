Metas de Aprendizado Qualidade e Teste de Software

Objetivos da Aula 14

O que você dominará até o fim deste laboratório prático:

⚙️
Ambiente

Configuração Zero-to-One

Configurar do zero um ambiente virtual de desenvolvimento limpo utilizando Python 3 e instalar as bibliotecas robustas do ecossistema Selenium 4.

🎯
Estratégia

Mapeamento de Elementos (Locators)

Inspecionar semanticamente o código HTML de formulários modernos. Mapear de forma precisa e sem ambiguidades os inputs de texto, botões de ação e alertas de interface.

🛠️
Automação

Scripts de Fluxo Completo

Desenvolver do início ao fim um script automatizado que realiza o login, valida o estado interno da área restrita e executa a saída segura da sessão (Logout).

⚡
Resiliência

Tratamento de Sincronismo

Mitigar de vez testes falsos-positivos (Flaky Tests) usando estratégias de esperas dinâmicas inteligentes (Implicit vs. Explicit Waits).
