Preparando o Ambiente Isolado

Como isolar dependências e instalar as ferramentas necessárias para rodar o laboratório prático.

Passo a Passo no Terminal

1. Criar e acessar a pasta do projeto
mkdir aula14_automacao_login
cd aula14_automacao_login
2. Criar e Ativar o Ambiente Virtual (venv)
python -m venv venv
# No Windows (PowerShell):
.\venv\Scripts\Activate.ps1
# No Linux/Mac:
source venv/bin/activate
3. Atualizar o Gerenciador de Pacotes pip e Instalar Selenium
pip install --upgrade pip
pip install selenium

Por que usar venv?

O ambiente virtual garante que as bibliotecas do Selenium instaladas para esta aula não entrem em conflito com outras versões de dependências de outros projetos no mesmo computador.

Gerenciamento Dinâmico

No Selenium 4, você não precisa mais se preocupar em baixar o executável do chromedriver manualmente. O próprio framework gerencia os drivers compatíveis com as versões locais do navegador em execução.
