Escrevendo o Script: Importações e Setup

As dependências necessárias e a inicialização limpa do navegador.

import time
from selenium import webdriver
from selenium.webdriver.common.by import By
from selenium.webdriver.support.ui import WebDriverWait
from selenium.webdriver.support import expected_conditions as EC

def executar_teste_login_e_navegacao():
    print("--- Inicializando o Navegador ---")
    # O Selenium 4 gerencia o binário do driver de forma transparente e nativa
    driver = webdriver.Chrome()

    # Configura a Espera Implícita de até 10 segundos para contingências no DOM
    driver.implicitly_wait(10)

    # Maximiza a tela para garantir que elementos responsivos fiquem visíveis
    driver.maximize_window()

Por que By?

A classe By agrupa as estratégias de seletores compatíveis com o WebDriver (ID, CSS Selector, XPath, etc.).

expected_conditions

Aliás abreviado como EC, contém as condições que a espera explícita aguardará que aconteçam na tela de forma assíncrona.
