Escrevendo o Script: Interação e Preenchimento

Navegando até a URL alvo e interagindo com o formulário de login.

    try:
        # 1. Ação: Navegar até a URL alvo
        driver.get("https://the-internet.herokuapp.com/login")

        # 2. Configura Espera Explícita para garantir prontidão
        wait = WebDriverWait(driver, 15)
        campo_usuario = wait.until(EC.presence_of_element_located((By.ID, "username")))
        campo_senha = driver.find_element(By.ID, "password")
        botao_entrar = driver.find_element(By.CSS_SELECTOR, "button[type='submit']")

        # 3. Ação: Enviar dados válidos de autenticação
        print("Preenchendo dados de autenticação...")
        campo_usuario.send_keys("tomsmith")
        campo_senha.send_keys("SuperSecretPassword!")

        # 4. Ação: Submeter o formulário
        print("Enviando formulário...")
        botao_entrar.click()

send_keys()

Envia uma cadeia de caracteres (digitação) diretamente para o elemento de input mapeado na tela do navegador.

click()

Executa um clique real de mouse no elemento de ação (botão ou link) para submissão do formulário.
