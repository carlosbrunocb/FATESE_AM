Locust: Teste de Carga Moderno

Framework de teste de carga baseado em Python para simulação de usuários virtuais
Locust Framework
Teste de Carga em Python
Locust é um framework de teste de carga moderno que permite modelar usuários virtuais como código. Ele oferece uma interface web intuitiva para execução e análise de testes, suporte a execução distribuída e cenários customizados.
Python 3
Web UI
Distribuído
locustfile.py - Exemplo Básico
1 from locust import HttpUser, task, between
2  
3 class CompradorVirtual(HttpUser):
4     wait_time = between(1, 3)
5  
6     @task(2)
7     def acessar_home(self):
8         self.client.get("/")

Instalação
pip install locust
Para começar, instale o Locust via pip e crie um arquivo locustfile.py com as tarefas de teste. Execute com locust no terminal.
