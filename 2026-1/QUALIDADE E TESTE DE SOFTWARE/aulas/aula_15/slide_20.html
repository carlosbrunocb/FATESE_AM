Ambiente e Instalação - Python + Locust

Guia passo a passo para configurar o ambiente de teste de carga
1
Preparação do Ambiente
Python 3 instalado
Verifique se o Python 3 está instalado e atualizado no sistema.
# Verificar versão do Python
python --version
# Ou
python3 --version
2
Instalar Locust
Via pip
Instale o Locust usando o gerenciador de pacotes pip.
# Instalar Locust
pip install locust
# Verificar instalação
locust --version
3
Criar Script
locustfile.py
Crie o arquivo de teste com a classe HttpUser.
from locust import HttpUser, task
class CompradorVirtual(HttpUser):
    @task
    def acessar_home(self):
        self.client.get("/")
4
Executar Teste
Iniciar Locust
Execute o Locust e acesse a interface web.
# Iniciar Locust
locust -f locustfile.py
# Acessar http://localhost:8089
5
Configurar Teste
Parâmetros
Configure usuários, taxa de spawn e tempo de espera.
# Configuração
wait_time = between(1, 3)
users = 500
spawn_rate = 10
6
Análise de Resultados
Métricas
Analise RPS, latência e taxa de erros na interface.
# Métricas importantes
RPS: Requisições/seg
Latência: p95/p99
Erros: 4xx/5xx
Dicas Importantes
Python 3.7+
Ambiente virtual
Script validado
