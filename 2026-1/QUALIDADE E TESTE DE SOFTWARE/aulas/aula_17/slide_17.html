Exemplo Prático 2: A Cobertura Ilusória

Demonstração teórica de como obter 100% de cobertura mantendo um bug crítico no sistema

Código de Produção (src/calculadora.py)
def dividir(a, b):
    return a / b
Teste Automatizado (tests/test_calculadora.py)
from src.calculadora import dividir

def test_divisao_simples():
    assert dividir(10, 2) == 5
⚠️

Análise de Risco

📊 Cobertura Obtida: 100%

Ao rodar o comando de teste, 100% das linhas do arquivo calculadora.py foram executadas com sucesso. A suíte relata cobertura perfeita.

O Defeito Oculto (ZeroDivisionError)

O teste não previu o cenário extremo onde b = 0. Se um usuário chamar dividir(10, 0) em produção, o sistema lançará uma exceção crítica e possivelmente causará um travamento (crash).

Este exemplo clássico mostra que cobertura foca em linhas executadas, não na complexidade e segurança funcional do negócio.

Conclusão: 100% de cobertura ≠ Código sem Bugs
