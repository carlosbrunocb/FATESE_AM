Introdução ao Pytest-cov

A extensão oficial do ecossistema Pytest para geração dinâmica de cobertura em Python

O que é o Pytest-cov?

O pytest-cov é um plugin poderoso e amplamente adotado na comunidade Python. Ele envelopa a biblioteca de testes coverage.py para executá-la nativamente em conjunto com o framework de testes pytest.

Vantagens da Ferramenta:
  • Mapeamento instantâneo via terminal na execução dos testes.
  • Geração automática de relatórios em HTML interativo de altíssima fidelidade.
  • Fácil integração com pipelines modernos de CI/CD (ex: GitHub Actions).
  • Suporte completo a filtros para ignorar arquivos de configuração.

Funcionalidades de Relatório

Relatório Terminal:

Imprime diretamente na saída de texto uma tabela detalhada com cada arquivo do projeto, quantidade de linhas, linhas perdidas (statements não cobertos) e o percentual geral.

Relatório HTML Interativo:

Exporta uma pasta web onde é possível abrir o código-fonte no navegador. As linhas cobertas são destacadas em verde, e as não cobertas (não testadas) brilham em vermelho.

Relatório XML para Gateways:

Formato ideal para integração com plataformas de qualidade estática de código como SonarQube e Codecov.

Parâmetro de terminal chave: --cov e --cov-report
