Passo 1: Preparação do Ambiente Prático

Como instalar as dependências necessárias para cobertura de código no seu laboratório de desenvolvimento

Terminal / Bash (Instalação Direta)
$ pip install pytest pytest-cov
Arquivo de Dependências (requirements.txt)
pytest==8.0.0
pytest-cov==4.1.0
Terminal / Bash (Instalação via requirements)
$ pip install -r requirements.txt

Verificando a Instalação

Sempre que instalar um plugin no Python, é uma excelente prática verificar se o mesmo foi indexado corretamente pelo gerenciador de pacotes pip.

$ pip show pytest-cov

Estrutura do Projeto Recomendada

meu-projeto/
├── src/
│   └── loja.py
├── tests/
│   └── test_loja.py
└── requirements.txt
Atenção: Garanta o uso do Python 3.8+ no laboratório
