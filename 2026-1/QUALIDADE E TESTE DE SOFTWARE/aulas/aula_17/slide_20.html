Passo 2: O Código Alvo de Teste (src/loja.py)

Construção de uma lógica de negócio com ramificações condicionais para estudo de cobertura

Arquivo: src/loja.py
def calcular_frete(valor_compra, estado):
    if estado == 'SP':
        return 10.0 if valor_compra < 100 else 0.0

    elif estado == 'RJ':
        return 15.0 if valor_compra < 100 else 0.0

    else:
        return 30.0

Análise de Fluxos Lógicos

Este pequeno código possui 5 ramos condicionais distintos (branches) baseados no estado e no valor da compra.

Ramos a Testar:
  • Estado SP com valor < 100 (Frete = R$ 10.0)
  • Estado SP com valor ≥ 100 (Frete grátis = R$ 0.0)
  • Estado RJ com valor < 100 (Frete = R$ 15.0)
  • Estado RJ com valor ≥ 100 (Frete grátis = R$ 0.0)
  • Outros estados (Frete fixo = R$ 30.0)

Nosso desafio prático será escrever testes que executem deliberadamente cada um desses desvios para alcançar os desejados 100% de cobertura estrutural.
