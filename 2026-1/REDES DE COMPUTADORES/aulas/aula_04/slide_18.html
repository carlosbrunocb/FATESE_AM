Protocolos na Prática

ARP e ICMP

ARP (Address Resolution) Resolve IP para MAC. Essencial para LAN via Broadcast.
ICMP (Control Message) Fundamental para diagnósticos e erros de rede.
Comando Ping Usa Echo Request/Reply para testar conectividade.

TCP, UDP e Simulação

TCP (Reliable) Conexão via Handshake de 3 vias (SYN, SYN-ACK, ACK).
UDP (Best Effort) Sem conexão, focado em velocidade (DNS, Streaming).
Visualização PDU No simulador, cada protocolo tem uma cor específica.
