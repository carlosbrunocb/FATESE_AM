IPv4

Classe A (IPv4)

Redes muito grandes • Máscara padrão /8 • 1º octeto identifica a rede

Arquitetura classful • Máscara padrão e faixa

Características

1º octeto define a rede (restante é host).

Indicada para redes muito grandes (muitos hosts em uma única rede).

Prefixo classful: 0xxxxxxx (primeiro bit = 0).

Faixa (1º octeto)

0 a 127

Máscara padrão

255.0.0.0 (/8)

Atenção: endereços especiais

127.0.0.0/8 é loopback (não é usado como rede “normal”)

127.0.0.1

Exemplo típico (faixa privada)

A rede 10.0.0.0/8 (RFC 1918) é muito usada em ambientes internos.

Divisão Rede/Host (Classe A)

1 octeto rede • 3 octetos host
