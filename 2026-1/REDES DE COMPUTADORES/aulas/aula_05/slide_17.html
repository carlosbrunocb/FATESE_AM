Máscara de Sub-rede (Subnet Mask)

Define quais bits do IPv4 são rede e quais são host

Base teórica • Rede vs Host

Conceito

A máscara de sub-rede é um valor de 32 bits que indica a separação entre a porção de REDE e a porção de HOST de um endereço IPv4.

Bits 1

Identificam a rede

Bits 0

Identificam hosts

Exemplo de máscara

/24 é comum em redes Classe C

255.255.255.0

Por que importa?

A máscara permite que o host decida se o destino está na mesma rede (envia direto) ou em outra rede (envia ao gateway/roteador).
