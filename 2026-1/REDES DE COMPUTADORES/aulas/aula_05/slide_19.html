Base teórica • Default Masks (Classful)

O que é “máscara padrão”?

Na arquitetura classful, cada classe (A/B/C) tem uma máscara default que define quantos bits são de rede e quantos são de host.

Dica de leitura

Memorize /8, /16 e /24

visão rápida

Classe A

/8

255.0.0.0

Classe B

/16

255.255.0.0

Classe C

/24

255.255.255.0

Conexão com o mundo real

Em redes atuais, costuma-se usar CIDR (ex.: /26, /20), mas as máscaras padrão ajudam a interpretar exemplos clássicos e identificar rapidamente a classe.
