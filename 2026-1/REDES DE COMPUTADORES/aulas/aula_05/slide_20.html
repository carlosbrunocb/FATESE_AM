IPv4

Identificando Rede vs Host

Exemplo: 192.168.10.45/24 — prefixo determina onde “corta” o endereço

Regra • /24 = 24 bits de rede + 8 bits de host

Onde está a separação?

Com /24, a rede ocupa 3 octetos. O 4º octeto fica para host.

Porção de rede

O que identifica a rede (prefixo).

192.168.10

Porção de host

O que identifica o dispositivo.

45

Dica rápida

Em redes /24, é comum usar .0 para rede e .255 para broadcast.

Notação

A máscara equivalente é 255.255.255.0

/24

Exemplo visual (rede e host)

Rede em azul • Host em verde

Rede (endereça a rede)

192.168.10.0

Host (endereça o dispositivo)

192.168.10.45

Interpretação

A rede é 192.168.10.0/24 e o host é identificado pelo último octeto.
