Operação AND lógica (cálculo de rede)

Como obter o endereço de rede a partir de IP + máscara

Base teórica • Bits de rede (1) preservam, bits de host (0) zeram

Ideia central

Endereço de rede = IP AND Máscara

Bit a bit

Regras do AND (0/1)

1 AND 1

= 1

1 AND 0

= 0

0 AND 1

= 0

0 AND 0

= 0

Interpretação: onde a máscara tem 1, o IP é preservado; onde tem 0, o resultado vira 0.

Para que isso serve na prática?

O roteador e o host usam o endereço de rede para decidir se o destino está na mesma rede (LAN) ou se deve ir ao gateway.

Exemplo: 192.168.10.45/24

192.168.10.45 AND 255.255.255.0
