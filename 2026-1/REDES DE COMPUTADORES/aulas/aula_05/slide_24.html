Aula 05

IPv4

Calculando o endereço de rede (método geral)

Algoritmo prático: IP + máscara → Rede (via AND lógico)

Base teórica • Rede = IP AND Máscara

Receita em 4 passos

Funciona para qualquer IPv4 e qualquer prefixo CIDR

Resultado: Rede

1

Tenha o IP e a máscara/prefixo

Ex.: 10.1.5.200/16 (máscara 255.255.0.0)

2

Converta IP e máscara para binário

Faça isso octeto a octeto (8 bits) usando a tabela de pesos.

3

Aplique AND bit a bit

Regra: 1 AND x = x e 0 AND x = 0.

Converta o resultado para decimal

O valor obtido é o endereço de rede.

Atalho importante (sem binário)

Para máscaras “fechadas” em octetos (ex.: /8, /16, /24), a rede é o IP com os octetos de host zerados.

Exemplo rápido: 10.1.5.200/16

Aplicando o método para encontrar a rede

Máscara: 255.255.0.0

1

Separe rede/host pelo prefixo

/16 significa: 16 bits de rede (2 primeiros octetos).

2

Zere a parte de host

10.1.5.200

→

10.1.0.0

host = 16 bits

Endereço de rede

10.1.0.0

rede encontrada

Em seguida, podemos calcular broadcast e hosts úteis.

Checklist mental
