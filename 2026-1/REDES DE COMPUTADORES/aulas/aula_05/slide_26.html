Aula 05

IPv4

Exemplo prático: 172.16.50.1

Classe B (máscara padrão) • rede, broadcast e faixa de hosts

Prática guiada • Classful (/16)

Identificação da classe

Pelo 1º octeto do endereço

1º octeto = 172

IP

172.16.50.1

Classe

B

Máscara padrão

255.255.0.0

CIDR equivalente

/16

Observação (faixa privada)

O bloco 172.16.0.0/12 é reservado para redes privadas (RFC 1918).

Interpretação rápida (classful)

Em Classe B, a rede ocupa os 2 primeiros octetos e o host ocupa os 2 últimos.

Cálculo (padrão da classe)

Rede, broadcast e hosts úteis

172.16.50.1 /16

1

Endereço de rede

Zere os bits de host (últimos 16 bits).

Rede = 172.16.0.0

2

Endereço de broadcast

Coloque 1 em todos os bits de host.

Broadcast = 172.16.255.255

Faixa de hosts úteis

Vai de rede + 1 até broadcast − 1.

Primeiro host

172.16.0.1

Último host

172.16.255.254

Quantidade (classful)

216 − 2 = 65.534 hosts

Conexão com CIDR

Na prática, redes reais costumam usar /20, /24, etc., para adequar o tamanho da sub-rede à necessidade (VLSM).
