Aula 05

IPv4

Exercício de fixação

Identificação de classe, máscara padrão e endereço de rede

Avaliação formativa • Aplicação prática

Cenário

Um desenvolvedor precisa configurar um servidor web em uma rede local. O administrador forneceu o IP com a máscara padrão da classe.

IP fornecido

10.0.0.50

Condição

Usar máscara padrão da classe

Dica (sem resolver)

Para encontrar a classe, observe o 1º octeto. Depois, aplique a máscara padrão e zere os bits de host para obter a rede.

O que você deve entregar

Classe • máscara padrão • rede

3 respostas objetivas

Questões

Use 10.0.0.50 com a máscara padrão da classe

resolva em 3 passos

1

Qual é a classe deste IP?

Dica: compare o 1º octeto com as faixas A/B/C.

Resposta: _______________________

2

Qual é a máscara de sub-rede padrão?

Escreva em decimal pontuado (ex.: 255.255.255.0).

Resposta: _______________________

3

Qual é o endereço que representa esta rede?

Regra: bits de host = 0 (ou “zerar” os octetos de host).

Resposta: _______________________

Gabarito

Use para correção rápida ao final

respostas

1) Classe

A

2) Máscara padrão

255.0.0.0

3) Rede

10.0.0.0
