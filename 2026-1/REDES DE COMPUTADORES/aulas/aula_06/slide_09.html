Máscara de Sub-rede: Conceito e Representações
Entenda como a máscara define a fronteira entre rede e host
O que é uma Máscara de Sub-rede?

A máscara de sub-rede é um número de 32 bits que define qual parte do endereço IP representa a rede e qual parte representa o host.

Como funciona?

A máscara funciona como um "filtro" que separa os bits de rede dos bits de host:

  • Bits 1 na máscara = parte de rede
  • Bits 0 na máscara = parte de host
IP: 192.168.1.34
Mask: 255.255.255.192
Network: 192.168.1.0
Dica Importante

A máscara define a fronteira; hosts válidos estão entre o endereço de rede e o endereço de broadcast.
