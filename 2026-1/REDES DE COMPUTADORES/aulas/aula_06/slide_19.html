VLSM - Máscaras de Comprimento Variável
Alocação eficiente de endereços IP para redes de tamanhos diferentes
Conceito VLSM
Variable Length Subnet Masking

VLSM permite usar máscaras de sub-rede de diferentes tamanhos dentro da mesma rede principal, otimizando o uso de endereços IP.

Ideal para redes com setores de tamanhos distintos, evitando desperdício de endereços.
