Introdução ao Roteamento

O que é Roteamento?

É o processo de selecionar caminhos em uma rede para enviar pacotes de dados da origem ao destino. O roteador utiliza sua Tabela de Roteamento para decidir por qual interface de saída o pacote deve seguir.

Camada de Rede (Camada 3)

A Camada 3 do Modelo OSI é responsável por:
Endereçamento lógico (IP) e roteamento
Encaminhamento de pacotes entre redes
Determinação do melhor caminho
