O que é Roteamento?

Definição Fundamental

Conceito
O roteamento é o processo de selecionar caminhos em uma rede para enviar pacotes de dados da origem ao destino. O roteador utiliza sua Tabela de Roteamento para decidir por qual interface de saída o pacote deve seguir.

Objetivos do Roteamento

Eficiência
Resiliência
Política

Componentes Principais

Endereços IP: Identificação de origem e destino
Roteadores: Dispositivos de encaminhamento
Tabela de Roteamento: Base de decisão

Tabela de Roteamento

A tabela de roteamento contém:
Rede de Destino: Endereço IP da rede destino
Próximo Salto: Endereço do próximo roteador
Interface: Porta de saída
Métrica: Custo do caminho
