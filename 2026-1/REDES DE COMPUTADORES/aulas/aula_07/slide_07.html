Componentes
Componentes Essenciais do Roteamento
Endereços IP
Identificação única de dispositivos na rede (origem/destino) usando IPv4 ou IPv6
Roteadores
Dispositivos que interligam redes e encaminham pacotes entre diferentes segmentos
Tabela de Roteamento
Estrutura de dados que mapeia destinos para interfaces de saída
Protocolos
Regras para troca de informações entre roteadores (estáticos/dinâmicos)
Métricas
Critérios para avaliação de caminhos (saltos, largura de banda, atraso)
Próximo Salto
Endereço IP do próximo roteador no caminho para o destino
