Campos Principais

Rede de Destino: Endereço da rede destino (ex: 192.168.1.0/24)
Máscara: Define o tamanho da rede (ex: 255.255.255.0)
Próximo Salto: Endereço IP do próximo roteador

Como Funciona

O roteador consulta a tabela para cada pacote recebido:
Verifica o IP de destino
Busca a melhor correspondência
Determina a interface de saída
Encaminha o pacote
