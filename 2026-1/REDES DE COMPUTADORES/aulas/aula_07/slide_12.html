Próximo Salto (Next Hop)

O que é Próximo Salto?

O Próximo Salto (Next Hop) é o endereço IP do roteador imediatamente à frente no caminho até o destino final. É o próximo ponto de passagem obrigatório para o pacote alcançar seu destino.

Características Principais

Identifica o roteador vizinho para encaminhamento
Entrada típica: destino/prefixo → IP do next hop → interface
Se destino é diretamente conectado, next hop é "direto"
Usado para resolver endereços MAC via ARP

Exemplo de Entrada

192.168.1.0/24 → 10.0.0.1 (Next Hop) → Gi0/1
PC Origem
192.168.1.10
Roteador A
10.0.0.1
Destino
10.0.0.5
Processo de Next Hop
Roteador consulta tabela → Identifica next hop → Encaminha pacote
