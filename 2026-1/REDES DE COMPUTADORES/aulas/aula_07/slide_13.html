O que é Gateway Padrão?

O Gateway Padrão é a porta de saída de uma rede local para o resto do mundo. É o endereço IP do roteador que conecta sua rede local a outras redes (Internet).
Responsável por encaminhar pacotes para destinos externos
Atua como "tradutor" entre redes diferentes
Configurado nos dispositivos como último recurso

Rota Padrão (0.0.0.0/0)

A rota padrão 0.0.0.0/0 aponta para o gateway padrão e é usada quando:
O destino não está na tabela de roteamento
Não há rota específica para o destino
É necessário acessar redes externas
