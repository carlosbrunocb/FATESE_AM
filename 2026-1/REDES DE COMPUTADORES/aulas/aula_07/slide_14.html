Roteamento Estático
O que é Roteamento Estático?

O roteamento estático é um método onde o administrador de rede configura manualmente as rotas no roteador. As rotas permanecem fixas até que sejam alteradas manualmente.

Configuração manual: O administrador digita cada rota individualmente
Baixa complexidade: Simples de entender e implementar
Previsibilidade: Comportamento totalmente previsível
Características Principais
Escalabilidade: Limitada
Resiliência: Não reage a falhas
Consumo: Baixo CPU/Memória
Segurança: Mais controlável
Configuração Exemplo

Comando básico para configurar uma rota estática:

! Configurar rota para rede 192.168.10.0/24
ip route 192.168.10.0 255.255.255.0 10.0.0.2
!
! Configurar gateway padrão
ip route 0.0.0.0 0.0.0.0 10.0.0.1
Casos de Uso
Filial pequena: Conectar à matriz
Segurança: Rotas controladas
Backup: Rotas redundantes
