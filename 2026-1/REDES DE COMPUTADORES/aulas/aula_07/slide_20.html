AS & BGP
Sistemas Autônomos (AS)
O que é um Sistema Autônomo?
Um Sistema Autônomo (AS) é um conjunto de redes IP sob uma única administração técnica, com uma política de roteamento unificada. Cada AS possui um número único (ASN) que o identifica na internet global.
Características Principais
  • Identificação única: Cada AS recebe um número ASN (ex: AS15169 para Google)
  • Autonomia administrativa: Controle independente sobre roteamento interno
  • Política de roteamento: Define como o tráfego entra e sai do AS
  • Conectividade: Conecta-se a outros AS via BGP

BGP e Sistemas Autônomos
O BGP (Border Gateway Protocol) é o protocolo de roteamento entre AS. Ele permite que diferentes sistemas autônomos troquem informações de roteamento e alcancem a internet global.
