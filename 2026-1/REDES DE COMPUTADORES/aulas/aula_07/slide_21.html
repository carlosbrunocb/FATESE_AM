Encapsulamento de Pacotes IP

Processo de Encapsulamento

O encapsulamento é o processo de adicionar cabeçalhos de controle aos dados em cada camada do modelo OSI.
Dados → Segmento (Camada 4) → Pacote IP (Camada 3) → Quadro (Camada 2)
Cada camada adiciona informações de controle específicas
Endereços IP permanecem constantes durante o roteamento
