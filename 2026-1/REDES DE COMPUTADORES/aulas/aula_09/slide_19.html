TCP - Use quando precisar de:
Confiabilidade, ordem e controle de fluxo
Navegação Web 80/443
HTTP/HTTPS - Sites, APIs REST, aplicações web
Email 25/587/465
SMTP, IMAP, POP3 - Comunicação segura
Transferência de Arquivos 21/22
FTP, SFTP - Downloads seguros
SSH/Terminal 22
Acesso remoto, banco de dados
✅ Recomendação TCP
Use quando a entrega perfeita e ordenação são essenciais. Ex: transações financeiras, transferência de arquivos importantes.
UDP - Use quando precisar de:
Velocidade, baixa latência, broadcast
Streaming Ao Vivo Var
Netflix, YouTube, Twitch - Vídeo em tempo real
Jogos Online Var
FPS, MOBA - Latência crítica
VoIP/RTC 5060
Chamadas, videoconferências
DNS 53
Resolução de nomes, consultas rápidas
✅ Recomendação UDP
Use quando a velocidade é prioritária e alguma perda é aceitável. Ex: streaming, jogos, VoIP.
Orientações Práticas para Desenvolvedores
Requisitos de entrega perfeita → TCP
Requisitos de tempo real → UDP
Segurança: TCP (TLS) vs UDP (DTLS)
Firewall: TCP atravessa melhor NAT
Equilíbrio: QUIC/HTTP3 sobre UDP
Fallback: Implementar retry se UDP
