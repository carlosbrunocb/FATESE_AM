Comparativo: netstat vs ss no Linux
Legacy vs Moderno: Comandos para análise de sockets e conexões
Módulo
3 - Transporte
netstat (Legacy)
Deprecado
Sintaxe: netstat [opções]
Parâmetros úteis: -a (tudo), -n (numérico), -t (TCP), -u (UDP), -p (PID)
$ netstat -anpt # Mostrar conexões TCP com processos
$ netstat -anpu | grep 53 # Filtrar conexões UDP na porta 53
Status
DEPRECATED
Performance
Lenta
Informações
Limitadas
Portabilidade
Universal
Requer sudo para ver PIDs
ss (Socket Statistics)
Recomendado
Sintaxe: ss [opções]
Parâmetros úteis: -t (TCP), -u (UDP), -l (listen), -n (numérico), -p (PID)
$ ss -tulpn # Mostrar sockets em escuta com PID
$ ss -ntp state established # Conexões estabelecidas
