Exemplos Práticos - Comandos CLI

Comandos para diagnóstico e configuração de DNS e DHCP

CLI Tools
Comandos DNS
C:\> nslookup www.google.com
Saída:
Servidor: 8.8.8.8
Endereço: 142.250.191.46
$ dig www.google.com +short
Saída:
142.250.191.46
C:\> ipconfig /displaydns
Saída:
Registros DNS em cache...
C:\> ipconfig /flushdns
Saída:
Cache DNS limpo com sucesso
Comandos DHCP
C:\> ipconfig /release
Saída:
IP liberado: 192.168.1.100
C:\> ipconfig /renew
Saída:
Novo IP: 192.168.1.101
$ sudo dhclient -r eth0
Saída:
DHCP lease released
$ sudo dhclient eth0
Saída:
DHCP lease obtained
