Categorias e significados dos códigos de estado

5 famílias
de status codes
1xx
Informativo
100 Continue
101 Switching
103 Early Hints
Requisição recebida, continuando processo
2xx
Sucesso
200 OK
201 Created
204 No Content
Ação bem-sucedida
3xx
Redirecionamento
301 Moved
302 Found
304 Not Modified
Ação adicional necessária
4xx
Erro Cliente
400 Bad Request
401 Unauthorized
404 Not Found
Erro na requisição
5xx
Erro Servidor
500 Internal Error
502 Bad Gateway
503 Unavailable
Falha no servidor
Quando usar cada categoria?
Guia rápido de aplicação
1xx: Use para indicar que a requisição foi recebida e está sendo processada
2xx: Indique sucesso na operação solicitada
3xx: Redirecione o cliente para outro recurso
Erros Comuns
O que evitar
4xx: Verifique autenticação e permissões
5xx: Verifique logs do servidor
Timeout: Configure timeouts adequados
Cada status code indica o resultado da requisição
