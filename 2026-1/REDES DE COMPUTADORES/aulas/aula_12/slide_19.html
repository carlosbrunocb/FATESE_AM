Status Codes 2xx - Sucesso

Códigos de estado HTTP que indicam sucesso na requisição

5 códigos
mais comuns
200 OK
Sucesso padrão
2xx
A requisição foi bem-sucedida. O significado de "sucesso" depende do método HTTP utilizado.
GET: recurso retornado | POST: resultado da ação
201 Created
Recurso criado
2xx
A requisição foi bem-sucedida e um novo recurso foi criado como resultado.
Usado em POST/PUT para criação
202 Accepted
Processamento assíncrono
2xx
A requisição foi aceita para processamento, mas ainda não foi completada.
Para processamento em lote
204 No Content
Sem conteúdo
2xx
A requisição foi bem-sucedida, mas não há conteúdo para retornar.
Usado em DELETE/PUT
206 Partial Content
Conteúdo parcial
2xx
O servidor está retornando apenas parte do recurso solicitado.
Para downloads em partes
Resumo
Códigos 2xx indicam que a requisição foi recebida, compreendida e aceita com sucesso.
200: OK 201: Created 202: Accepted
2xx = Sucesso na requisição
Todos os códigos são idempotentes
