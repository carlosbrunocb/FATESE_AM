Status Codes 4xx - Erros do Cliente

Códigos de erro que indicam problemas na requisição do cliente

8 erros comuns
para identificar
400 Bad Request
Requisição malformada
Erro do Cliente
O servidor não pode processar a requisição devido a erro de sintaxe ou formato inválido
401 Unauthorized
Autenticação necessária
Não Autorizado
Requisição requer autenticação. Credenciais ausentes ou inválidas
403 Forbidden
Acesso proibido
Proibido
Servidor entendeu a requisição, mas recusa autorização
404 Not Found
Recurso não encontrado
Não Encontrado
O recurso solicitado não existe no servidor
405 Method Not Allowed
Método não permitido
Método Inválido
O método HTTP não é suportado para este recurso
409 Conflict
Conflito de estado
Conflito
A requisição conflita com o estado atual do recurso
415 Unsupported Media
Tipo de mídia não suportado
Mídia Inválida
O formato de dados da requisição não é suportado
429 Too Many Requests
Limite de requisições
Rate Limit
Número excessivo de requisições em curto período
