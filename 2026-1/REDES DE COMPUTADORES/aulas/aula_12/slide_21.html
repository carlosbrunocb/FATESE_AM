Status Codes 5xx

Erros do Servidor - Quando o servidor falha ao processar a requisição

6 códigos
principais 5xx
500

Internal Server Error

Erro Genérico

O servidor encontrou uma situação que não sabe como lidar. Erro genérico quando nenhum código 5xx mais específico se aplica.

501

Not Implemented

Não Implementado

O método da requisição não é suportado pelo servidor e não pode ser processado.

502

Bad Gateway

Gateway Inválido

O servidor, enquanto atuava como gateway ou proxy, recebeu uma resposta inválida do servidor upstream.

503

Service Unavailable

Indisponível

O servidor não está pronto para lidar com a requisição. Comum em manutenção ou sobrecarga.

504

Gateway Timeout

Timeout

O servidor não recebeu uma resposta a tempo do servidor upstream.

505

HTTP Version Not Supported

Versão Não Suportada

A versão HTTP usada na requisição não é suportada pelo servidor.

Resumo 5xx

Erros Genéricos 500, 501
Problemas de Gateway 502, 504
Indisponibilidade 503
