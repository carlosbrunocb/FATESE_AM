REDESCOMP
Componentes Básicos e Endereçamento
Componentes da Arquitetura 802.11
STA (Estação)
Dispositivos sem fio (laptops, smartphones, IoT)
AP (Access Point)
Ponto de acesso que conecta à rede cabeada
BSS
Basic Service Set - AP + STAs associadas
ESS
Extended Service Set - múltiplos BSS
Identificadores de Rede
SSID (Service Set Identifier)
O "nome" da rede Wi-Fi que identifica um BSS. Ex: "MinhaRedeWiFi"
BSSID (Basic Service Set Identifier)
Endereço MAC do AP para o BSS. Ex: 00:11:22:33:44:55
Beacons e Descoberta
Beacons (Quadros de Sinalização)
O AP transmite periodicamente quadros beacon para anunciar sua presença. Contém informações como SSID, taxas suportadas, capacidades de segurança e timestamp.
Probe Request
STA busca redes
Probe Response
AP responde
