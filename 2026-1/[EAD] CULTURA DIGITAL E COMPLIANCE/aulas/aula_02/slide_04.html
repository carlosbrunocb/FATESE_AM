O que é Compliance as Code?
Compliance as Code é a prática de definir, gerenciar e aplicar requisitos de compliance através de código legível por máquina, integrando validações automatizadas no pipeline de desenvolvimento.

Objetivos principais:

Prevenir violações
Padronizar controles
Gerar evidências
Reduzir custos
Como se Materializa
Políticas como Código
OPA/Rego, Sentinel para regras de segurança
Infra as Code
Terraform + políticas, validações de config
CI/CD com Gates
SAST, SCA, DAST, IaC scan
Logs Imutáveis
Trilhas de auditoria codificadas
Benefícios
Detecção precoce: Shift-left security identifica vulnerabilidades no início
Repetibilidade: Controles automatizados garantem consistência

Exemplo de Controle Codificado
# Exemplo de política OPA
package kubernetes.admission

deny[msg] {
  input.request.kind.kind == "Pod"
  input.request.object.spec.containers[_].securityContext.privileged == true
  msg := "Pods privilegiados não são permitidos"
}
