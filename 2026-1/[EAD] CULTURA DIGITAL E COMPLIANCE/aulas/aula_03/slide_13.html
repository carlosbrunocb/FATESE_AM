Ferramentas de Acessibilidade: WAVE, Lighthouse e Axe

Guia prático para validação de acessibilidade em desenvolvimento web

WAVE
Validador visual de acessibilidade
1
Instalação
Instale a extensão WAVE no Chrome ou Firefox
2
Uso
Abra a página e clique no ícone WAVE
3
Interpretação
Analise marcadores: erros, alertas e recursos
Lighthouse
Auditoria automática do Chrome
1
Acesso
DevTools (F12) > Lighthouse > Accessibility
2
Geração
Selecione "Accessibility" e clique em Generate
3
Análise
Verifique score e oportunidades
Axe DevTools
Extensão para desenvolvedores
1
Instalação
Instale a extensão Axe DevTools
2
Execução
DevTools > Axe DevTools > Run all tests
3
Correção
Veja regras, severidade e elementos afetados
Checklist de Testes Manuais
Teste de teclado (Tab/Shift+Tab)
Foco visível nos elementos
Ordem lógica de tabulação
Skip links funcionais
NVDA/VoiceOver
Contraste adequado
