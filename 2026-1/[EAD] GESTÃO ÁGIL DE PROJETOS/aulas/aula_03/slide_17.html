Start, Stop, Continue - Exemplos Práticos

Aplicação da técnica no contexto acadêmico e profissional de projetos ágeis

Exemplos por Categoria

Contexto Acadêmico
Contexto Profissional
START
• Definir limites de WIP por coluna no Kanban
• Revisar PRs diariamente
• Checagens de qualidade antes do merge
STOP
• Iniciar várias tarefas sem terminar
• Pular code review
• Mudar prioridade sem alinhar com PO
CONTINUE
• Comunicação ativa no WhatsApp/Discord
• Dailies pontuais e objetivas
• Dividir histórias grandes
START
• Registrar impedimentos com label Blocked
• Reuniões de refinamento quinzenais
• Demo para stakeholders
START
• Automatizar testes de integração
• Daily standups de 15 minutos
• Pair programming em tasks críticas
STOP
• Deixar deploy 100% manual
• Abrir issues sem critérios de aceitação
• Ignorar débito técnico
CONTINUE
• Code reviews colaborativos
• Documentação de processos
• Métricas de performance
START
• Monitoramento de métricas
• Feedback contínuo
• Melhorias incrementais

Dicas de Implementação

1
Priorize: Escolha 1-3 ações de maior impacto para implementar
2
Documente: Registre decisões em Issues do GitHub
3
Acompanhe: Monitore resultados na próxima Sprint
4
Itere: Ajuste processos com base no feedback
