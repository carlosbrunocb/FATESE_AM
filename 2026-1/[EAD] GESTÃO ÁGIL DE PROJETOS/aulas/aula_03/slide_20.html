Como Marcar uma Issue como Bloqueada (Labels)

Passo a passo para sinalizar impedimentos e gerenciar bloqueios no GitHub

1
Criar a Label de Impedimento

Acesse Settings > Labels > New label no repositório:

  • Nome: "Impedimento" ou "Blocked"
  • Cor: Vermelha (#D73A4A) para destacar
  • Descrição: "Tarefa bloqueada por impedimento externo"
git label create "Impedimento" --color "#D73A4A"
2
Aplicar Label na Issue

Abra a Issue e na barra lateral Labels:

  • Clique em Labels
  • Selecione "Impedimento/Blocked"
  • Visualize a label aplicada no card

3
Gestão Visual e Boas Práticas

Configure automação para gestão visual:

  • Criar coluna "Blocked" no board
  • Política: "Não iniciar novo trabalho se houver bloqueio"
  • Notificações automáticas para Scrum Master
