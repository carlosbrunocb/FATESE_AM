Matemática simples. Impacto grande.
Lead Time
Lead Time = Data de Conclusão − Data de Criação
Inclui tempo de espera no Backlog. É o que o cliente sente como "tempo de entrega".
Cycle Time
Cycle Time = Data de Conclusão − Data de Início
Conta só o tempo trabalhando. É o que o time sente como "esforço real de entrega".
