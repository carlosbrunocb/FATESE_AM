Parte 1. Calculando as métricas
1
Acessem o quadro do GitHub Projects da equipe.
Façam login com a conta usada no Projeto Integrador.
2
Garantam pelo menos 3 Issues na coluna "Done".
Se ainda não tiver, movam as tarefas já finalizadas para a coluna "Done".
3
Abram cada Issue e leiam a timeline.
Coletem: Data de Criação · Data de Início (movida para "In Progress") · Data de Conclusão.
4
Apliquem as duas fórmulas:
Lead Time = Data de Conclusão − Data de Criação
Cycle Time = Data de Conclusão − Data de Início
